『Approaching，Ready for content』

一到冰冷的電子音把我的意識拉回現實。

「睡飽了嗎？」

「才沒有睡！」

是啊、不過是靠著一下。

「就當作是那樣吧。」

什麼啊，一派輕鬆的樣子。

「痛！別捏我的肚子啊！」

「這是你的錯。」

「我什麼都沒做啊⋯⋯痛！」

真不曉得他是真的不知道還是故意無視的。

「好好好、是我的錯。」

區區詠太還敢囂張。

「總之、長話短說吧，這次目標是龍種，要做的事是調查和討伐。」

說來我還不知道這次的任務是什麼呢，不過。

「不過為什麼要叫上我們？他們就能解決了吧？」

「話是這麼說沒錯，不過妳的相位偏移裝置還沒實際使用過吧？」

相位偏移裝置，左手不自覺的摸向束在大腿上的，那個裝置。

「叫上妳也是為了確認妳的裝置能確實的正常運作。」

此時重機正好通過禁止通行的標誌。

「而且我剛好也有一個小小的個人委託。」

個人委託！

「哼……」

「怎……怎麼了？」

「沒事！」

一定是那隻母狐狸，她一定還有瞞著什麼秘密，作為女人的直覺，我真的無法對她抱有好感。

「唉……」

詠太像是察覺了什麼似的嘆了一口氣。

是說這個男人完全不遲鈍嘛！

氣死人了！

「先不說這個了，還記得怎麼啟動吧。」

我把額頭敲向他的背作為回應。

「那麼接下來……」

「就隨機應變吧。」

事到如今戰前討論或工作分配對我們來說沒什麼意義的。

詠太笑著回應我到。

「那就這樣吧！」

實在是，太狡猾了。

詠太用左手拿下安全帽後，扔向路邊。

我也和詠太一樣，解開了安全帽。

反正之後會回收，而且這裡也封鎖了。

接著，詠太加大油門，衝過了藍色薄幕。

「「Activate！」」

隨著我們的呼喚，白色輕甲就這樣的出現在我們身上的各個部位。

真是奇妙的感覺，我將左手向外伸了出去。

「可以看到了喔！」

碰！

戰區是在一座公園裡面。

可以看到不遠處城護正用盾牌彈開一隻黑龍的右前爪。

「哇！公園都破破爛爛的了。」

「是啊，他們還真是鬧騰啊。」

草地上翻起的泥土，破碎的花壇和折斷的路燈。

「大概是在等我們才弄成這樣吧？」

「好在事後賠償不是我們負責的。」

接著凱替補上前，將手上的大錘往立在地上的左前爪，用力的砸下去。

碰的一聲，失去平衡的黑龍就這麼倒在地上。

「準備好！」

起身後的我單膝跪在後座，把手環過詠太的脖子。

接著重機藉著斜坡衝進公園內。

「來遲了抱歉！」

「太慢啦！」

城護抱怨到。

「之後就交給我們了，盧薩！Type A！」

此時不遠處停著的越野車後方收納箱，如同砲台般升起，並向這裡發射出某項東西。

詠太向前伸出了左手一把抓住了飛行物。

一把白色美麗的單刃劍。

「那什麼！好酷！」

「這次的小委託，剛好有機會拿來測試。」

「還有一把，等下就交給妳測試吧。」

說著說著，重機也來到剛起身的黑龍前。

詠太順勢揮出手中的劍，一劍斬過黑龍的雙前足。

震耳欲聾的怒吼隨之傳來，看來這次黑龍是真的憤怒了。

黑龍拍動著雙翼準備起飛。

不過一道白色光束從遠處飛來射穿了單翼。

與此同時在空中待機的薇恩。

「White Roses[白薔薇]！」

一把白色的長槍出現在薇恩手中，然後向下投擲出去的長槍貫穿了另一片單翼。

黑龍的怒吼轉為了悲鳴。

我並沒有錯過這個機會，從重機上起跳到到黑龍的上方後，翻轉180度。

高度10米。

是在幾個月前我不敢想像的高度。

我看向下方的黑龍。

為什麼作為女高中生的我卻在這裡討伐魔物呢？

不能再逃避了。

其實早就知道了，從那天開始，自己的生活將不再平靜。

我向遠方望去，映入眼簾的是。

如同那天見到般，美麗且動人的黃昏。

接著重力將我向下拉扯，我也將注意力重新集中到黑龍上。

「比翼鳥！」

我將兩把長刀顯現在雙手。

龍首、脖子、雙翼根部、胸口兩道和兩隻前足。

八道斬擊，在我落到地面前所完成的劍舞，每道都深深的斬進了黑龍的身體裡。

落地後的我將雙刀收起，輕輕的原地起跳。

詠太騎車從我的身後駛來，用左手輕輕的接住我後把我放到肩上。

似乎把剛才的劍先丟在一旁了。

「再來就讓妳測試了。」

「盧薩！Type B!」

收納箱再次發射。

這次換我伸出右手抓住了飛行物。

一把潔白的步槍，沒見過的形狀。

「Access code 0003061，mode 1」

『Access code confirmed，normal mode ready』

「妳只要射擊就好，指令由我來下。」

「好喔！」

我端詳起手中的槍，這大概就是為了那個計畫的產物吧。

「要折返了！」

於是我空著的左手，握住了詠太同樣空出的左手，向左側跳出。

著地後拉著詠太、連同重機加速迴轉，再跳回到後坐。

回到後座的我，將槍口對準了我剛才在胸口斬出的X字傷口，射擊。

藍色的子彈傾瀉而出，不斷命中黑龍的胸口。

因激痛而悲鳴的黑龍，胡亂的向四周釋放烈焰，不過舉起盾的城護好好的護著了自己和身後的凱。

「mode change！」

『burst mode ready』

碰！

步槍從連射模式轉變成續力射擊。

嗚，反動有點大。

不過相對的，子彈在胸口炸開讓黑龍稍微後仰，並打斷它的烈焰。

見機不可失，凱一個箭步上前，同樣將巨錘敲上了胸口。

「burst!」

隨著凱的一聲令下，黑龍的胸口再次盛大的炸裂，經過了一長聲悲鳴後，黑龍便倒地不起。

詠太將重機駛到凱和城護旁，薇恩也抱著愛麗絲飛來回來。

「這樣子，任務就解決了。」

「太慢啦！要試的東西試過了？」

「是的，飛鳥的裝置也確認沒問題了。」

詠太順手把手放到了剛下車的我頭上。

我不耐煩的把手撥掉。

「……啊！辛苦了！」

「嗯，辛苦妳了愛麗絲，起飛時的牽製做的很好，還有薇恩也是。」

薇恩向我比出了一個Ya。

「對了飛鳥，那把槍感覺怎樣？」

我看向了手中白色步槍。

「連射時感覺不錯，對付B級魔物大概都很有效，不過續力射擊時反動有點大……」

殺意！

黑龍還沒死。

放開手中的步槍，將比翼鳥的單翼連同刀鞘顯現出來後衝了出去。

拔刀斬，拉出了一道長長的金色斬擊痕跡，貫穿了龍的脖子。

大家也真是的，太鬆懈了。

龍首就這麼的再次落到地上。

我把刀收起來後看向了大家。

只見大家欲言又止的看向我。

怎……怎麼了？我又沒做什……

「啊！」

「比翼鳥……光華一閃？」

「太遲了。」

哇！被凱吐槽了！

「只是不小心忘記了！」

「沒……沒關係的！反正附近沒有人！」

愛麗絲向我安慰到。

「嘛，下次能記得的話就好了。」

什麼啊，那個陰險眼鏡。

「反正我也很常忘記，沒事的」

最後我們一如往常吵了起來。

我再次看向了天邊的夕陽。

美麗且動人的黃昏時刻。

「……跟那天一樣啊。」

世人口中的那天。

也是事後，被眾玩家以諸神黃昏稱呼的那個事件。

那天，我們改變了整個事界。

時至今日。

我們是否能向那個。

以悲劇作為結尾的世界。

給出一個好的交代。

最終，我們都不得而知。

但是，我們會繼續前行的。

「這次一定要好好守護。」

「一定會的。」
