        「 沒想到……陪伴我們多年的遊戲居然就這樣結束了。」

        「 嗯，是啊……真沒想到就這麼突然…… 」

        地球上的某個夜晚……

        《阿爾托爾》前百大公會－－【九影夜遊】的成員們，在某間居酒屋中進行網聚，就像平常在遊戲中一樣，他們也用遊戲中的名字稱呼對方。

        「 真是可惜，我都還沒升到滿等的說…… 」

        一位女玩家如此說著。

        「 哈哈……喵美子是比較晚期才加入的，沒辦法的嘛…… 」

        就像這樣，公會成員們聊著對遊戲的感慨。

        「 對我們這些老玩家而言，帶來的快樂已經足夠了，感謝《阿爾托爾》帶給我們這麼多的回憶，也因為有這款遊戲，所以此刻我們才能聚在這裡。」

        【九影夜遊】的會長向大家如此說著。

        「 不過會長，你已經決定要玩什麼遊戲了嗎？」

        「 嗯，已經決定了。《阿爾托爾》結束營運後，也會有很多致敬之作出現，其中有一款叫《庫托斯卡之旅》的遊戲在先行測試版中玩起來還不錯，我應該會去玩那個吧。」

        「 這樣啊，那我也去玩玩看吧！」

        「 我也要、我也要！」

        「 我也去！」

        聚集在圓形餐桌的成員們，送別《阿爾托爾》的同時，也已經開始為下一款遊戲做準備……

        「 哈……但是可惜呢……我的願望居然就這樣結束了…… 」

        然而，在成員的愉快氣氛之中，卻有一位成員面帶憂愁的自言自語著。

        敏銳的會長注意到，並向他問道……

        「 大樹，你還在意PVP的事情嗎？」

        「 嗯哼…… 」

        接著，一個名為"大河信佑"的某個成員跳進來參與了話題。

        「 也是啦……大樹可是競技場的【世界冠軍】。遊戲就這樣結束營業多少也會不捨吧？畢竟看著紀念獎盃就會懷念什麼的，也是正常的呢！ 」

        「 不……不是的。」

        《阿爾托爾》營運十五年中，總計舉辦了十三次競技場大賽。因為光參加就有獎品可拿，所以加入人數都超過百萬以上。

        而遊戲名稱為"鮭魚大樹"的他，是第三、七、十到十三屆（最後一屆）的【世界冠軍】。職業為敏捷派系的【萬象獵殺者】，是使用苦無、手裏劍和"忍蛛絲"的極高操作難度職業，可謂是最頂尖的PVP競技玩家之一。

         但是……

        「 其實……我一直有一個想擊敗的對手，只是現在已經沒有機會擊敗他了。」

        「 疑？可是大樹你不是【世界冠軍】嗎？居然有想要擊敗的對手？」

         「 嗯，而且……其實……我一場都沒贏過他…… 」

        「 疑？！大樹一場都沒贏過！？」

        「 嗯…… 」

        面對大樹失落的反應，信佑只能露出尷尬的微笑問道……

        「 哈哈－－雖然說一場都沒贏過，不過該不會也就只有比過一場吧？」

        身為百大公會的【九影夜遊】，理所當然的也會參加限時開放的公會戰。所以即使平常不玩PVP的玩家，在公會戰中也會看到大樹那碾壓對手的驚人實力。

        但是……

        「 是打了一百多場……一場都沒贏過呢…… 」

        聽到這，信佑一臉不敢置信的睜大眼睛，在他的認知與記憶中，有看過與大樹陷入膠著的對手，但那都是與大樹一樣為世界賽中的常客。而絕大多數得情況，都是對方被大樹壓在地上打。其中更有公會面對【九影夜遊】，因為大樹的存在而選擇棄權，或著進行完全針對大樹的作戰計畫。

        然而面對這樣的大樹，居然有人在其之上……？

        「 信佑你沒在打PVP所以不知道，大樹想要贏過的人可是個傳奇人物呦？」

        對此有所了解的會長，一臉無奈的向信佑進行說明。

        「 我沒記錯的話，信佑你是第八屆那時才加入的玩家。大樹要贏過的對手，可是第四到第六屆，競技場三連霸的【世界冠軍】呢。」

        「 三連霸？可是真要說的話，大樹是四連霸吧？這一點不就代表大樹比對方厲害嗎？」

        「 那只是僥倖罷了。」（大樹）

        【萬象獵殺者】雖然操控難度極高，但換來的是在PVP中擁有相當高的優勢，其千變萬化的招式與技巧，能夠給對手料想不到的攻擊。

        「 如果單看這樣是沒錯，不過那是因為對方從第七屆開始就沒有參加而已。蟬聯三連霸的那三季，大樹可都是輸的呢。」（會長）

        「 聽起來好像很厲害…… 」

        「 不是聽起來……再說一件事吧，那傢伙同時也是大樹的師傅呢。 」（會長）

        「 欸？」

        聽到驚人消息的信佑，將震驚的視線轉向大樹……

        「 哈哈……當時剛玩遊戲的時候我沒想那麼多，在徵召徒弟的看板中隨便挑一張……然後就…… 」（大樹）

        「 該不會……大樹你PVP的技巧也是那個人教的？」

        「 嗯…… 」

        「 好、好厲害……我都不知道呢…… 」

        「 師傅的職業為【六翼熾天聖騎士】，他厲害到我都懷疑師傅是不是真的人類呢…… 」

        「 居然能讓大樹這樣說……嗯？等等……【六翼熾天聖騎士】？ 」

        在遊戲《阿爾托爾》中，並不存在這樣的職業，除非……

        「 莫非那個傢伙，是七轉職業的玩家……？」

        對這個問題，大樹默默的點了頭……

        原本就震驚的信佑，現在更加得感到吃驚了。

        七轉職業比起直接性的變強，更像是一種象徵性的代表，完成轉職的玩家會獲得稱號【超越神之境界者】。

        而七轉的特色在於，在遊戲中不僅可以讓玩家透過官方審核來為自己取職業的名稱，甚至也可以量身打造專屬於自己的技能。

        然而七轉職業的玩家，在【九影夜遊】中並不存在。

        但那是沒辦法的事情，七轉是在遊戲運營第十三年時才加入的。達成條件異常嚴苛，前六轉分別是10級、50級、100級、200級、400級、700級。

        而七轉不僅需要玩家達到999級，而且還要限界突破十三次才能完成。

        雖然【限界突破】在營運第一年時就存在了，但每一年才會多開放突破一次，而且每一次也都必須儲存【限界值】，而那樣的量也已經相當於讓玩家從1練到999等。

        光達到999等就有一定的難度，有些玩家為此花了一年甚至兩年的時間，更別說還要這樣反覆十三次才能達成七轉。

        雖然每完成一次【限界突破】，角色基礎能力都會增加10%，但七轉需要十三次限界突破的嚴苛程度，到遊戲結束營運為止，也僅僅只有37人達成此一成就。

        而連999等都不到的玩家信佑，對大樹的師傅居然是那37人中的1人感到震驚。

        順帶一提，大樹只差一次限界突破而已。

        「 莫非大樹的師傅一天有48小時……？不過會那麼厲害，多半也是因為有限界突破增加的能力值吧？」

        然而大樹搖搖頭否認了。

        「 不是……競技場有PVP補正系統，限界突破增加的能力值與七轉的專屬技能在裡面是無效的。」

        身為玩家的信佑聽到這，終於也真心佩服起那位大樹的師傅了。

        「 聽起來好扯喔……大樹的師傅究竟是誰呀……？」

        「 師傅的名字叫"呱太郎"。」（大樹）

        身為《阿爾托爾》玩家的信佑，對有名的玩家多少也會有所耳聞，然而……

        「 呃……我怎麼好像沒聽過？」

        「 噗哧－－ 」

        聽到這一句話的會長，不自禁的笑了出來。

        雖然會長的PVP實力不如大樹，但也時常會在競技場中進行切磋。

        「 沒聽過是正常的，因為那傢伙，可以說是狂人啊，根本把競技場當成家了呢。」

        「 會長也很了解他嗎？」

         「 那當然囉，你不認識他是因為那傢伙是獨行玩家，所以壓根不會在公會戰中出現。但是在天梯排行前十名中，唯獨第三名的那個傢伙在顯示公會的那一欄是空白的，這突顯出那傢伙有多特立獨行了。

順帶一提，因為天梯是分數制，所以前兩名都是知名公會洗分洗上來的，而且大樹在天梯中也只有第七名。」

        面對會長補充的話，信佑機靈得察覺到了話中的意思。

        「 這麼說得話…… 」

        會長對此點點頭承認了。

        「 那傢伙在一群頂尖玩家跟洗分渾蛋中穩居第三名，也就是說不用洗分的話，就永遠都無法超過那個傢伙。那傢伙的戰績是1016482勝147敗，就是如此誇張。」

        對此信佑先是算起了位數……

        「 個、十、百、千……百萬？！」

        然後感到震驚的他，再對後面的敗場數，感嘆這樣厲害的人居然也會輸呀……
而看穿這點的會長，帶著惡趣味的表情向大樹說道。

        「 大樹，你向信佑說說看，那個傢伙連贏了幾場？」

        然後……大樹說出了答案。

        「 師傅目前的連勝紀錄是……1016481場。」
