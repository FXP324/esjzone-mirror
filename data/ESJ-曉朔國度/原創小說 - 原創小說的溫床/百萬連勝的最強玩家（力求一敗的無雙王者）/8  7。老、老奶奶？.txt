       「 快把地震的原因找出來！」

       「 嗚哇…… 」

       在草叢中偷偷觀察村莊入口，站著一群像極了村莊衛兵的人。雖然很想直接跑出去抱住那個看起來很像隊長的人，同時大喊「好久不見的人啊啊啊啊！」，但是理智馬上阻止了我。

       偷聽後，貌似是因為福利熊造成的騷動傳到這裡，他們才會如此緊張吧？ 

       然後如果這時一個不知從何而來的男子突然出現，說著「福利熊？被我幹掉囉！」的話，根據輕小說的老套劇情，一定會被當作恐懼的對象，然後就此走上魔王之路的啦！

      說來原本在遊戲《阿爾托爾》 內，就是因為福利熊對村子等其他地方造成嚴重破壞，作為劇情主角的玩家才會現身討伐，結果又因為福利熊同時是被當地部份村子視為守護神而祭拜，所以玩家被他們當成了邪惡的象徵。

       「 不過可不好辦呢……要怎麼才能跟他們對話呢…… 」

       不知道這村子是哪一方的情況下，迫於無奈只好離開了村莊……

       過了一段時間，村子越來越遠……

       「 嗯？」

       然後在更偏僻的地方，發現一處略微破舊的木屋。

       「 廢棄的嗎？」

       稍微觀察一下後，察覺到了裡面有人的氣息，因此準備轉身離去……

       「 是誰？」

       此時門打開了，從裡面探出身子的是一名女性……老奶奶。

       「 啊？真抱歉……我馬上離開…… 」

       彎腰道歉後想著馬上逃走，可正當轉頭踏出一步時……

       「 啊呀！我的腰…… 」

       被老奶奶的叫聲嚇到的我，馬上回頭查看，看到的是老奶奶痛苦的摸著腰……

       「 老、老奶奶……沒事吧……？」

       結果老奶奶的太陽穴上冒出青筋。

       「 沒禮貌的臭小子！說什麼老奶奶呢！信不信我把你給碎－－噗啊！」

       「 啊！」

       老奶奶不小心跌倒在地，我嚇得馬上衝上去扶起來……

       「 那、那個…… 」

       年邁的老人摔倒受的傷是不容忽視的，尤其是眼前一臉痛苦的老奶奶看起來已經有80歲左右了……

       立刻拉出背包，從裡面打開一瓶【HP超級恢復藥水 · 大】，為了以防嗆到而慢慢得流入老奶奶的嘴裡……

       很快得老奶奶的氣色慢慢好轉，她的臉上先是露出震驚，接著轉變成不滿，其中一隻手握緊……

       「 說誰老奶奶！」

       「 欸？」

       我的臉頰隨即被揍了一拳……

       「 嗚嗚……疼疼疼…… 」

       被揍不痛不癢的我，無奈的繼續治療手受傷的老奶奶……


       ……


       「 雖然很沒禮貌，但你很善良呢，雖然很沒禮貌。」

      特地強調沒禮貌兩次的老奶奶，坐在椅子上看著站在的我……

       「 那個很不好意思…… 」

       「 哼！我就原諒你吧！話說你是從哪裡來的？」

       「 這個嗎…… 」

       稍微在腦中思考著，在不理解這個世界的情況下隨便撒謊，很容易被看穿的吧？

       於是在實話與謊話之間，我選擇了前者。

       「 欸兜……我是從《巨木神林》那過來的…… 」

       「 《 巨木神林》？那不是在《拉尼斯懸崖》上方嗎？」

       「 是、是的…… 」

       同時這裡我理解了，至少遊戲的地名還是能用的。
       「 可能講了不相信……其實我失憶了，一醒來就發現自己正在懸崖上落下…… 」

       「 喔？那還真虧你沒事呢？」

       「 是、是的…… 」

       「 而且失憶了還能記得地名？」

       「 這個嗎……有些記憶我還能記得，像地名那些的…… 」

       「 嗯哼。」

       老奶奶那「非常可疑」的想法透過那雙質疑的眼神徹底傳了過來……

       「 哈哈……哈哈哈…… 」

       仔細慎選回答下，與其說「我是從地球來的呦！」這種莫名奇妙的話，我還是掰了一段謊言，不過我現在對現在的情況也是一知半解，所以用"失憶"也不是不對。

       「 啊……算了。」

       最終老奶奶選擇放過了我……

       「 你這臭傢夥，剛剛是不是在心中說我是老奶奶？」

       「 不！並沒有！ 」

       「 哼！聽好了，我好歹也是梵夫爾的魔女，名叫辛格琳 · 梵夫爾 · 拜倫。要叫只能叫我拜倫奶奶，不準叫老奶奶！不能有個"老"！」

       「 是！我知道了！拜倫奶奶！ 」

       緊張的回答後，拜倫奶奶滿意的點頭了……

       「 那麼－－時候也不早了，你就留下來休息吧。」

       「 欸？」

       我不敢置信的看了老－－拜倫奶奶。

       「 欸什麼欸，你不是說自己失憶，那不就代表你無家可歸了嗎？我怎麼可能放任毛頭小子在這麼晚的時候出去呢。」

       「 可、可是……不怕我亂來嗎……？」

       放任陌生人闖進家中，不是生命危險就是東西被偷吧？

       「 嗚？！沒想到小子你好這種啊…… 」

       可是老奶雙手環抱自己，似乎想成別種意思……

       「 不不不……我不是那個意思…… 」

       眼見我急忙否定的拜倫奶奶笑了出來。

       「 呵哈哈，我知道的。不過別看我這樣，我可是很厲害呢，想當年…… 」

       說起自己一段豐功偉業的老－－拜倫奶奶相當得意。

       「 嘛！反正不用你擔心，我這裡還有一間空房間，那就給你用了吧。」

       「 真、真的可以嗎？」

       「 啊呀，是男的就不要囉嗦，我說可以就是可以！」

       於是在老奶奶的指引下，我在空房間裡的床上睡覺……

       「 嗚哇啊……我也睏了，晚安。」

       「 晚安…… 」

       拜倫奶奶關起門後，只剩我獨自一人……

       「 嗯……是個好人呢…… 」

       這種確信來自她自稱為【梵夫爾的魔女】。簡單來說，在遊戲中有兩大派魔女組織，分別為邪惡的【柯沃德】以及正義【梵夫爾】，而且我的第六感也是如此告訴著我。
       「 嗯嗯…… 」

       然後不知不覺的睡著了……


       ……


       「 長谷同學昨天很晚睡呢。」

       不知為何作了一個夢，夢中是高中的時候……

       「 嗯？是啊……昨天打遊戲打很晚，都是可惡的寶物加倍活動害的…… 」

       「 哈哈……那還是老樣子嗎？午餐我幫你買，不過當然的跑腿費也是一樣的。」

       「 嗯，拜託你了大叔。」

       「 真是的，就不能別給我這"大叔"綽號嗎…… 」

       「 哈哈！沒差啦！反正念起來也很順口。 」

       「 抱歉打擾了阪藤同學……有空的話午休可以到學校花園一趟嗎？」

       「 嗯？怎麼了嗎？」

       「 真希同學找你…… 」

       「 嗯……會比較晚可以接受嗎？」

       「 嗯，可以！」

       不知為何，夢到了以前的回憶……

       說來那個傢夥好像是在這之後就被班上同學討厭來著……

       貌似是拒絕了校花真希同學的告白，所以被後援會與粉絲聯手排擠了呢？

       真是的，如果是我肯定馬上點頭接受的，拒絕的結果導致那傢夥徹底被排擠了，如果沒有我的話，那傢伙大概就是自己一個人直到畢業吧？雖然那傢夥貌似很習慣孤獨什麼的，而且嘴上還說「嗯？我完全不後悔呦？」

       「 大叔，雖然他們不敢欺負你，但光是排擠你這點，你也肯定很不好受吧？」

       「 嗯？我完全不後悔呦？」

       他露出溫和的笑容，平常不與別人接觸，就只是一個人坐在窗邊最後的位置上發呆，也只有坐在前面的我會回頭跟他聊天而已。

       「 話說長谷同學很喜歡最後一排倒數第二的位置呢，一直都選這個位置坐。」

       「 那當然！這可是動漫主角最常坐的位置呢！」

       「 原來如此。」

       「 你這傢夥……剛剛是在敷衍我吧？」

       「 不，沒有那種事情呦。」

       「 對了……最近很流行轉生轉移題材的輕小說，真好啊……我也真想試試看跑去異世界什麼的……」

       「 嗯哼。」

       「 你沒什麼興趣呢？」

       「 畢竟我沒在看輕小說呀。」

       「 不看輕小說，你作為人的價值已經沒了！」

       「 居然這麼嚴重？！ 」

       「 你啊……感覺很隨波逐流呢…… 」

       「 呵呵，畢竟我是信奉"平凡過活；平凡至死"的人呀。」

       「 真像老爺爺…… 」

       「 說不定我真的是老爺爺呦。」

       「 噗…… 」

       為什麼這時候會想起那傢夥呢……？

       「 不過長谷同學，肯定去了異世界也會像個主角一樣呢。」

       「 嗯？真的這麼認為嗎？」

       「 ……  」

       那傢夥之後說了什麼，我已經不記得了……

       不知道坂藤桂二那傢夥最近過得好嗎？
