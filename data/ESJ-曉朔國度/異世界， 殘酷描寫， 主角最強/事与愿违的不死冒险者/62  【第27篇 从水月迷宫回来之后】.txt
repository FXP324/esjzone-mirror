【第27篇 从水月迷宫回来之后】
 
 
「……老爷，这里就是我的店喔。怎么样，还不错吧？」
。
　从迷宫回到城镇之后，男子想要让我到他店里所以就被带过来了。
　那是在真要形容应该要形容为「这种地方还有这种店啊」微妙地点上建成的店，就连在马路特居住十年的我都没有进过店里一次。
　嘛，虽说是边境，都市马路特还是挺广阔。
　多亏附近有两座迷宫，才得以吸引众多冒险者。嘛，想要出人头地也不错。
。
　男子的店如同他说的那样，外观上并不差。
　那是一间雅致的石造店家，不会令人有种不想进去店里的氛围。
　要是知道有这种感觉的店，或许还会让人觉得偶然也进去看看吧。
。
　难道说这还算糟糕到惨不忍睹？
。
　这家店在经营上会逼入绝境都足以让人产生这个疑问，真是个谜。
。
　由于男子打开门扉进入其中，于是我也跟在他身后进去。
 
　◇◆◇◆◇
 
「……亲爱的！你平安无事吧……！」
。
　在我们进入店里的同时，有个人如此大喊着抱住了男子，那是位美丽的女性。
(飒:莫名想到这小说第一视角其实是GAL男主的单身基友吧)
　女性那亚麻色的长发往脑后梳成一束，整体散发出精明能干的氛围，全身也玲珑有致。
　用一句话概括，就是美女。
　然而，这样的一位女性却称呼男子为亲爱的，大体上我也知道她是什么人了。
　男子对女性说。
。
「伊莎贝露！我不是说过不用担心我吗……你看我活蹦乱跳的喔」
。
「可是……去迷宫什么的……。我早就说了多少次了，只会做料理的你是不可能探索迷宫的吧！」
。
「喂喂，我起码也是个男人吧？迷宫这种程度……虽然我想这么说，不过这一次就像你说的一样。其实在我快被魔物杀死的时候被人救助了。我能回到这里也是多亏了这位老爷」
。
「……？　啊、啊啦。我真是的。明明还有客人……这一次真的感想您救了我家老公……」
。
　男子的妻子伊莎贝露在听到男子所说才终于注意到我的存在，脸泛红晕地深深低下头。她既美丽而又可爱，有魅力到足以令人想怒骂……为什么会成为这种男人的妻子啊。
　虽说如此，我也不可能老实地说出这番话。
　我朝着伊莎贝露微微点头、
。
「不、用……只是、碰巧、而已……不用、放在、心上」
。
　如此说着。
　随即伊莎贝露露出一抹疑惑神情，男子对此说明道。
。
「虽然老爷是位厉害的冒险者，不过以前好像是被魔物伤到了。不怎么擅长说话的样子。不过，他真的是一个不错的人喔。毕竟救了我……对了，还给我了个魔石」
。
　男子如此说着，从怀里取出那头巨型骷髅头的魔石，展现给伊莎贝露看。
　而伊莎贝露则睁大眼睛注视着魔石、
。
「这、这是……诶，这是这么回事！？为什么把这东西……」
。
「跟老爷商量事情完之后，就把这个给我了……」
。
　恐怕男子还想继续说明下去，不过遗憾的是他没能继续说。
　因为伊莎贝露、
。
「不行！我们不能收下这么……这么昂贵的东西吧！？　给路过的人添麻烦是不行的！被救了一条命，还要这样……」
。
　大叫着。
　从氛围来看，似乎并不是讨厌我的施舍。而是不能再劳烦别人这样一份心情。
　不过这魔石也不是免费送给他。
　我看着男子，催促着他对她说明。
。
「伊莎贝露，其实这也不算是施舍。的确，我利益成分是比较大……不过作为替代，我接下来要为了老爷去工作。还有就是我们约定过在这里可以白吃白喝吧」
。
「……又要、做什么危险的事物吗？」
。
「不是，应该不是……这样吧？」
。
　男子也没自信的样子，对我询问说。
　对此我点着头。
。
「你看，我没搞错吧。基本上我只要代替老爷去冒险者公会报告委托和收取素材而已」
。
「这种事情自己做不就……」
。
　由于伊莎贝露疑惑地说道，于是我自己来说明这个问题。
。
「……因为、我、是、这幅、样子、……不、怎么、想、去、公会……」
。
　我如此说道，稍微展露点手腕。
　虽说我把整条胳膊都展现给男子看，不过我觉得这副样子对于妇人稍微有些刺激。
　意料之中，伊莎贝露吞了口气。不过却也理解的样子。
　她低下头对我说。
。
「真的非常抱歉，让您说些难以启齿的事情……这个人实在太容易上当受骗了，所以我很担心……、如果可以的话，这件事能拜托你吗……？」
。
　这件事指的是魔石的事情吧。
　也就是重新请求确认清楚。
　对我而言也没有拒绝的理由。
　我点头说、
。
「一开始、我、就、这么、说了……交易、算是、成立、了、吧……」
。
　于是伊莎贝露伸出手、
。
「是的，请多多指教！」
。
　对我说。
　应该是要握手吧……刚才看到我的手腕难道不觉得害怕吗。
　我心想要怎么办而望着伊莎贝露背后的男子，而他的表情仿佛诉说着「跟她握手吧」，并用下巴指着她。
　似乎她就是这种女人。
　我也伸出手，接过伊莎贝露的手、
。
「啊啊……」
。
　我如此说着，跟她握手了。
 
「说、起来……还、没、问过、你、的、名字、吧……」
。
　离开店里，我对男子询问道。而男子一副「啊啊」的表情说道。
。
「说起来的确是这样……嘛，因为老爷看起来不想自报姓名的样子，而且感觉也没有把我的事情放在眼里……那么我重新自我介绍吧。我的名字是洛里斯。洛里斯?卡里埃罗。是这家店「赤龙亭」的店长喔。老爷呢？」
。
「我的……名字、还是、别、打听、比较、好吧……」
。
　正确的来讲，应该是我讨厌报上名字会发生问题的情况。
　然而男子――洛里斯似乎无法接受、
。
「什么啊！我还是想知道救命恩人的名字……不行吗？」
。
　他这么恳求我。
　对此我也觉得没办法了，只不过叮嘱他一下。
。
「说是、可以、不过、你、能、跟我、做个、约定吗？」
。
「什么约定？」
。
「别把、我的、名字、告诉、给、其他、的、家伙」
。
「虽然我不知道老爷为什么这么想把名字隐瞒起来……不过我知道了。老爷我只会用老爷来称呼。即使谁来询问老爷的名字我也不会说。这样可以吧？」
。
「啊啊……那么、我说、了。我的、名字……是、雷特。雷特、梵纳……」
。
　当洛里斯听到这个名字，表情顿时显露出微微惊讶，然而立刻说、
。
「我知道了。谢谢啦，老爷。那么，下次有机会再来我家吃饭。到时候会热烈欢迎你的」
。
　他如同自己说过的一样，没有叫到我的名字对我搭话说。
　对此我点点头，然后转身朝着城镇里走过去。
 
　◇◆◇◆◇
 
　老实说，洛里斯有家人这点实在遗憾。
　我还想过或许他是在撒谎，既没有家人也没有负债，只是在欺骗我也说不定。
　如果是这样的话、
。
　「还比较好」。
。
　对，我如此想道。
。
　要问为何，是因为我体内的冲动在这样说。
。
　吃人。
　如果只是尝试一下，是个坏蛋就可以了吧？
　如果是欺骗他人的家伙，应该就没有什么问题了吧？
。
　然而，洛里斯是个不错的家伙。
。
　所以已经没办法再吃掉他了。
　我是想着既然要吃掉，还是不要问他的名字比较好，明明都可以说是心理准备都万事俱备了。
　遗憾……。
 
 
 
。
 
 
　突然，我察觉到自己在心中某处想着这些事情，不禁想道「啊啊，糟糕了啊」。
。
　我边这么想，边迈出步伐。
　目标是罗蕾露的家。
。
　要是能够吃到人类的肉应该很美味吧。
　血绝对能如同浓厚的葡萄酒般湿润我的喉咙。
。
　不，不对吧。
　这不能做……。
。
　脑袋都无法顺利地统一思路。
　啊啊，往罗蕾露、罗蕾露的家……。
