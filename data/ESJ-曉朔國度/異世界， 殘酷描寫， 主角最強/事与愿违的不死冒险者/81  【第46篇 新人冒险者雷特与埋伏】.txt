【第46篇 新人冒险者雷特与埋伏】
 
 
「人、人类……！？」
。
「为什么、他们要埋伏我们？找我们有什么事吗……？」
。
　莱伊兹和萝拉动摇起来，向我询问道。
　对此，我回答说。
。
「这种、可能性、也不是、没有。不过、十有八九、不是、这样。如果有事、找我们、也用不着、隐藏、气息……」
。
　对，如果他们有什么话无论如何都一定要告诉我们，那么犯不着采取埋伏这种迂回的方法，只要普通的过来搭话就好了。
　在迷宫里有这条规定，当冒险者双方都偶遇的情况下，不要抢夺彼此之间的猎物。不过，就算不能搭话也不是说连接近都不可以。
　明明如此，他们不这么做也就是说，有什么不得不隐藏的事情才要隐藏起来，应该要这样想。
。
　对，我对两人叙说之后、
。
「……这、难道说」
。
「果然、是这样吗……！？」
。
　他们似乎也察觉到了，我对两人点点头来告诉他们的想象是正确的。
。
　――两人的想象，换句话说也就是冒险者袭击其他的冒险者。
。
　嘛，虽说是当然也不是百分百一定是这样。
　因为其他某种特殊的事情而造成这种情况也还是有的。
　虽然会期待这些的肯定是愚蠢至极，不过自以为是的应对这种情形也是不对的吧。
　所以、
。
「……那就、确认、一下、吧」
。
　我对两人说道。
　当然，声音小到远方听不见。
　两人听完，歪着头、
。
「要怎么……确认？」
 
「直接去询问吗……？」
。
　他们这么说着，不过我说道。
。
「总之、先做好、准备、以备、随时都、能战斗……由我来、去弯道」
。
　说完，我便踏出脚步。
　要确认的事情就是走到实际上有埋伏的所在地，试试看是否会被袭击，就只是这样。
　这件事无法交给那两人来做。
　无论是莱伊兹还是萝拉都拥有一定的实力，不过经验不足，而且对于以人类作为对手的战斗会感到犹豫不决也说不定。
　其实最重要的原因是，两人一旦被砍到就会死。
　这不是在开玩笑，我的情况是稍微被砍几下也大概死不了。
　不管怎么说，我是不死族。
　在不死族之中的尸鬼，可是拥有无法想象的生命力。它是即便砍飞头部也还是死不了的知名存在。
　嘛，就算是我，只剩下头部我也不觉得自己还能行动。不过，只是被刺穿胸口也死不了也算是巨大的优势吧。
　所以由我来做比较合适。
　虽说两人都打算要制止我，不过在两人的手抓到我之前，我就已经前往迷宫通道的拐弯处。
　见状，两人似乎都死心了，纷纷紧握武器等待着那个时机。
。
　不错的判断。
　明明应该可以选择大声叫住我，不过两人却没有做，只是看清现实并行动起来。
　这对于冒险者而言，是非常重要的资质。
　这个世界很残酷。
　一旦大意，被欺骗，立马就死。
。
　――像我这样。
。
　虽然不好笑就是了。
。
　我边想着这些事情，边来到目标拐弯处。紧接着……。
。
「呜啊啊啊啊啊！！！」
。
　空气随着这道声音剧烈颤动着，于是有位像是冒险者的男人从旁边冲过来。
　男人手里握着剑，高高举起。
　应该是打算砍我吧。
　在他后方还有持弓的男子与像是魔术师的男人。
。
　果然么。
　现实如同我所预料的进行，让我的嘴角立马露出笑容来。
　我拔出剑，把冲过来的男人的剑弹开、
。
「……你们、两人、要小心」
。
　我对背后那两个人说道。
　不过，这或许是多余的关怀。
　因为还用不着我来说，两人的表情就已经完全是冒险者该有的样子。
　在他们的表情上，完全没有因我传达的事实而感到的任何困惑。
　如今，两人的表情上都只是充满着解决眼前的敌人这种感情。
。
　果然是大有前途啊。
。
　如此想着，我决定也认真战斗。
　我向莱伊兹使了个眼色，总之先把像是担任前卫的男人交给莱伊兹对付。而我就解决掉弓箭手和魔法师那边，于是我全力蹬地。
　比以前还要遥遥领先的身体能力让我立即来到弓箭手的面前。
。
「……什！？」
。
　看到突兀出现在眼前的我，弓箭手显露出震惊的神色。
　不过，他并不是无动于衷，而是立即用弓把打算朝我射箭。从这一点来看，实力应该还是不错的。
　然而，我在他放箭之前就已经迅速切断弓弦，并且以剑身全力打中他的胸口，使其昏倒。
　紧接着，我也同样以打击胸口的方式把在后面打算释放魔术的魔术师给打晕。
。
　这样一来，就剩下一个人了。
。
　我如此想道并回头一看，于是便看见莱伊兹与萝拉在跟剑士战斗中。
　恐怕这男人是这一群人里面实力最好的吧。
　从他如今的战斗就足以看出来。
　而能够与这男人对峙的莱伊兹和萝拉也很了不起。
　一瞬间，我迷茫着是否要上去帮忙。不过这是珍贵的对人战。
　为了他们的实战经验，在这种时候还是乖乖看着，直至出现紧急情况比较好。我如此想着，于是放弃了。
　在这段时间里，我把晕倒的两个人绑起来，搬到迷宫边缘处。
　这样就不用担心他们醒过来之后，突然就来攻击我们吧。
 
虽然也不是没有其他问题……嘛，应该是没问题才对。
。
　在那之后过了一会儿，莱伊兹弹开剑士的剑。趁着这一瞬间产生的破绽，潜入他怀里并用身体撞开他。
　而剑士无法完全抵消这份冲击力，导致步履蹒跚。
　于是在这时候，萝拉所释放的土制岩石从后方而来。
　照这样下去，以炮弹的轨道就会打中莱伊兹。不过，应该是事先有练习过吧。
　莱伊兹在接近极限的时机上离开原地，躲开了萝拉的魔术。
　从男剑士的角度来看，就会是眼前突然出现一块岩石吧。
　男人的腹部被岩石打得如同凹陷般，然后男人也失去了意识。
 
　◇◆◇◆◇
。
「干掉、了吗」
。
　我向打倒了男剑士的两人搭话说。
。
「啊啊……勉勉强强吧」
。
「真是吓我一跳……为什么冒险者要攻击我们？」
。
他们困惑地说着。
　于是我对他们说明。
。
「……公会、也、说过了吧。因为、这是、竞争、所以、最快、抵达的、就是、胜利者」
。
　这种说法里面有些可疑的成分。
　如果照原话来理解，那就是只要快点到达目的地就算合格了。
　而这样一来，必然的，慢抵达终点的就会落选。
　从这里开始只是我的推测，如果是这样，只要减少参加者的数量不就可以提高合格的几率？
　参加者之中有人会这样想。
。
「也就是说这些家伙是参加者，打算要让我们落选吗？」
。
「八成、是这样」
。
　这种家伙每次考试都会出现。
　而冒险者公会则会特意用这种容易出现这种家伙的说法来传达考试概要。
　会用那种说辞其实是有这种理由。
　而问到为什么要做这种事，这就要说到冒险者公会的教育话题了。
　嘛，关于这方面的事情我是打算在事情全部结束之后再来告诉这两人，所以现在先放到一边吧。
。
「……总之、接下来、也会有、这种事情、谨慎、前进、吧、别犹豫、不决」
。
　我如此说完，看到两人都深深点头之后。我们就继续在迷宫前进。
　我看着走在前面的两个人，站定住，往背后回过头。
　然后、
。
「……快点、给我、收拾、干净」
。
　我只留下这一言，确认到在后方蠢蠢欲动之物，便追赶着前面那两人。
 
　◇◆◇◆◇
 
　当雷特他们离开这个地方之后，有位人物从黑暗中现身。
　那位人物身穿着犹如融入黑暗的迷宫之中，毫不起眼的漆黑装束。往刚才离开这里的那三人组的方向看去、
。
「……那个人，注意到了吗。不不不，明明就是新人……？」
。
　人物如此嘀咕道。
　从声音来看，似乎是名男性。
　而倒在这个地方的三位男子开始慢慢动起来，其中一人，也就是男剑士说道。
。
「这种事随便啦，总之快帮我解开绳子」
。
「啊啊，抱歉抱歉。不过，虽说是工作，这也太惨了吧」
。
　这人物爽快地对男剑士搭腔说，而男人也笑着说、
。
「彼此彼此吧。嘛，虽然你看起来好像是被察觉到了」
。
　他瞧不起地说道。
　然而，漆黑装束打扮的男人也反过来一副嘲讽的表情说。
。
「你这说法简直就像是自己没有被察觉到，然而你们大概也被注意到是冒险者公会的奸细哦？被那位身穿长袍的人呢。另外两人姑且不论」
。
　男剑士听到这句话，不禁睁大眼睛。
。
「呜诶！？真的假的……什么鬼，那家伙？」
。
　虽然男剑士如此说道，不过漆黑打扮的男子歪着头，不过以一副若有所思的表情说、
。
「……鬼知道。虽然我也不是没有想到某些人的头绪呢……看来应该是如「她」所说吧」
。
「啊？」
。
「没事，我这边的事情。那么，总之先撤退吧？　你们还要削减其他的冒险者吧？」
 
「噢噢。差不多有两组了吧。最近的年轻人根本就上不去啦。不过刚才那些家伙应该能合格吧……」
。
　他们边聊着事情，边逐渐离开这里……。
