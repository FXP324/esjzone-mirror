【第1篇 序章】
 
　
――糟糕，死定了。
。
　我会如此想道，纯属是因为看到眼前迫近的巨大魔物张开大颚朝我猛冲而来的时候。
。
　存在于大陆边缘地带的边境国家亚兰，在王国亚兰边境处的小城镇马路特。
　在这城镇附近的某座低级迷宫《水月迷宫》里，每天通过狩猎魔物赚取生活费的我、青铜级低位冒险者雷特?梵纳。今天也跟平时一样在低阶层一个劲地猎杀骷髅人与哥布尔，收集素材和小魔石。
　这俨然已是每日课程，今天也一如往常的在夕阳西下之时回到城镇里，让冒险者公会收购素材赚取几天生活费的薪水。
　原本我是如此打算。
。
　明明如此。
　那家伙却突兀地出现了。
。
　由于我每天都行走于迷宫之中，所以我根本不会在迷宫里迷路。不过那一天不知为何，在平时通往的道路上发现了一条截然不同的通路。
　那是运气太差了。
　对，运气太差了。
　本来的话，应该要无视这条路才对。
　虽说冒险者是指冒险的人，可也不是毫无计划性横冲直撞的无谋之者。
　不过在现实之中，想也没想就跳进去送死的人非常多，我也是包含其内的愚蠢之徒。
。
　不管怎样，在以前就听说过《水月迷宫》已经被发现、被探索完毕。可如果出现新通路或者新房间，那可就是大发现。
　或许还会有那种高级魔导具和魔武具的装备。即便不是如此，只要探索某种程度并绘制地图，还是可以赚一笔钱的吧。
　带着这样一份愚蠢的想法，我往那条通路踏出脚步。
。
　就结果而言，我在走了一会儿所到达的宽敞场所里与巨大魔物对峙着。
。
　如果不是我眼花的话，那应该就是龙。
。
　龙，它是最高级的魔物。一般情况下，听说是连几名白金级或者神银级的冒险者都匹敌不了的怪物。
　它的外观外貌五花八门，一般是龙巨大化的样子。不过好像也有细长的蛇形态，或者青蛙巨大化的样子。
　为何是好像，是因为遭遇到龙的人如果不是被幸运所眷顾根本就无法活下来，而且它也鲜少出现在人类面前。在历史上，能够真正确认到龙的数量屈指可数。
　它的强大，听说足以匹敌世界仅存的四大魔王。而且它的存在与其说是生物，倒不如说是接近于神的怪物。
　换言之，像我这样的万年青铜冒险者在遭遇到它的时候，连一根手指都不用就已经可以确定我输定了。
。
　这种东西出现在我眼前，怎么可能不惊讶。
　而且，连想要正面对抗的想法都没有。
。
　所以我立刻打算要逃跑，不逃就是死。如此想着设法要让双脚动起来。
。
　――但是。
。
　应该说对方不愧是怪物吗。
　它察觉到我打算要逃跑。
　完全被它注意到了。
。
　――双脚，动不了。
。
　不，纵使身体的某处想要动弹也纹丝不动。
。
　如果是一般人大概会想「这是这么一回事」。
　不过，对于这个现象我有记忆。
　我有学过，当对峙双方的实力差距过于悬殊，就会形成如今的状态。
　由强大魔力所形成的威压使得身体完全无法自由行动。
　虽然这种事例极其罕见，不过也不是没有。
。
　而如今，正是如此。
。
　我没法抵抗龙释放的压力，完全是身体无法动弹的状态。
。
　当理解到这些，我打从心底想着饶了我吧。即使想着这些事情也还是无计可施。
　这种时候我能够做到的，无非就是盯着眼前的魔物看，在心中祈祷着请务必不要吃掉我。
。
　然而，现实并没有那么美好。
。
　那条龙，一看到我就大大张开它的嘴，就这样直接往我这边气势汹汹地冲过来。
。
　――啊啊，打算要吃掉我啊。
。
　性命濒临危机之时，我悠哉地想着这些。
　虽然我也想着「糟糕，要死定了」。可无论如何，在这样的状况下，我已经无能为力。
　不管怎么说，身体都已经动弹不得。
。
　我在十五岁的时候当上冒险者，这十年。
　梦想着总有一天会超越白金级，晋升到人数寥寥无几的神银级别。就这样以冒险者的身份度过来。
　每天靠着简单的委托赚取生活费，即使如此我还是没停下白日梦。一旦委托完成，就开始进行训练。
。
　明明努力到这地步，结果却要在这里结束吗。
　别给我开玩笑了啊。
。
　非常后悔的心情与在这里结束的解放感满载于我心中――
。
　于是，我的身体被吃进龙的口里。
 
　◆◇◆◇◆
 
　然后过了一会儿，奇妙的是我醒了过来。
。
　对，我醒了过来。
。
　无关于被龙所吞食，毫无疑问死翘翘的事实，我醒了过来。
。
　于是，我注意到了。
。
　――不不不。这是不可能的吧？
。
　当我一醒过来，立刻就确认下情况。于是打从心底如此想着。
。
　是什么不可能。
　那是关于我身体的事情。
。
　试着看下手。
。
　于是，在那里已经不存在原本应该拥有的肉。
　没有皮肤。
　在那里的只是洁白纤细的骨头。
　仅此而已。
。
　而且不单单只是我的手是这样，整个身体亦是如此。
　双脚是没肉没皮的骨头。
　大腿也是如此。
　双手，也是一样。
　脸……虽然脸就不知道了，不过肯定也是跟这个情况是一样的吧。
 
 
换句话说，我、青铜级低级冒险者雷特?梵纳不知何时从冒险者转变为骷髅人。
。
　――这是不可能的，对吧？
 
话说回来，接下来要怎么办呢。
　我最应该先思考的事情就是这件事。
。
　总而言之，被龙吃掉这件事就当做是事实。既然如此，我还活着就十分侥幸了吧。
　不对，还能说是活着吗？
　说到骷髅人，那可是一种不死系的魔物。听闻早就已经死透了。
　让教会的司教或者神官什么的使用浄化系魔术就能够超简单消灭掉的超弱小魔物。
　能够被浄化魔术消灭掉，听说是因为骷髅人乃为违背神之定理的邪恶魔物。那位神的定理是「死者无需存在于此世」。
　正因为违背定理才会被浄化魔术消灭掉，这套说词姑且还是有一番道理。
　无论这是不是事实，既不是神官也不是司教的我根本就不清楚。在这一番道理之中，骷髅人姑且算是死者这一事实对我来说才是最重要的吧。
。
　我已经死了。
　也就是在死亡的状态下还存在于这世上。
。
　这就非常大事不妙。
。
　刚才我也说过了，已经死亡却存在于这世上。好像是违背神之定理的样子。
　如果就以这幅模样回到城镇，无论我再怎么主张自己是雷特?梵纳。肯定会是，总而言之先净化掉吧。
　如此一来，我的存在就会被彻底消灭。
　我绝对不要这样。
　虽说我已经是骷髅的状态，可是我还活着。
　纵使骷髅人以定义上来讲已是死亡的存在，只要我还有意识，我就还活着。
。
　所以，我不能回到眼睁睁地看着自己被杀死的城镇。
。
可是，那要怎么办呢。
。
　这就是问题所在。
　是要一直居住在这里吗。
　不过这里可是迷宫。
　为了讨伐魔物，冒险者还会过来这里。再怎么说是边境迷宫，比我还要强的家伙还是有一大堆。
　如果我随意地乱逛，大概会被普通的消灭掉吧。
。
　如果是这样……。
。
　在我深入思考的时候，「说起来」，想起了一件事。
。
　我如今正是魔物。
　魔物持有不可思议的性质，积累经验、度过岁月的魔物会慢慢朝着上位存在进化。
　我究竟是不是魔物连我都完全不清不楚，总之从外观上，我与骷髅人(魔物)是完全一模一样的。
　这么一来，我也可以做到不是吗？
　存在进化。
。
　我突然想起一件事。
。
　由于工作性质，我也对魔物的研究书略有了解。在那里面有记载过骷髅人的存在进化是食尸鬼。
　当然，食尸鬼也是不死系魔物。也正是世人常说的不死族怪物。比起骷髅人拥有更与人类相近的容貌。
　而且也有肉。嘛，我觉得只要用长袍还是面具什么的掩盖一下，应该能够硬说是人类不是吗。
　这样一来，既能够前往城镇，也能够得到跟熟人说明的机会也说不定。
。
　虽然我也知道这是相当荒唐无稽的想法，不过，如今我能做到的事情就只有这样。
。
　所以，我想着「好」。
。
　总之，先以存在进化为目标。
。
　在这座迷宫里变成食尸鬼。
