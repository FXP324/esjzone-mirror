西特莉和露西婭都在努力，而平時總是負責周邊的警戒這回無事可做的莉茲和盧克在附近玩著。

安塞穆在用恢復魔法消除疲勞，我讓緹諾給盧克帶來的特產救生圈充上氣。

隨著深入地面的挖掘，再次被露西婭的魔法挖掘出來的熱水積聚起來。

西特莉心情愉快地哼著歌，用藥水穩固地面。宛如環繞西特莉的土地一般，地面立起著連安塞穆都能遮擋住的高牆。比城鎮的外牆還要高。

吹起救生圈的緹諾怯生生地問道。

「Master，那個... ... 露西婭姐姐大人用的是什麼魔法呢? 」

「唉... ... 我不知道... ... 」

「！？」

我又不是魔法的專家。我只知道露西婭做的事很厲害。

和西特莉合作的話大部分事情都能辦到。不過，好像是把現有的魔法組合起來了的樣子... ..。

瞬間設施就建成了。不論品質如何，就速度而言，連專家也無法與之匹敵。光是看著就有點有趣。露西婭連咒語都沒念。最開始有在念，但半路就不念了。

西特莉種植的種子迅速成長為大樹。哥雷姆將之砍倒。被砍倒和修整形狀的木材被魔法烘乾，組裝在一個大而平坦的地基上。

從像湖泊一樣廣闊的溫泉裡冒出了一股可怕的熱氣。顯然熱水的溫度已經不適合人類使用了。

正在往救生衣裡打氣的時候，盧克穿著便服盛大地跳進了溫泉。

水沫飛濺，緹諾發出了小聲地尖叫。

「好燙! ！克萊伊，好燙！啊，我知道了。這也是鍛煉對吧！該死，這太糟糕了！不禁要想起火山口的寶物殿了呢 ——啊，露西婭。不要降溫啊! 」

「這是我要做的鍛煉。」（這個應該是莉茲說的吧？）

「你說什麼... ... ? 該死，這種事... ... 沒辦法，這次我特地讓給你吧。可是，下次我可不讓哦! 」

還是那麼精神充沛啊。

我躺在充氣的救生圈上，大大地打了一個哈欠。

空氣中彌漫著溫泉的獨特氣味。以往我的在隊伍的職責就是制定方針。這種感覺真是久違了。

就在這時，我聽到有人叫我的名字。挺起身來，朝著唯一入沒有被隔離開來的入口處看去。

一群身穿黑色盔甲的人正凝視著正在絕贊進行中的溫泉工程。他們是葛萊蒂絲的騎士團。

是有什麼事嗎。沒辦法，作為隊長要表現出優秀的一面。

拉著身旁的緹諾的手，去他們談談吧。順帶一提，緹諾是作為護衛帶去的。

因看起來很閑而被選中的後輩雖然有些困惑，但臉上卻洋溢著喜悅的表情，非常好使。

「有何事……？」

領頭的長著精悍的臉龐的壯年男子，恐怕是隊長的男人看著我的臉，皺起了眉頭。

「 ... ... 啊，啊啊... ... 我們是來談談抓到的「巴雷爾」的... ...請問，這些是什麼? 」

「溫泉啊，用看的就知道了吧? 」

「…………」

「正在挖掘住。你看，我們的安塞穆個子有點高，普通溫泉的話……」

「那個子，算，有點高? 」

「 ... ... ... ... 是成長期啦。」

是啊... ... 因為瑪娜源的力量。

因為在帝都很有名，所以安塞穆走在路上也不至於有人那麼吃驚。但是葛萊蒂絲騎士團基本上不離開領地，所以第一次見到吧。所以才說這些鄉巴佬... ... 。

的確這可能有些難以置信，但這是現實。

雖然橫看豎看都太大了，但日常生活中最辛苦的還是他自己。儘管如此，還是繼續保護著隊伍成員，並且他個人還積極接受著國家的委託，因為如果不接受的話有很多人會困擾。

嗯？怎麼？個子高就不行嗎？那個比其他成員更溫厚，更有大人樣的他。

還有個可愛又小巧的神職人員女朋友。而且非常恩愛。現在他們並肩走在一起，看上去已經超出了犯罪的範圍，但是就是這樣才更加難能可貴，安塞穆越變越大，已經大到連葛庫在他面前也像個小孩了，但是他本人似乎並不介意。

「那麼，有何貴幹? 實在不好意思，我現在很忙... ... 」

看著因為有同伴在身邊而變得有底氣的我，團長露出了不悅的表情，但是很快就進入了正題。

事情是關於那一群變成了青蛙的「巴雷爾」的。

看來，他們已經完成了大致的篩選工作，現在正要運送中。

不好意思，我不打算跟著去。如果接受了指名委託的話是應該去確認結果，但是我沒有接受，而且很麻煩。

「不好意思，我還在度假... ... 打倒「巴雷爾」也是順理成章的事，也沒費什麼功夫。你們好好地做就行了。我也不打算接受委託」

「嘖……」

「哦，對了。是阿諾德桑幫了我大忙。為什麼你們不帶阿諾德桑去呢？持有別稱的阿諾德桑。哎呀，真不愧是表現最活躍的。我效仿不來呢。」

順便說一句，阿諾德雖然不在這裡，但還是對我們懷有敵意。但這就是等級8獵人的處世之道。

沒有人會因為別人的讚美而感到不快。對了，順便也誇誇葛萊蒂絲伯爵吧。

「啊啊，你們也幹得漂亮。可是幫了我大忙呢。行動十分迅速。不愧是以精悍而聞名的葛萊蒂絲騎士團。讓等級8也吃驚到無話可說呢! 」

「M、Master... ... 最好還是先別說這個了... ... 」

「……唉？」

「嘖……」

仔細一看，不知為何團長的頭上浮現出青筋。真奇怪... ... 暫時還是別開口比較好。

他身後並排站著的其他騎士們，也都低著頭顫抖著。不管從哪個角度看，都不像是在忍著笑。

團長咬緊牙關好一會兒，然後用足以壓死人的聲音說。

「嘖…………感謝您的合作。會好好傳達給範伯爵的。」

「…………」

「那麼... ... 我想知道有沒有人會不小心被變成青蛙... ... 暫且守護這座城市的士兵變回來了，但是數量還是太多了... ... 」

「嗯... ... 大概沒問題吧? 好像是通過瑪娜源的量鑒別出來的... ... 」

露西婭的魔法是無差別的。如果這座城鎮是帝都的話，一定會很不得了吧，但這個城市幾乎沒有獵人。

我也沒有變成青蛙，所以我認為大多數普通人應該不會變成青蛙。除了阿諾德的隊伍，但阿諾德已經把他的同伴恢復原狀了。

緹諾突然睜大了眼睛。

「Master... ... 那個... ... 那個... ... 說不定，露達她們... ... 」

「 ... ... ... 啊，我給忘了。」

由於發生太多事情信息量我的頭腦已經跟不上了。

確實，可能變成青蛙了。但是我們要從三百多隻青蛙身上找到它們可不是一件容易的事。

我把手放在緹諾的頭上。

「緹諾，你去把她變回來吧。頭髮的顏色會反映在身體皮膚表面的顏色上。」

「唉! ? 由我來嗎? 」

露達的頭髮顏色很普通。我無法辨別，最重要的是我運氣不好，隨便選的話肯定不會中獎的。我也無能為力。

如果我們錯誤地把「巴雷爾」裡的人變回來，騎士團裡的各位在移送上肯定變成吃盡苦頭的地獄模式，那樣可不行。

「反正也沒什麼事可做。 ... ... ... ... 這也是訓練。」

「唉... ... 唉額... ... ... ... 是的。我明白了，Master... ... 」

這是也是訓練。現在想起來，這是多麼方便的詞啊。從現在開始像這樣更積極地使用吧。

我推了她背後一把，緹諾垂下肩膀，走到團長旁邊。

因為消沉的樣子有點可憐，所以就事先跟她說下。

「緹諾，有溫泉在，結束了就回來吧。大概我想你回來的時候也已經做好了。

「是，是，我會努力的! 」

向著我展現出笑容，後輩眨了眨眼睛，精神抖擻地大聲地回答道。

緹諾回來時，太陽已經下山了。

明顯帶著疲憊表情的緹諾，看到了被魔法燈光照亮的溫泉睜大了眼睛。

我坐在寬闊的走廊上，向緹諾揮手致意。緹諾像被主人召喚的小狗般跑過來，坐在我旁邊。

「辛苦了，緹諾，我們剛剛做好了」

「是、是的。Master! 好、好厲害... ... ! 」

到目前為止只有地基的土地上，已經建成了優秀的溫泉浴設備。

用木材、石頭和泥土建造的簡易屋頂上鋪滿了盧克跑去買的榻榻米。裡面在休息的安塞穆也可以悠閒地躺著。

沒有牆壁和門，如果什麼都不想就安裝上的話，安塞穆就很難進去了，畢竟這只是暫時使用的建築。

雖然沒有衛生間和廚房，但是露西婭好像還不使用能變出旅館的魔法，這個，作為替代品也還不壞吧。

但最棒的東西果然還是溫泉。

一個方圓幾十米的巨大浴缸，深度隨地點不同而不同，即使是安塞穆全身都泡進去都可以。雖然只有一個浴缸，但面積比我住的旅館還大。

熱水的溫度也非常穩定。西特莉負責排水機構，溫度調節等由露西婭用魔法負責。

溢出的熱水通過四面挖出的溝渠向外排放。看來是和現有的溫泉設備的排水系統相連接了。不知道有沒有得到許可。

「M，Master，那一邊的是瀑布？還是什麼? 」

「是瀑布哦。」

在邊上，盧克被瀑布沖刷著。看起來亂七八糟的。

像濁流一樣沖刷下來的水的來源是浮在上面的厚厚的雨雲。那是露西婭契約了的水之精靈，它正在勤勉的工作著。盧克似乎也非常高興滿意。水流都把臉蓋住看不清了。

連續使用了魔法多少有些疲憊的露西婭說。

「每次都會想，基本上每個人只能和一柱精靈簽訂契約，會為了修行而讓水變出瀑布

而去契約了的魔導師你不認為我是唯一一個嗎，隊長? 」

「是嗎，這不是我的錯吧? 」

「因為隊長對盧克桑說了一些奇怪的話！什麼說到修行就是瀑布啊! 」

很有用，這不是很好嗎... ..。水也不壞。我也很喜歡。我也想要訂契約呢。

但是，和精靈的契約是魔術的奧義一樣的東西。如果只是借助力量，門檻很低。但是，要這樣使役的話，難度似乎就會大大提高。不過詳細情況我也不知道。

」M，Master，那一邊的是大漩渦，還是什麼? 」

「是大漩渦哦。」

為了維持這種狀態，必須不斷削減魔力，所以只有擁有龐大魔力的露西婭才能做到，不愧是我的妹妹。

」克萊伊醬，我拿酒和菜來了! 」

興高采烈的莉茲滾著三個可以容納人類的桶進來。看來今天是打算是完全喝酒的意思。

後面跟著雙手上捧著兩張放著料理的桌子的安塞穆。

「辛苦了... ... 安塞穆哥哥，請多關照」

「 ... ... ... ... ... ... 嗯。」

唔... ... 我一直還以為是一片足夠廣闊的土地呢... ... 難道很狹窄？

放上幾張桌子就滿了。上面放著我們的成員不管怎樣吃喝都吃不完的量。

但是，控制溫泉的面積也不太好。

節日即將開始的跡象讓我很興奮。西特莉抱著煙花走過來。

」克萊伊桑，我買了煙花喲」

「哦，真上道呢，西特莉」

太完美了。這才是我想要的度假。

有夥伴，有溫泉，有食物，有酒，甚至還有煙花。

如果有艾麗莎在的話就更棒了，但是她是我們隊伍裡最有自己的節奏的，所以沒辦法。

演員到齊，食物和飲料也都齊全。

雖然因為有牆壁看不到外面的景色，但天空中浮著大大的月亮。

好了，度假開始了。我拿著救生圈，帶著認真的表情站了起來，緹諾好像注意到了似的說。

「Master... ... 那個，這裡只有男浴池嗎... ... 」

「啊... ... ... ... ...我家，基本上都是混浴的。你看，因為在外面也沒法在意。用東西遮住，忍耐一下吧」

一開始是隔開的，但是莉茲會翻牆進來，露西婭和西特莉也有打破牆的歷史。

分開的話體型稍大的安塞穆會感到更狹窄吧，而且本來大家各自都是像半個家人一樣的。

就這樣算了吧……作為交換好好遵守禮儀就行了吧。

我的回答讓緹諾凝固了。

「唉？」

「好，緹諾！來鍛煉吧！通過訓練精神，提高精神上的免疫力就行了! 」

從瀑布裡，盧克走了出來。這種堂而皇之的樣子讓蒂諾瞪大了眼睛。

盧克全~~裸著。整個身體只有局部被水蒸汽掩蓋著（霓虹牌水蒸氣專業遮擋值得信賴，遮哪裡你懂的），但他雙臂交叉，完全不知什麼叫優雅，因為頭髮被熱水弄平了所以一點都不帥。

「呀! ? 」

緹諾發出奇怪的聲音，滿臉通紅地躲到了我的身後。

看來要讓她加入《歎息的亡靈》還為時過早。
