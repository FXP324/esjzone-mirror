它的體型比柳一韓所獵殺的巨魔還要小，高度大約是2.5米。而且，它那奇怪的單薄的身體看起來很虛弱，甚至它給人一種弱不禁風的錯覺。

然而，它真正的力量並不是那樣的。

被那黑色的兜帽下所隱藏的，它的身體有一種強大的力量——足以地使高地巨魔首領下跪。而它的黑色鐮刀左右揮舞，好像要割開柳一韓的脖子。

如果第一次攻擊是針對於柳一韓的話，那麼即使是他也很難看清楚。當然，因為它看不到他，所以它攻擊了艾達。

[這是一種尚未被記錄在案的突變體，而且是在怪獸們相互吞噬之後出現的突變個體！幸運的是，它還沒有完全長大，但是它的行為模式仍然是難以預測的，所以一定要小心！]
"明白了。"

柳一韓很快回答著，並不斷移動他的身體。笑話從他的口中消失了，這意味著是情況一點也不悠閒。

他所擁有的最強大的武器——釘矛炮，已經被摧毀了。而在他第二強大的武器僅僅是死亡之神的黑矛。

他的盔甲倒是有一種特殊的能力，但現在發動突然襲擊的可能性幾乎消失了，除了強悍的防禦數值外，它沒有別的優勢。儘管地球經歷了"大災變"之後已經過了不短的時間，但這是他迄今為止遇到的最大危機。

{記錄! }

記錄個P！這讓人可能會認為這是它唯一知道的詞。在它的呼喊下，柳一韓並不打算要硬抗即將來臨的鐮刀，而是選擇避開。

因為他已經見識到了剛才的恐怖襲擊，他非常清楚這鐮刀的威能。柳一韓在力量上是無法取勝的。即使他激活了非人之力，狀況也不會改變。

然而，如果有人問他不能打敗它，柳一韓就一定會嘲笑那個人。柳一韓訓練的核心是他的武術技巧，而不是他的技能、等級或力量。

這一刻將成為他真正的能力被揭示的時刻。

{你快，給我，那個！}
"呼。"

柳一韓並不急於進攻。如果這是他壓制對方的情況，那麼最好手段就是以力破之；但是面對一個強大的，堅定的，而且是不可預測的對手，他就必須先看穿對手。

{ 你——！ }
"來我這。就是這樣，來吧。"

柳一韓嘲笑它，並不斷移動著自己的身體。它的身體，它的運動範圍，它的攻擊半徑，它背後的力量，變化的可能性——他把它們都仔細觀察了一番。

它的身體，它的運動區域，它的攻擊半徑，它背後的力量，變化的可能性 - 他仔細觀察了它們。

它的鐮刀反覆揮舞，似乎反映它內心的瘋狂，但它還是趕不上柳一韓。而柳一韓用盡全力睜大眼睛去觀察它。

就在這時，艾達驚聲尖叫：

[柳一韓！小心魔法！]
"嘖！"

如果能夠繼續進行貼身肉搏的話，那該有多好！然而，現實並不那麼容易。它的戰鬥力包括還包括它的魔法，而這對柳一韓來說仍然是未知的!

[下面！]
"喝！"

大喊的同時，柳一韓蹬地閃避。然而，這些動作並不誇張，而且在任何時候他都可以把這些動作撤回來。

就像他預期的那樣，當他離開那個地方後，一根發著黑色光芒的尖錐從地面上噴了出來，並在空中分裂出尖錐無數。與此同時，怪物的主體也襲向了柳一韓。

"它的攻擊模式是簡單明顯的，這很好。"

幸運的是，它的武器掌握程度比我的預計得要低得多——在這樣認真思考著的時候，柳一韓輕輕地踩地面。這是為了躲避死神的鐮刀。

拜此所賜，一個黑色尖錐碰到了他的盔甲，但幸運的是，盔甲的高防禦力量完全阻擋了攻擊。

{ 記——————錄 ! }

怪物咆哮著，瘋狂地揮舞著鐮刀。所有這些攻擊都變成了風之刃，攻向了柳一韓！

每次怪物的武器一揮，柳一韓就按他所看到的"軌跡"移動身體，並向前跑去。如果他不這樣做，他將不得不受到他身後尖錐的攻擊!

"先吃這個!"

在它的鐮刀沿著對角線向右揮動的時候，柳一韓直接切近它的身體並把矛送了出去——在他觀察的這段時間裡，他發現敵人的攻擊太過銳利了。因此，他有必要通過反擊來限制敵人的運動。

柳一韓和這個"死神"之間有著近100級的差距。他的長矛刺擊，本來不應該造成任何傷害，柳一韓的驚人的技術和他完美的武器召喚了一個奇蹟。

它命中並劃破了那黑色兜帽，並成功擊傷了隱藏在其下的怪物身體！

[暴擊！]
{ Kuaak ! }
"明明是暴擊，但它怎麼只起到這點作用？！"

他一咋舌頭，移動了身子。不過，他確實取得了一項小小的成就，因為風之刃在怪物受到攻擊之後就停止了。

柳一韓立即後退，重新調整自己的架勢。他晃動著長矛，擺脫了他長矛矛頭上的黑色物質，並調整自己呼吸以準備下一次進攻。

{弱，你弱。但又強。給我！你的紀錄! }
"我就知道你會這麼說。"

黑色尖錐從地面上突起，它的鐮刀並沒有結束揮舞，而是又一次產生了像風一樣的刀刃。柳一韓感覺自己像是在玩一場噩夢般的射擊遊戲，這是他曾經玩過的，但現在卻是真人版。

在那個時候，一個小失誤導致他的飛機發生了爆炸；而現在，如果他一個失足，他的身體就將被魔法撕裂。

"再來一擊！"
{ Kuaaak ! }

就像是在他製作鋼矛的時候，柳一韓把注意力集中到了極致，他的身體飛速移動起來。他安全地通過了一層一層攻擊的洗禮，又一次攻擊了"死神"，在確認了他所造成的傷害之後，他會迅速後退。

他忽略掉了一些攻擊他的黑色尖錐，以某種方式避開風刃和鐮刀本身，並攻擊。

觸發暴擊會暴露出怪物的弱點，使柳一韓有機會進行追擊；但如果沒有觸發，他就會毫不猶豫地撤退。他一遍又一遍重複著所有的動作，彷彿預先和死神商量好的一樣。

一點一點地，但毫無疑問，他能夠看到死神身上的傷害在不斷疊加。它的兜帽變得越來越破爛，那受傷的身體正在噴出黑色的光暈。

更令人難以置信的是，柳一韓看穿了所有的攻擊模式和軌跡。

[驚人。]

艾達下意識地摀住了自己的嘴巴。這是因為她害怕她的低語會打擾柳一韓的注意力。

在現實生活中積累技能和運用它們是完全不同的兩碼事。大學入學考試不就是這樣的嗎？有些人會比平時更容易取得成就，而有些人則會因為緊張而產生可悲的結果。

柳一韓則不屬於這二者。他能夠完全發揮他學到的東西，也能精確應用他所知道的知識。對自己能力的低估所引發的挫折對他來說是難以想像的。如果他估計的不正確，他又可以向誰來傾訴自己的挫折感呢？他還是一個泛宇宙級的邊緣人！

那麼只要他做到最好，他就不必傾訴這些了。不要說什麼由於緊張不能正常發揮的鬼話，只要認為他的實際能力就是他的真正能力就行了。
這和目前的情況並沒有什麼不同。如果他不能把他積累的所有技術都拿出來，他就會死，所以柳一韓在盡其所能。

沒有任何人會聽取他的藉口，所以他也不會找藉口給任何人。

{Kuaaaa，痛苦。身體在燃燒。你有多弱，但強壯？給我你的記錄。讓我明白。我想知道。我想要記錄。記錄，記錄記錄記錄記錄記錄記錄記錄記錄記錄記錄！}
"嘿！"

當它的身體上的傷口數量增加時，其鐮刀的擺動精度也在變低，但速度和破壞力反而上升了。風之刃沒有消失，並且在一旦到達某個距離後又會沿來時的軌跡折返造成二次攻擊。並且在地面上的尖刺也變得更厚，數量也變得更多。

無論柳一韓是如何完美的躲避這個怪物，他都無法避免所有的攻擊。他的盔甲上的劃痕增加了，甚至他的頭盔和面具內側都有劃痕。如果他的防護設備全部破壞的話，那麼他根本堅持不了多長的時間。
這真的是一種令人牙齒打顫的力量。柳一韓能充分感受到魔力的力量，姜浩今的魔法根本不能與之並肩。

"*****！我將來一定要學會使用魔力！"
{記——————————錄——————————！}

他不能就這樣被擊敗。即使他必須承擔風險，他也不得不發出一次可能逆轉局勢的攻擊！
柳一韓起身離開地面，忽略了尖峰，避開了風的刀刃，躲過了鐮刀的攻擊，並將黑矛重重地釘在了它的臉上。

[暴擊！]
{Kyaaaak！}

那一刻，一個巨大黑色尖錐從它的兜帽內射了出來。到目前為止，這一招從未被柳一韓納入行為模式，所以這次襲擊是他所預測不到的。
而且，它的目標是在柳一韓頭頂上的艾達！

柳一韓驚恐地移動了他的身體。在這個過程中，他的手臂被釘住了，而盔甲就像紙一樣被撕破了！

"疼死啦！"
[你還好嗎！？]
"我！"

我！習慣了痛苦。這是正常的，因為我習慣了！——柳一韓喃喃自語，好像催眠了自己，同時他馬上與敵人拉開了距離。

從他的傷口上，滾燙的血液滴落而出，並將高地染成鮮豔的紅色。更嚴重的是他的受傷部位變得越來越黑。

[甚至連劇毒抗性都不能……不，詛咒。它的攻擊有一種可怕的力量，是足以紮根於像我這樣的存在的可怕詛咒！]
"無論如何，如果我活下來，你總會做點什麼吧？"
[當然……不，柳一韓，我現在就必須做點什麼。我認為我不在這裡的話對你的幫助更大。]
"……那你怎麼辦呢？"

出於某種原因，這個怪物對待艾達的反應比柳一韓要激烈的多。艾達終於明確意識到它現在的攻擊目標是她自己。
如果柳一韓在保護她的同時再次受傷……艾達無法忍受這一點。

然而，柳一韓卻為她的離開而擔心。由於某種原因，她就算面臨生命危險都無法把她受到限制的天使力量使出來。

而這個地下城的某些特點在針對她。她不知道它是什麼，也不知道如何避免它。這種不可以常理度之的扭曲在侵襲她。

[我是一個高等存在。你不用擔心。]

儘管柳一韓擔心，但艾達在離開前勇敢地說道。那一刻，死神的鐮刀瞄準了她，但是艾達和柳一韓都預測到了這一點。
與此同時，她躲開了，柳一韓的矛則擊中了怪物的肩膀。通過一次可怕的攻擊，將怪物的仇恨從她身上拉開了。

[暴擊！]
{Kyaaak！}
"如果它真的死了，你應該會知道吧？去散散步吧。"
[柳一韓，別死啊。]

留下這些話，艾達像箭一樣飛走了。在看到她正在毫不猶豫地飛到某個目的地後，柳一韓也意識到了什麼。看來她並不僅僅是為了減輕柳一韓在戰場上的負擔而離開。

在目送她離開後，柳一韓直面死神。被詛咒的手臂上的知覺消失，這使他緊張，但柳一韓卻莫名其妙地沒有感到任何恐懼。

怎麼會這樣？他詢問自己並理解了。有一個非常簡單的原因。

"是的，還有一個尚未發生的預兆啊！"

他在開玩笑——這意味著戰況仍然還有餘地。

所有的痛苦和擔憂一掃而空，柳一韓面對艾達離開後變得更瘋狂的死神，露出了獰笑。

然後，他把他的身體投向了魔力的風暴，那裡是死亡的棲身之所。
