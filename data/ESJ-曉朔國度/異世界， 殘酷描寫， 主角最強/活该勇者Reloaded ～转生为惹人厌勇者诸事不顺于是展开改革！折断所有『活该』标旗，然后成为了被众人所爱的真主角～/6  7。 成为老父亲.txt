... ... 咦? 怎么回事? 这种状况？




一个少年纯洁的眼睛闪闪发光地大声喊道: 「太棒了!」「不愧是保罗!」。


冷酷的美少女冷眼看着我，脸上的表情让我脊背发凉。



然后... ... 问题的根源，一个贴在我脸上不停亲吻我脸颊的小女孩。




我一边承受着情感上的煎熬，一边回想着为何事情会发展到这个地步。




◎




「哇哇哇哇哇! ! ! 谢谢! ! 」


「哇! ?」




少女的感谢之声，给人一种空灵的印象。紧接着是男孩惊讶的声音。


这是，在耀眼的光线平息之前，在晶体破碎声的回响消失之前所听到的。



在昏暗处揉了揉被闪到的眼睛，并慢慢地睁开。


于是看到了，站在水边的乔在脸上贴着一个小人的状态下一个人呆呆地原地踏步。



夏尔比我更快从致盲状态恢复过来，手搭着剑柄立刻跑了过去。




「乔君! 你没事吧? ! 」


「是、没事。」


「干什么啊? 没看到我在和master打招呼吗? 别过来打扰我啊!」




一个小女孩在空中飞舞，她离开了乔的脸，并与夏尔保持距离。


与其说是「精灵」，倒不如说是「妖精」更贴切。




水精灵温蒂妮。


有着像流水一样的浅蓝色长发，和同样颜色的眼瞳。


身上穿着一件既不是水也不是纤维所制成的像羽衣一样的衣服




突然回过神来，好像自然而然地听到了她的声音，回想了一下，原作也是如此。


自从温蒂妮开始和乔一起行动以来，和乔以外的人也都很普通的聊天呢。


至少应该给一些解释吧，比如『签了契约的话，普通人也能听到你的声音』。



在我一个人越想越生气的时候，温蒂妮突然向我们鞠了一躬。




「再次... ... 谢谢你们的帮助。我是水之精灵温蒂妮。以后会为master而努力的。」


「哦，哦呜，请多关照。」




精灵和少年分别用食指指尖和双手握着手。


就这样，现在诞生了传说中的职业类别，可以借助精灵的力量无限制地操纵自然的力量的职业------精灵魔法师



嗯嗯，真不错，少年。


年纪大了泪腺就变弱了真是没办法。虽然身体还是年轻人保罗的。




为了不被发现，我转过头，吸了吸鼻子。


但是事情突然开始变得奇怪起来。




「darling，谢谢你！是darling为我把master带到这里来的吧！」


「「「 ... ... 啥?」」」




直接了当地转向了我的温蒂妮的发言，加上我、乔和夏尔三人像白痴的声音。


紧接着，不知道是不是因为欢喜而颤抖着的小精灵，像箭一样飞过来紧紧地贴在了我的脸上


我已经宕机了哦，宕机。


darling... ... 我记得这是温蒂妮用来称呼乔的，对吧？




但是温蒂妮现在一边叫我darling，一边不停地亲吻我的脸颊。


乔的眼睛也闪闪发光，直直地看着我，好像能穿透一切。




「darling找到我，并告诉我一切都会没事的，我真的感到很高兴! 谢谢你! darling! 」


「没错!！是保罗先生把我带到这里来的！不仅知道这个地方就连我的能力都知道... ... 保罗先生！真是太厉害了！」


等一下？为什么男主角乔变成了「好耶!」机器？


说起来，这个场所，这种情况，虽然从我（保罗）和夏尔在一起的时候开始，就很奇怪了... ... 这是不是变得越来越奇怪了？




在亲吻和「好耶!」的混合风暴中，夏尔小姐的眼神变得异常冰冷... ..




◎




温蒂妮，昵称《蒂妮》。




本来是乔给温蒂妮起的昵称，但不知道为什么少年和精灵都要我来给起昵称。


于是我就按照原著取了《蒂妮》这个名字... ..




「加油! master! 」


「... ... 那个... ... 蒂妮小姐? 为什么在我头上?」


「……」


「夏尔小姐能说点什么吗? 」




蒂妮在我头上为乔加油，夏尔在我旁边一直瞪着我。




这个场景，这个故事的主人公应该是乔，但是为什么我的紧张感比现在正面对两只哥布林的乔还要强烈呢？




乔紧张地把手对准哥布林，完全没有意识到他的背后的混乱修罗场。

然后乔下定决心驱使这股力量。




「我、我上了！ 」




就像是在收集周围的水分一样，在乔举起的手前面生成了两个孩子拳头大小的水球。




那是... .




「发射! 水弹!」




随着乔接下来的话，两个水球就像子弹一样射出去，发出了「砰」的沉闷的声音后在两个哥布林的胸口各开了一个洞。


哥布林们连临终的叫声都没有发出就留下魔石化为雾气消失了。




少年转过身来，兴奋得满脸通红。




「我，我做到了! 保罗先生! 」


「哦，我看到了哦，真厉害啊，乔。」




把长枪搭在锁骨附近后和蒂妮，夏尔一起为乔鼓了掌。

太好了，我也被解放了。




「不愧是master! 」


「不用咏唱就能施展出那么强大的魔法... ... 太厉害了，乔君」


「咦? 嘿嘿... ...」




受到夏尔和蒂妮的赞美，乔害羞地笑了笑。




对了对了，这个这个。


乔大显身手→「乔君，太棒了!」


这就是应有的流程。


「乔，太棒了!」不过呢




『水弹』这招的名字，是我即兴想出来的。


本来应该是... ... 像《精灵圣水神击弹--水瓶座》一样痛苦的感觉... ..


如果像原著那样... ... 在一起的时候喊出这个名字，有点... ..




「你现在已经是一个够格冒险者了。」


「 ... ... 真的... ... 真的非常感谢... ... 」




我猜乔自从成为冒险者以来，没有得到过多少表扬。


被我们的赞赏之后乔看起来真的很高兴，在我抚摸他头的时候时，他的眼角有什么东西在闪烁。




我觉得... ... 我有点像个父亲呢。


我不会允许任何人让我的孩子学坏哦？




虽然作品中没有详细说明，只是整合了一些零碎的信息，但是精灵魔法师所使用的精灵魔法与普通魔法所使用的魔法有几个不同的地方。


第一，不像这个世界上普通的魔法，精灵魔法不需要吟诵咒语。而且也不需要『魔力』这种东西当作燃料。




但是如果被问「那么没有风险吗?」的话，答案是没有，因为乔在原著中即使使用了太多的力量，他也只是会感到头晕。


虽然会消费「某种东西」，但终究还是不知道「某种东西」到底是什么。




现在乔只和水精灵蒂妮达成了契约，只能控制水，但只要是和水有关的事情，乔几乎『无所不能』。


虽然范围不清楚，但在故事的后半部分有「我们的用的水足够让大海枯竭了。(原文)」这样的叙述。

                                                                                【我耳的原文】



嗯，乔也总有一天会和土，风，火精灵签订契约。


... ... 而且... ... 这一切也都有肉体关系... ..




... ... 不要做，不要做啊！


我做为爸爸会好好养育这个孩子的！




看着吵闹的孩子们，我不知不觉地紧握了拳头。




◎




「阳光! 森林的味道! 」




从阿奎尔出来的那一刻，蒂妮就从我头上离开了，快乐地在空中飞来飞去。




别说很久了，甚至有可能是第一次到外面的世界去... ... 不过就算不谈论这个也可以。反正也不会得到答案。




乔眯着眼睛看着蒂妮，突然向我鞠了一躬。




「保罗先生... ... 非常感谢你! 你不仅没有抛弃我... ... 还把我带到这里来... ... 而且你还救了蒂妮。」




乔的态度并不是说『我很高兴自己获得了力量』。


他只是单纯地为能让蒂妮出去感到高兴。




这孩子真是... ... ！别怪大叔泪腺崩坏！




为了不让他知道泪腺中了暴击，我斜着头看着天空。身为王牌的夏尔也向我鞠躬。




「我也想感谢你。谢谢你。你果然是我们最好的领袖。」


「别、别这样!」




连我自己都不明白，为什么我要在这里说「别这样!」... ..


但是，如果不说出这样的台词的话，可能真的会哭出来的... ..




就在这时，蒂妮从天而降，又紧紧地贴在我的脸上。




「我也是! 我也是! 谢谢你! darling! ... ... 啊? darling，你哭了吗?」


「我、我可没哭啊! ? 喂! 要回去吃饭了!」




嘶嘶！我抽了抽鼻子，拍了拍乔和夏尔的背。


然后，为了不让她们看到我的脸，推着她们的背，走了起来。




　


... ... 我也不知道这种状况还会会持续多久... ..




但是！既然这样，我就要以我自己的身份来改变这个扭曲的故事！
