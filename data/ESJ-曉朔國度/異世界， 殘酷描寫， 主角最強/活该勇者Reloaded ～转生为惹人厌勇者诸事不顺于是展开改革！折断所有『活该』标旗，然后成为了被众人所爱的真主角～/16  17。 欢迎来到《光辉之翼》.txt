「那些人... ... 他们本该成为乔的同伴吗?」


「是啊... ...」




据点二楼，保罗的房间。


在那里，我绞尽脑汁的向夏尔解释事情的经过。



把一切都告诉了这个孩子真是太好了。


多亏了你，我就不再用一个人默默的头疼了。




夏尔坐在椅子上，我坐在床上，继续着这个话题。




「在我所知道的那个世界中按流程，他们会被《光辉之翼》粗暴地赶走。毕竟他们是e级，这再正常不过了。这导致的结果就算他们会和乔相遇并一起组建一个新的队伍... ..


「 ... ... 是吗... ... 多亏了『你』，乔才能继续留在《光辉之翼》中... ... 所以，即使拒绝他们加入，也不会成为小乔的同伴。」


「理解的这么快真是帮大忙了。」




我忍不住叹了口气。


那么，现在你可能会问: 「为什么不让他们加入《光辉之翼》呢?」... ... 因为有一个问题现在让我很头疼。




「那你为什么不让他们加入呢? 」


「对吧... ... 我也是这么想的... ... 但是... ...」




夏尔的反应和我所想的一模一样，紧接着我又叹了口气。


然后我用手捂住了脸，从手指的缝隙中看着歪着头的少女。




我必须告诉她... ... 我为什么会这么头疼... ..


非要说的话，我不想让那些《女孩》接近乔。


这么做的理由当然是... ..




「 ... ... 据我所知，乔和那些女孩子... ... 会成为朋友... ... 那个... ... 『男女』意思的朋友... ...」


「 ... ... ... ... ... ... ... ... 欸! ?」




过了一会，沙尔发出奇怪的声音并站了起来。


她手忙脚乱地乱动着，好像在跳一支奇怪的舞。




「哪、哪一个? ! 」


「 ... ... 所以说... ... 那些个女孩『们』... ...」


「 ... ... ... 两个人! ? 为，为，为什么会这样? !」


「咕! ? 我、我也不知道! 你去问作者啊!」


「zuozhe是谁? ! 」




我发出奇怪的声音是因为混乱的夏尔抓住了我的衣领。


话说，你既然这么重视伦理为什么还加入了后宫？




没错，乔成为《打桩机太郎》的开端，就是莎拉和艾米丽。


实际上《开始》---- 不如说是《脱衣... ... 咕噜咕噜。




但是，如果《两个人》就让她变成这个样子，那么我最好还是不要告诉夏尔关于蒂妮，其他的精灵，还有即将到来的女孩们的事了吧。其中也包括我自己的... ..


她可能会精神崩溃。




按住她的肩膀让她坐在椅子上，像拳击助手一样让夏尔平静下来。




沙尔调整了一下呼吸，同时脸上也露出了轻松的表情。




「 ... ... 这次就当没有缘分吧... ... 」

「哦哦，夏尔小姐，你可真是残忍。不过... ... 你也知道，不管是好是坏，乔的成长也有一部分是得益于和他们的相遇... ... 。」




我也再次坐到床上，双手抱头。


即使是坏朋友，对当事人们来说也是个重要的交际关系。


如果把这件事当作与我无关我【爸爸】，那又会变得怎么样呢... ..




我有一个贪念，我只想让那个男孩【托什】加入。因为他成为乔的好友起到了【好与坏】里面大部分【好】的作用。




但是，那三个人是青梅竹马这样的设定。


如果因为我的一己私欲而拆散他们，这就太残酷了。


在只让他加入的情况下他们之间的关系是不可能能维持原样的... ..




嘛，乔从他好朋友那里抢走了青梅竹马，甚至每天晚上等他睡着后在旁边《狂欢》。




在后宫作品里男性友人模板的角色是非常少见的。果然模板才是很重要的，嗯。


这部作品教会了所有读者这件重要的事。




以下是网络和评论栏中的反应。




「乔 = 我(作者)。托什 = 你(读者)。作者想这样来表达自己压倒性的优越」


「打桩机太郎这混蛋... ... 所谓令人作呕的恶，正是指这件事。」


「把托什也带上吧! ! 」


「托什: 喔喔喔!！！ 乔君!你太棒了!！！！！ 」


「不，不是指那样。」


「我感觉还挺不错的呢! 」


「我也是！」


「变态涌出来了，草。」




哎呀，还真是混乱啊。


因为托什可怜过头了，甚至出现了以他为主角的同人创作。


创作出的故事能让读者的心灵受到如此震撼，也许作者其实是个天才。




嘛，先不说这个了... ..




「 ... ... 怎么办?」


「你说怎么办... ...」




我困惑地问道，夏尔小姐看起来也是一脸迷茫。




我第三次发出的叹息声，与夏尔发出的叹息声完美地重叠在了一起。




◎




「啊! 找到了! 我回来了! darling!」




我和夏尔一起下楼的时候，伴随着欢快的声音，蒂妮紧紧地贴住了我的脸。


因为已经习惯了所以没有一点情绪波动。




「哦呜，欢迎回来。果然很快啊。」


「哼，毕竟我也好好努力过了。」




离开我的脸后，蒂妮一脸得意地挺起了胸膛。




乔和蒂妮整个早上都和加入的新手团队一起外出了。这是为了采集一种可以作为魔法药原料的珍惜蘑菇。




这东西非常不容易找到所以稀有的很，但对于乔来说应该是轻轻松松吧。


乔可以通过《神之耳》来收集情报，而且现在还有一个叫蒂妮的优秀精灵【无人机】。




「啊! 保罗先生! 夏尔小姐! 我回来了!」




乔在蒂妮后面发现了我们，脸上洋溢着天真的笑容，慢慢地向我们走来。


我也笑着抚摸着他的头。




「欢迎回来。比我预想的快好的啊。不愧是乔。」


「真厉害。换我的话搞不好在山里呆上几天都找不到。不愧是乔君。」


「对吧对吧? 我的master可厉害了。」


「欸，欸嘿嘿嘿... ...」




我们的赞美让他红着脸害羞地笑了。


面对这样的情景，我和夏尔也自然而然的笑了起来。




嗯... ... 怎么回事？这种压倒性的『爸爸，妈妈，儿子，宠物』感。


虽然告诉夏尔和蒂妮的话一定会被骂的。




我看着站在我旁边的夏尔的脸，她也看着我，表情紧张的点了点头。




「我现在想和你们谈谈，能占用你俩一点时间吗?」


「好的，没关系。」


「有什么事吗，darling?」




「有点事。」我只回答了一句之后，把蒂妮放在了乔的头上，推着他的后背向前走。


目的地是无所事事的三个人所在的位置。




看到我和夏尔回来，他们似乎松了一口气，我先向让他们久等了这件事道了个歉。




「对不起，把你们放置了。因为我在和夏尔讨论你们的事。」


「不，不! 我们没事! 」




托什和其他人刚要站起来，我就先把他们拦住，并从附近的桌子旁边拉过来一把凳子。


一边是搞不清状况的乔，另一边是一脸微妙表情的夏尔，而我则坐在剩下的空椅子上。




我和夏尔经过讨论后得出的结论。那就是... .




「那么，经过讨论... ... 我和夏尔决定暂时同意你们的加入。」


『库..！』




就像原著一样，让他们成为乔的伙伴。


决定这件事的原因是为了最大限度的减少未来的变化。



尽管我知道原作里的未来，但如果变动太大，说不定会出现意外。


主动放弃难得的优势，还是有点鲁莽吧。


但取而代之的是，我们会进行密切监督和指导，以免他们过上糜烂的生活＆性活！




不过，这也只是「暂时」的。




「 ... ... 太... ... !」


「等等! 现在高兴还为时过早! 」




他们大概是想要发出「太好了！！」这样的喜悦吧。

我用伸出的手掌制止了刚要摆出【万岁】姿势的三人。让他们先坐到凳子上。




「姑且.....说一声。你们也应该听公会那边说过了吧，加入《光辉之翼》的最低标准是c级，你们现在连初出茅庐的d级都不是，只是可以说是业余者的e级吧。」


听到我所指出的事之后，三个人都「唔... ...」了一声，低下了头。


虽然不是直接针对他们，但他们听起来一定也不太舒服吧。乔也低下了头。




所以，这是保罗【爸爸】和夏尔【姐姐】的作业哦。




「然后，我给你们介绍一下。这家伙是乔.麦克斯韦，还有契约精灵蒂妮。他们会成为你们的前辈，顺带一提他也是 e 级。」




当我向他们介绍乔和蒂妮时，乔客气地鞠了一躬。和蔼可亲的蒂妮也微笑着挥手。




我清了清嗓子，夏尔接着说我没说完的话。




「从现在开始，乔和你们三个要一起行动。然后... ... 作为正式加入的条件，你们要在一个月内升到c级。」



看着对着三人，或者说，对莎拉和艾米丽投以严厉目光的夏尔，我哈哈大笑。




本来的话一个月就从e级跳到c级是不可能。


但是，没关系没关系。因为在原著里就是这样的。




『 ... ... 欸欸欸欸欸欸欸欸欸欸欸欸欸欸欸欸! ?』




愣了一会儿之后，四个人同时发出了完美的惊叹。




嘛，如果一开始就这么默契也不用再担心什么了吧？(笑)
