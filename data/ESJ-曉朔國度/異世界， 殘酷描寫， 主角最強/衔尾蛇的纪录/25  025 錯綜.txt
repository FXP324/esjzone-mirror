轉載自貼吧
 
 
[ 無誤，無誤……]
 
[ M-14向M-22指出，誤差確認。根據設計圖紙，那個石料的配置還需向右移動五釐米。]
 
[ M-22，瞭解。矯正格雷姆的舉動。誤差修復。]
 
在作為我的實驗室的頭頂上的山頂，採光設施兼迷宮的建造高速進行著。為了吸收太陽光而在地下鋪設的大型水晶（用在礦山出產的石英做）裝置以及光纖型禮服（由折射率變更的光量調整功能附著）完成之後，還需要做的就只剩下以後作為侵入者擊退用的週邊部機關。
 
那個和這個的進度如此之快，都是多虧了此前大量增加的批量生產型奴隸們。礦山奴隸的籌措為避人耳目而讓魯貝爾去買,不過，因他談判時不斷討價還價。托此大幅度超過預定籌措數而有五十二人新加入了。在某個O女O女是高級奴隸的傢伙不斷叫喊著吵鬧之後，最終這個數字。魯貝爾很自重嗱。是因為與我的利益有關，為了我所以這麼拼嗎？
 
由於那樣所為，我暫時度過了每天玩弄奴隸腦子的日子,(啊！魯貝爾很有愛嗱！)每天不停地做，因而技術有所提升，最終有這個成果也不錯。有這個量的批量生產型奴隸全體出動作業，會比起預定的工期縮短大半時間。
 
如果把視線轉移到山下的原野，在那裡鬱鬱蔥蔥的大森林展開著。種的是靠近瑪爾蘭的土地、氣候近的地域收集而來的樹木、特倫托之類的植物型怪物。之所以有此成果，是因有E系列事件而改造精靈們熱情地培育著，因而這一大樹海才得以完成。其他乾燥地區的也放出各種魔獸類型的怪物充實著。這樣半瓶醋的冒險者即使好不容易走到這個採光設施也很難出去，那些抓進來的就更別想了。交給精靈的事也沒問題，她本來就工作很認真。
 
從冬天開始持續的我的新實驗室建設計畫，完全順利地推移著。
 
領土內政也沒有問題。去年施給的土壤改造計畫以成功圓滿結束，今年農業收穫量好的數字也能期待。治安的方面嘛，由因這個那個而鬱憤積累的杜耶應對盜賊團兼消磨發洩。和平無比。礦山方面現在把奴隸採礦工人放入當臨時工，應該再過一陣就能正式開始動工。
 
好像全部順風滿帆，一切都一帆風順……的樣子。
 
[ 怎麼了，主人？有什麼不適嗎？]
 
和平常一樣，在我身邊的優妮不安地詢問道。
 
[ 不，沒什麼。只是什麼都順利地進行著，那個反過來讓人覺得不安。單純的顧慮，一時的迷茫罷了。]
 
與其說是對著她而答，倒不如說是為了自己的疑惑而辯解著。
 
雖然表面沒有什麼不安，但是心裡還是會有所……雖然我覺得是有些過於神經質了。
 
在我亂思著，心裡亂麻時，優妮說出了意想不到的話。
 
[ 雖然有些失禮，但現在並不是一切都是順利的，我是這麼認為的。]
 
[ 哈？為什麼這麼說？]
 
[ 到現在為止，主人為實驗室的整備、及領土內政而集中精力，因而即使心裡有想建議也未敢獻上。雖然有些失敬，不過關於這個，是無意識不安的嗎？難道真的毫無頭緒嗎？]
 
嗯。被她這麼一說，試著考慮的話，確實稍微有些問題放置過多的領域有可能想起。坦白地說非常麻煩，再加上進入我自己相當疏遠的一方領域的話，到現在都沒積極地專心致力的部分。
 
[ 關於外部的諜報，完全不行嗱。]
 
[ 誠如所說。主人到現在為止，對兄君的計謀對策都屬於被動的，無法做到有效預防，只能防守。現狀，由於並沒有發生什麼事，因此無法預做什麼對應，並且也不知道會持續到什麼時候。]
 
就像優妮說的那樣。為了萬一時刻有處可避而開始製作這個大規模三重迷宮,不過，終究只是實驗室兼避難所。如果因政治的謀略下臺，丟失社會的立場，在這裡蝸居會有大幅度縮小研究規模的囧境。那些為保身製作的奴隸，其身份也需保障。為了不變成那樣，必須經常持續抵禦哥哥的謀略。不過，對面要是不能取得成果，次手會更強有力且下面的手會愈加巧妙。也必須要想辦法解決那個，要從這邊伸展手。
 
這個瑪爾蘭大體上作為我的領土如裝置般受我控制擺弄。但是，那是徹底作為孤壘而只能進行防禦戰。對方如果開始鼓起勁進攻，不知何刻就輸了。無論怎樣，對方是在這個國家的權力中樞——王都。如果將時間給予對方，對面不知會把怎樣不明不白的工作送來。所謂鬥爭，總有一天一定會迎來攻進的局面。向那裡挺進的準備絕對地不足啊。在陰謀開始之後不僅是防禦，還要積極地察覺並弄碎擊潰！要徹底擊潰對方伸過來的手，現在的我是沒有的。
 
那個正是我的不安的原因，優妮正好指出我的不周。
 
[ 的確，你說的不錯。正好加上加強防守的課程。或者說這邊差不多也可以在進攻的準備上著手的時刻？]
 
[ 明察，恕言不敬。]
 
她那樣說著然後深深地垂下頭。想那樣這般道歉的是我。說不定如果不具體地被指出，就那樣只是當成因為心情的原因，放置著這個問題，恐怕日後會喪失一步。
 
[ 好吧，如果決定那樣就立刻著手。最好是首先跟維克多商量一下嗎？]
 
[ 是。像這樣的事，向擅長的人詢問較好。]
 
無論怎樣，他本來就是到這來偵探我的間諜。這種事在我的勢力裡沒人比他更精通這方面的，一定會有什麼好的智謀。
 
——不過。
 
[ 刺探王都形勢的諜報工作？……現狀難。]
 
對著一返回宅邸就詢問的我，一副愁眉苦臉的維克多答道。
 
伯爵家的庶子、侯爵的私生子的他關於貴族社會的複雜奇怪的結構，在我的手裡的棋子中最為詳細的男人。有這樣那樣背景的維克多說難。這個與我當初的預期完全脫離。
 
[ 那麼難嗎？譬如，對你親密的貴族多少人一股腦洗腦，編織――]
 
[ 對方是那個拉瓦萊侯爵喲？是派出我到你這地方的之時，有倒戈的可能性就算計著。會計算我被閣下拉攏，因此背叛奪取這個家的權力這個可能。如果被逮到尾巴的話，恐怕至今與我的友人接觸會被捏造成我們在進行著某種不軌陰謀，被王國通殺。]
 
嘗試伸出手爪的想法，被遺憾的拒絕。而且即使是自己的親生父親也那麼心狠。嘛，我也是被親生哥哥瞄準生命的，沒資格說別人嗱。
 
[ 也就是說，粗心急挫的進攻可能會招致適得其反的效果……但是我覺得伸展手臂的時機已經到了，只要把手放穩些。]
 
[ 好象有少部分欽佩閣下這麼快就有爵位且感新奇的貴族。不過遺憾的是，即使這樣，現在到王都越級留手恐怕會被嚴厲對待。]
 
話都說到這個份上，也就沒有辦法，這是無可奈何的。
 
想到此我不禁仰望天花板。
 
[ 維克多，你意外地壞心眼啊？]
 
這時插入對話的是腹黑財奴——內政官西蒙·傑克·魯貝爾。
 
[ 嗯？怎麼說，魯貝爾？]
 
[ 領主閣下，維克多說以現狀探聽王都的政情是難,不過沒對進行諜報工作反對喲，沒注意嗎？]
 
其話如立板上的流水般灌入我的耳朵。
 
唉，也就是說？
 
[ 嗯，也就是？欲將先射馬？要捅人一刀先把他的腳給掃了？]
 
[ 是，誠君所言。突然在身為城堡中心的王都竄走亂插線，是魯莽無謀的。因此首先從周邊的諸侯開始部署安孔，並以他們向外延展拉線，網就織成了，閣下只要趁機利用就行了。您是這麼想的，對吧？]
 
魯貝爾以詼諧頑皮的表情遞眼色。
 
可是，遺憾的是男人使的眼波並無法讓我興奮。維克托也應該是這樣吧。
 
他誇張的歎了口氣。
 
[ 魯貝爾，別太嬌縱閣下。閣下如不學習下您自己哪些會被貴族社會所惡的行為，對此有所自覺的話，家臣的我們也會感到為難啊？
 
要是其他的話題的時候，我也會那樣做。但是，如遇到事情發展過快，沒有儘快回答應對的話，那正是最容易導致危機的事態。如果您不儘早接觸這些的話……]
 
看來維克多現在開始要對我實施帝王學教育。
 
不過我覺得出生於這個世界已經快二十年，坦白地說感到現在學已經為時已晚。
 
[ 即便那麼說，做不來的事真的難以做到。]
 
[ 一直與宅邸完成之宴的時候那樣，那個內部遲早會出現漏洞的。]
 
被極其嚴冷地對待了。
 
無奈的我向從剛才開始一直沉默的優妮試著傳送水波……
 
[ 身為僕人的我，難以干涉，無法判斷話題。]
 
居然徹底貫徹不干涉原則。居然連優妮都轉到敵面！多麼可怕的局勢啊！
 
我體會到了被布魯圖背叛的愷撒、明智光秀向本能寺攻進時信長一樣的心情。以前這麼沒用的前世知識的有這麼活用嗎？
 
[ 女僕長是參與制作的關係人員，沒問題的。]
 
[ 嘛，由於那個所為我和維克多是被洗腦了，不過沒什麼。]
 
[ ……謝謝。]
 
[ 啊—，嗯。關於那個我的貴族交際一直沒怎麼進行，因此具體怎樣做？清楚地說，周邊諸侯對我的好感度都很低啊？]
 
強行撇開話題。
 
魯貝爾說的周邊貴族的拉攏，也是有相當難度高的。
 
無論怎樣，關於我的評價是最壞的。
 
因此與貴族們的交際發展是挺重要的，雖然不需現在就急急忙忙地做，不過用來轉移話題是……
 
[ 那樣想的話，請更好好地做喲。]
 
何等刺人的話呀。
 
[ 因此那個作為今後的課題……有怎樣的策略，魯貝爾？]
 
[ 乾脆從一端洗腦怎樣？]
 
……到底是在說啥，這傢伙。
 
我一邊一動不動地看著魯貝爾挪開的臉，一邊嘗試打算讀出那個意圖。
 
是因有什麼考慮才這麼說吧？打算像維克多那樣讓我自己考慮正確的解答嗎？那麼，魯貝爾考慮的正確的解答是什麼？
 
[ 深讀也是沒有喲。這個男人大概是認真地說著。]
 
對暫時沉思的我，維克多如此說。
 
哈？是真的麼？
 
我禁不住直眨眼。
