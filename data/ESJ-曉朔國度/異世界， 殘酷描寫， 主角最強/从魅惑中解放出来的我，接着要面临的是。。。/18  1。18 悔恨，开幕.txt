抵达王都的我，和勇者大人一起来到了王城。
到达的时候，那些看起来像是达官贵族的人想要把我给赶出去。
这时勇者大人对他们大声呵斥道: 「这个人是我的女人莉娜，从今天开始她就要住在这里了。」随后很快我的房间就被准备好了。
（勇者大人的权力，看起来真的是相当强大呢。）
有些人似乎反对我住在城堡里，因为我只是个村姑，但是，当勇者大人用圣剑指着他的时候，他也变得沉默不语。
「孝二大人... ... 能住在城堡里什么的，简直就像做梦一样。谢谢您。」
「你在说什么啊， 莉娜是我的女人吧? 那样的话，你住在这里也是理所当然的哦。」
带着充满热情和情欲的视线… … 我和勇者大人的目光交汇着。
（明明已经在马车里那样激烈的做了，但是，我又想要勇者大人来疼爱我了嘛❤❤... ... ）
所以，我又一次地向勇者大人求欢，在准备好的房间里，就这样做了一整天。伴随着不停的做爱…我对勇者大人的迷恋也变得越来越深。
 
 
 
在那之后的一段时间里，勇者大人每天都会来我的房间，索求我的肉体。
每当听到床吱嘎作响的声音，正在被勇者大人这样激烈地爱着的实感就会从心中涌现。
而且，为了让我的勇者大人更加高兴，我在身体的那个部位也纹了一个纹身，在上面刻上了「孝二大人专用（❤）「的字样。因为是用魔法刻上的，特别的，我的爱的标志，所以这个纹身是永远都不会消失哦。勇者大人看到它的时候，一边比平常更猛烈地上着我，一边带着无比兴奋的表情。
这之后，想要生下勇者大人孩子的我，使唤了城堡里的仆人，让他们带给我在王都贩卖的，可以诱发排卵的药物，积极的服用着。
 
 
 
 
 
 
 
 
 
 
但是，我没能怀上孩子。
 
 
 
 
 
并且，从那之后，勇者大人来找我的次数，也逐渐减少了。
后来我才知道，勇者在和我上床之前，每次都会对我用上避孕的魔法。也就是说，从最开始，他就没想过和我结婚，我对他来说，只是玩物而已。
听说是因为，让最开始被他玩弄的女性怀孕，不得已把她给处理掉了，为了避免麻烦而采取了那样的手段。
离开村子的时候，对我许下的结婚的诺言也完全没有任何兑现的意思，就这样随着时间流逝，不到一年，他就迷恋上了另外一个女人，终于，再也没有来过我住的地方。
 
 
 
 
尽管如此，我还是继续盲目地爱着勇者大人。
 
 
 
每天把身体打扮的艳丽漂亮，穿着勇者大人喜欢的衣服，不论何时，我都准备着，等待着他来找我。
一年过去了，两年过去了，因为勇者大人终有一天一定会来找我，和我交欢，我每天都这样继续等待着。在这城堡里，除了我之外，还有很多其他的，属于勇者大人的女人，也和我一样等待着。
我们的关系并不恶劣，正相反，我们彷佛组成了迎合和称赞勇者大人的盟友一样的关系，并为此而感到由衷的喜悦。勇者大人的勇猛... ... 被勇者大人尽情使用发泄欲望的，那样的幸福感... ... 令人毛骨悚然的是，我们所有人，所有的感情都是一样的。
 
 
 
 
 
 
 
 
 
 
 
　＊＊＊
就这样，两年过去了— —然后、那个时刻、终于到来了。
 
今天，我也在和与我境遇相同的女性聊着天，突然，原本正在微笑的她，表情突然剧烈地变化。
「咦... ... 什么... ... ? 诶、这里是.. ... 到底是怎么... ... 我... ... 哈... ...」
「怎么了? 感觉不舒服吗?」
「这样的、这样的事情…. … 不... ... 不要! 不要啊啊啊啊啊啊啊!」
和我对话的女人突然开始发疯一般地暴走，不断的开始流下眼泪，不知道是对着谁，不停地说着谢罪的话。随后，哭泣的她，很快失去了意识。
刚才还在笑着聊天，为什么... ... ？
并不知道已经发生了什么事情的我，正处在困惑之中。
 
 
 
但是— —，很快，那个突然变成这样的理由，我也知道了。
我的内心，对勇者大人的爱，所有的痕迹都突然消失了，仿佛从噩梦中苏醒过来一般，一一直以来，所坚持的，那样污秽的价值观也是... ... 简直，就像混浊的思想被净化了一样... ... 发生了我自己也不太明白的现象。
但是，只有一件事我现在已经明白了。我，被从某种所不了解的力量中，解放了出来。这样的感觉，清楚地提醒着我。
虽然有些许放心和开放感，但是因为没有办法定下心来思考，我开始混乱起来。
诶... ... 到现在为止，我，都在做些什么？
还有，阿尔托，在哪里？你去哪里了？
我环顾四周，这里并不是我和阿尔托曾经住过的村庄，而是像我这样的村姑完全不配居住的，豪华的城堡内。
我被眼前的光景惊呆了，然后，就在这时，强烈的记忆，第二次向我袭来。
那些对我来说太过可怕，太过惨烈的记忆，以暴力的方式显现在我的脑海中。
这样的东西，我当然没有办法忍受，随后我的意识，在中途断掉了。
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
　＊＊＊
醒来的时候，我发现自己躺在一张华丽的床上。
然后，在我面前站着一位来自教会的，带着温柔表情的神官大人。调查了我的身体状况，认为没有问题后，他用严肃的声音说了出来。
「莉娜小姐... ... 根据调查的结果显示，包括您在内的，许多的女性... ... 都被勇者所魅惑。」
「诶？... ...」
「那个勇者使用的是一种叫做【伪爱之侵蚀】的技能。这是一种可怕的技能。会使其爱上并只思考对方的事，被侵蚀到一定程度，甚至会逐渐变得和对方一样，连思考模式都逐渐同化的，可怕的技能。」
「我，我... 也就是... 所以... ... ?」
「是的。就在今天，勇者在魔物的袭击中死去了，因此技能效果也随之解除，这个问题才得以解决。我用技能调查仪检查了一下，发现你的侵蚀情况尤其厉害，应该是被反复使用了多次技能，所以被影响得也特别严重的缘故。
「勇者大人... ... 死了? … …诶? … …还有、侵蚀的意思是... ...」
虽然听到神官大人的话，但是一丝实感都没有。
但是，在他解释的过程中— —我的记忆，开始慢慢恢复了。
「啊... ... 啊啊啊啊! 我，我做了... ... 我... 做了... !」
所以，我和勇者大人... ... 不，不对！是勇者— —是勇者让我做了这么可怕的事情。
最爱的，一直喜欢着的青梅竹马... ... 一直敬爱着的，我的父母.. ... 最喜欢的村里的大家
全部、全部、全部全部全部都被! ！
被我破坏掉了。
终于... ... 我终于... ...清醒了过来
虽然已经太迟了，但是我终于从魅惑中被解放了。我终于真正地，从噩梦中苏醒了过来。
 
 
 
 
 
然而，从噩梦中醒来之后，我要面临的，将是无尽的悔恨。

技能《伪爱的侵蚀》
通过特定的关键词和行为来发动，一旦发动就会发生非常恶劣情况的，魅惑技能的一种。
关键词由技能拥有者自由决定，只对异性有效。
可以累积。无条件地爱使用者的同时，如果被技能侵蚀得愈发严重，价值观和思想也会被使用者所同化。使用者是垃圾的话无疑是最糟糕的技能。
但是，因为只有人渣才会使用这些技能，所以不管怎么样也已经是最糟糕的技能。
 
 
啊，你们想问的莉娜酱纹纹身的地方— —
肚脐下面一点点哦，啊，就是那个，小孩子会从那里出来的地方，就是那里哦❤。
