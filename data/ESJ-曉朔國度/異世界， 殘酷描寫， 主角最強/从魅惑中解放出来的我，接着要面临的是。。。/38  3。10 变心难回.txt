「 ... ... 你能放开我吗?」
「不要! 听我说阿尔托! 你一定是被骗了!」
「我被骗了?」
 
 
 
很好，看来阿尔托还是会好好听我说话的。
虽然有些不知羞耻地抱住了他，但是阿尔托似乎并没有发怒，也就是说，他的心里说不定还有我，他其实还爱着我！
既然这样的话，那我应该还有机会！
 
 
 
 
「是的! 阿尔托被那个现在成为你恋人的女人给骗了!」
「 ... ... 能不能请你，不要这样随便说她的坏话?」
「不是的，听我说！！因为，她是利用阿尔托的内心受伤的时候趁虚而入的吧，像这样的女人，怎么可能值得信任呢。」
 
 
 
「…………」
 
 
 
」求你了阿尔托，相信我，相信我!」
我更加用力地抱住了他，能感受到阿尔托的体温传递了过来。
请和我的体温一起，把我对他这份最喜欢的感情，一起传达给他吧。拜托了... ... 请一定要达给他。
奇迹发生了，也许是我的愿望实现了，阿尔托把他的手和我的手重叠在一起。
 
 
 
 
「莉娜...」
 
 
 
 
「 阿尔托... ... !」
果然，阿尔托就是阿尔托。
你总是那么善良，知道我想要说什么。
你果然是我无可替代的，最重要的青梅竹马啊。
只要花时间和他好好谈谈的话，他一定会恢复正常的。
我一边感受着阿尔托的温暖，一边想着这样的事情。
但是— —这份温暖，伴随着强烈的冲击一起消失了。
 
 
 
 
他粗暴地抓住了我缠在他身上的手，把我从那具朝思暮想着的身体上一把拉开，然后重重地推了我一把。失去了平衡，跌倒在地的我，就这样屁股撞到了地上。
「痛、好痛... ... 好痛啊、阿、阿尔托... ... ?」
是明确的拒绝。
这是第一次阿尔托对我做这样的事。一直以来都那样温柔的他，会这样猛地把我推开的事，我从来没有想象过。
抬起头，我再一次目睹了他的身姿。
那是一副丝毫不为我的痛苦担心，目光冰冷地凝视着我的身姿... ...
「我简直不敢相信，这种事你都能说得出口啊?」
「诶?」
「的确，我想我也许是一个容易被欺骗的男人。要说为什么的话，那是因为我居然被一直相信着，一直信赖着的你给背叛了！」
「— —! 阿尔托，不是，不是这样的! 拜托请听我解释— —」
「我刚才听到了。被勇者大人魅惑了？是因为我一直像个傻瓜一样，所以你以为我才会相信这样的借口吗？... ... 但是啊，我当时居然还是无数次地相信了你啊。一次又一次，我一次又一次地试图相信你！但结果呢？— —我得到的结果，就是被你给抛弃！
到现在为止一直很安静的阿尔托，终于把这段感情发泄了出来。
那份向我袭来的感情，仿佛是充斥着愤怒、失望、憎恨等各种各样的东西的扭曲的结合体，狠狠地在我的心上穿刺着。与此同时，我也终于明白了。
阿尔托，从一开始，就再也不会原谅我了。
他只是假装漠不关心，其实他从心底，早已对我— —
「就这样默默回去的话，我本来不打算说这样的话。但是你，居然敢侮辱我的恋人！那位把我从绝望中拯救出来的那位女性，对她说的任何不好的话我都绝不会原谅！毕竟，她是那样温柔，那样优秀的人啊。」
如果自己所爱的人被侮辱，感到生气，也是理所当然的。
如果阿尔托被侮辱了，我也一定会生气的。
但是对他来说，那个对象已经不是我了... ... 这个无法改变的事实，让我陷入了绝望之中。
「 ... ... 那个，对不起— —」
尽管如此，我也不想失去和他的羁绊，所以我赶快想道歉。
但是，就在那时。
 
 
 
 
 
 
「阿尔托? 好久没听到你这么大声地吼了呢，是发生什么事了吗?」
伴随着平静的声音，有一位女性从房子里走了出来。
那是一位有着几乎垂到肩膀上的清爽的蓝发，散发出高贵之美的女性。
寒气，亦或是冷气？
仿佛被一股冰冷的气场围绕着一般，让人望而却步。
但只有眼睛，充满着热情与温柔，似乎是因为目光所及之人。
她用那样的眼神，交替地注视着我和阿尔托。
 
 
 
 
 
」对不起，我没事的— —凯特。」
「明明都摆出那副样子了，还强撑着说自己没事 ... ... 难道，是因为那个人吗? 她是谁?」
「 ... ... 啊，那个女人吗，她就是莉娜。那时我们一起看到的，我的那个青梅竹马。」
当阿尔托这么说的时候，那个叫凯特的女性看着我的眼神，突然变得充满了厌恶和冰冷。
简直像是，看到了什么最肮脏的东西一样的— —无比轻蔑的眼神。
「唔，那个背叛了阿尔托的人，就是你吗。」
「那个，不是这样的.. ... 我现在还喜欢着阿尔托。」
 
 
 
「哪里、不是了？ 」
 
「你居然说、喜欢阿尔托吗… …」
 
「抛弃了奄奄一息的他，就这样消失的你，在我眼里，似乎并不具备此种资格。」
 
 
「啊，呜，那、那是因为... ...」
「若真心相爱，并愿意以此互相扶持，在那种时候，不应该是呆在他的身边的人吗？在阿尔托遭受痛苦的时候，和另外一个男人一起走掉的女人— —这样的你… …」
 
 
 
「没有说这种话的资格。」
 
 
 
为什么... ...！! 明明、不是这样的。我没有，我从来没有背叛过他。
明明这不是我的意志... ... 明明全部都是勇者做的！
... ... 明明，是这样的。
 
为什么，为什么我没有办法反驳？
 
为什么，为什么我会这么心痛。
 
即使这样问着自己，我也其实早就明白了。
因为她说的，都是事实。
不管有怎样的理由，现在她说的事... ... 全都是事实。
但是，没有资格什么的... ... 请不要在这样说了！
喜欢着阿尔托，想要留在他身边，这样的感情被否定什么的，我是不会原谅你的！
只有这种感情，是绝对的真物。
被全盘否定什么的，我绝对不要！
 
 
 
 
「随你怎么说都可以... ... 但我对阿尔托... ... 我都会一直喜欢着阿尔托。」
」你是在和我开玩笑吗？刚刚已经提醒过一次，你，不具备这种资格。」
「开玩笑的人是你才对! 说起来，你又是谁啊? 不属于村里人的你，凭什么现在站在这里对我这样说话! ?」
仿佛是在互相质问骂街的言语不断升级，我最后终于忍不住对她大声怒吼了出来。但是，她的表情，并没有因此而产生一丝波澜。
 
 
 
 
 
 
「名为凯特。」
 
 
 
 
「我，就是来自王城的大祭司。「
「诶?」
「我来这里，是想在做好本职工作的同时，顺便赚点零花… …顺便做些其他工作。」
但是，从那一天起，从我第一次遇见他起… …从第一次对一个人，一见钟情的那一天起，我也变了。
即使失去了恋人，即使失去了父母，即使被悲伤所环绕，被痛苦所压倒— —但是即使如此，仍然决定顽强活下去的他，对那样的他，我早就下定了决心，一定会让他幸福！！」
那副充满了决意和自信的表情说着这番话的身姿，和满怀着了悔恨和内疚的表情的我，形成了完美的对比。
「所以— —对于背叛了阿尔托的莉娜小姐来说，让我拱手相让… …」
 
 
 
「抱歉，我是不会答应的。」
 
 
 
 
我知道，我赢不了了。
以那个拯救了阿尔托的生命，在工作上又有着很好的相性，还愿意给予他真正的，无暇的爱的人... ... 已经污秽不堪的我— —想要赢什么的，简直是在做梦吧。
我再一次看到了，那个温柔地抱着说完话的凯特小姐的肩膀的，温柔的阿尔托。
那是曾经只对着我的，看向恋人的表情。
是对于现在的我来说，永远不会做的— —最喜欢的笑容。
 
 
 
啊... ... 原来，是这样啊。果然，我已经... …
 
 
 
我能乘虚而入的空隙，才是，早就消失了才对... ..
