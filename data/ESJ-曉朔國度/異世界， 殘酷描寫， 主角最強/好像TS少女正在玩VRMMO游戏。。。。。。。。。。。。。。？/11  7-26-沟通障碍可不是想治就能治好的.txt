「樱怎么了? 」

「 ... ... 那边。」

「 ... ... 我觉得有点不太妙呢。」

「 ... ... 快去叫她」


在接待处前追上铃火后，我就让铃火回去了。

毕竟拆弹工作就要交给炸弹制造者。


「都填好了吗? 」

「好了，给你，我想另外两个人很快就会来了。」

「…嗯。」



「非常感谢。那么，等到大家都到齐了之后，我再向你们介绍协会的详细运作方式吧。」

「好的。 ... ... 对了，魔法要在什么地方才能学会呀? 因为我们是『来访者』，所以对这方面的知识一无所知。」

「「来访者」 ... 原来如此。好吧，首先要纠正一下，魔法并不是学来的。你可以去每个城市的魔法协会，并学习该协会的魔法公式。然后通过结合这些魔法公式来自己创造魔法」


是每个人都定制属于自己的魔法这种感觉吗？


「哦... ... 每个协会的魔法公式难道各不相同吗?」

「虽然有些共同点，但细节处是不同的。... 你所处的这个城市的魔法公式的独特之处就在于... ... 不行，你最好还是去魔法协会亲自看看比较好。」


亲眼看比听别人说更可靠，也更容易理解。


「也是啊，我登记完之后就去看看。魔法协会在哪里呀? 」

「请稍等一下，我给你画一张简单的地图。」


这不是超级优秀吗？那为什么这么空荡荡的时候还会待在接待处？




「久等了... 给你，这是登记用纸。」

「让您久等了... 今后也请多多关照。」

「好的，我收到了。 ... ... 然后首先是这个，去魔法协会的地图。」


接待员把四张纸放到了后面的台子上，紧接着被另一个人拿着走进了里面的房间里。

然后接待员递给了我一张手绘的地图。


「谢谢。。。哦，这个很容易看懂啊。」


这是一张只有最低限度的基本路线和标记的地图。

而且到魔法协会只需要拐两个弯。


「很荣幸得到你的称赞。那么，既然大家都到齐了，就请允许我来详细的解释一下。」

「就靠你了。」

「好的。... 首先，佣兵协会是指为佣兵们提供工作的组织。另外，我们也会把工作转给佣兵个人，但在大多数的情况下，都是会通过各个公会的。」

「各个公会? 佣兵协会还分公会吗?」

「你可以认为公会就是一支志同道合的佣兵团队。每个公会都有自己的活动方针，协会也会介绍符合这个公会方针的委托给公会。」

「有没有可能注册了的佣兵不去接受委托? 」

「是有可能，但每过一个月，我们就会收取登记费进行续期，但如果因为某些原因申请退出的话我们是不会收费的。」

「如果委托需要一个多月才能完成呢? 」

「这取决于委托具体的内容，以及接受了委托的佣兵的行动。如果你把委托放置一个月，就理所当然的受到惩罚。此外，如果委托的难度很高，或者存粹需要一个月以上的时间，则不会受到任何惩罚。」

「我可以取消承接的委托吗? 」

「你可以向工会去提出申请，但根据取消的原因，我们可能会收取一些取消的费用。」


也是，如果你因为一个非常私人的原因而取消你承接的委托，那就很让人困扰了。

我对于协会的结构如此稳固感到很惊讶。

这只是个游戏，对吧？但这一切都太真实了不是吗？


「成立公会有什么条件吗? 」

「最低人数不得少于五人。此外，我们需要对公会领袖拥有一定的信任度。」


五个吗... 照现在的样子，我们几个不能成立工会啊。

还有信用吗。是能拿到什么凭证吗？


「我需要怎样才能获得信任呢? 」

「我们会通过实际接受委托后进行的流程及其执行能力，以及对包括在委托人、街道和村庄在内的委托相关方去进行调查来确认。如果协会认定没有问题的话，就会给予证明物的。」

「证明物是什么呀?」

「是一个徽章，佩戴着它的人就代表协会认定了这个人的人品和实力。还有其他的问题吗?」


「委托是实力无关吗，如果是的话什么委托都去能承接吗? 」


委托的限制还有能否承接与本人实力不符的委托。


「在协会中公示出的委托你都可以自由的去承接。但请注意，如果因缺乏能力而未能完成委托的话，将会受到惩罚。」

「ok———我大概已经明白了。如果有不明白的地方可以再来问吗?」

「没关系的，我们随时恭候。... ... ... 这是你们的佣兵证。如果遗失的话，请立刻来协会提出遗失申请，协会会根据你的理由来决定是否进行免费发放。请注意，如果只是丢失了的话，发放费为两枚银币。」

「谢谢。 ... ... 那么，我要去魔法协会了。」


我想在承接委托之前先在魔法协会里创造出魔法。

铃火和樱可能并不需要，但对我和苏来说这是必须的。


「如果创造出了魔法的话，请再到协会来提出一下，因为每个佣兵的个人信息会与协会对其推荐的委托挂钩。」

「创造出魔法之后我还会再来的... ... 啊，对了，最后能问一下你的名字吗?」

「虽然可能有些迟了。我是奥格玛 · 卡诺普斯，是这个协会支部的事务总管。」

「事务总管? ... ... 难道是个了不起的人物?」


总管，也就是负责人，对吧？


「不，不，我并不是支部长，我最多也就是个二把手。」

「那也已经够了不起的了... ... 嘛，就这样吧。再见了，奥格玛先生。苏，铃火，樱，我们要走了哦?」


我叫了一直在后面沉默着的三个人一声，然后走出了工会。


「啊，我明白了。再见，奥格玛先生」

「那么，告辞了。」

「 ... 拜拜。」

「我期待着你们的再次光临。」


她们三个人也和他打了个招呼之后站到了我的旁边，开始一起说起这件事。


「从现在开始，这种事就全权交给你了。」

「有道理，你想问的事都已经告诉你了，很轻松呢。」

「 ... 米酱，真厉害，今后也都拜托你了。」

「 ... 你们想把麻烦都推给我吗... 」


这三个人是不是觉得跟别人说话太麻烦了... ？
