现在我应该怎么办呢？


结束了社团活动的梨香一个人走在昏暗的校舍里。


平时的话我可以通过无感情的游泳来忘掉不愉快的事情，但是今天我所见的却无法被轻易忘却。


自己的恋人和自己在男人中唯一能说上一点话的人，做爱了。


看到的时候，我理所应当的感到了愤怒与悲伤。


但是，最让人兴奋的还是那种心跳加速的感觉。




「……我是、变态？」




梨香「不不不」的摇了摇头。


虽然我的确窥视着自己恋人的不忠行为自慰了，但是现在并不是考虑这个的时候。


现在开始，自己应该用什么样的表情和他们见面呢。


荣治还好。因为只是同学关系，所以只要不再和他有关联的话就没什么关系了。


但梅不同。她和我不仅仅是同学关系还是我的恋人。我肯定会和她产生联系的。


说到底，我应该怎么办才好？


追究她和荣治进行性交的事？


还是把它隐瞒起来，继续我们之间的关系？




「（应该怎么办，我不知道啊... ...）」




梨香一个人步履蹒跚地向着校门走去... ..。




「晚好。」


「…………！？」




靠着校门站立的人，正是让自己烦恼的两个人之一的荻原荣治。




「 ... ... 干、干什么?」




梨香露出警戒的样子并向后退了几步。


虽然我认为他不会袭击我，但如果他真的这样做了的话... ... 我凭借自己的细胳膊根本无法反抗。


看到了梨香的反应之后，荣治并没有特意变换表情，只是张开了嘴。




「你看到刚才的那家伙了吧? 」


「…………咕」




刚才的那家伙。换作别人的话应该根本不会明白他在说什么吧，但是梨香可以理解。


和自己有关，还提到了刚才的事.. ... 就只有那个了吧。


她紧张地咽了一口口水。


就连游泳比赛的时候我都没有这样的紧张过。


荣治到底会说什么呢... ... 他有可能会威胁我。


想到这里，我感觉我的喉咙都已经有些发干了。




「照这样下去的话，你还是会不清楚到底发生了什么了吧?我们到附近的公园去谈谈吧。强行袭击什么的我是不会做的，这你就放心好了。」


「…………」




说完，荣治转过身去。


真的好大只... ... 我一边这么想着，一边试图寻找他的意图。


他到底在思考什么呢... ..。




「如果你无论如何也不愿意信任我的话，那就没办法了。」


「 ... ... 不，我跟你去。」




总之，我得试着跟荣治谈谈。


而且，我还必须问清楚为什么梅会被他侵犯。


因为那比起强奸更像是和奸。


为了了解这些事情，梨香不惜冒险也要去跟随荣治。






















　♡






「茶，行吗? 」


「 ... ... 橙汁。」


「没想到你也这么任性啊... ... 真不愧是她的恋人吗。」




荣治苦笑着走到了自动贩卖机前。


他们两个人来到了公园的附近。


如果是白天的话，在这里可以看到孩子们快乐的玩耍的身影，但因为已经是傍晚了，所以现在一个人也没有。


梨香从回来的荣治那里接过橙汁，然后他们两人一起坐到了长椅上。


当然，他们两者之间的间隔足以容下两个人。


他们两人都润了润喉咙，然后沉默了一会。


在沉默中，先开口的是荣治。




「然后，你想怎么做? 是由我来告诉你一定程度的事，还是你问你想知道的事情，而我来回答? 反正对我来说怎样都无所谓。」




他给了我选择。


虽然让他从头到尾解释一遍比较好，但是和荣治在一起很长时间这种事. ..。




「 ... ... 那就由我来问你吧。梅跟你... ... 做那些事，是梅的意思吗?」




她用冰冷的湛蓝色眼睛注视着荣治。


虽然从那个反应来看，这种可能性很小... ... 但如果他是强行进行侵犯的，那么梨香就会为了自己的恋人而去把能做到的事情都去做一遍。


... ...嘛，如果梅是自愿的... ... 这么一来，就会变成普通的出轨，所以她在脑内思考着如果这样的话应该怎么办。


在梨香锐利的目光的注视下，荣治并没有表现出害怕的样子，而是为难的挠了挠他的脸颊。




「啊—... ... 真难说啊。如果硬要说是哪边的话，应该算是我这边硬逼她的吧。那家伙还挺不错的。至少，她并没有想要积极地背叛你的想法。」


「 ... ... 是这样吗? 」




梨香眨了眨眼睛。


对于感情表达薄弱的她来说，这已经可以算是程度相当大的惊愕了。


这是因为，荣治说出了一些袒护梅的话语。


在这种情况下，我还以为他会把锅全都推到梅的身上来让自己看起来好一点... ..。


虽然这并没有提升他的评价或者使梨香放松警惕，但起码没有让梨香的情绪以及对他的评价恶化。




「我想鬼冢你也知道的吧，那家伙是个自恋狂。 」


「 ... ... 嗯。」




梨香点了点头。


确实，梅是个自恋狂。甚至都已经足够用超级来形容了。


自己是绝对会比其他的女孩子更可爱的... ... 她的这种意识非常强烈。


因为是她的恋人所以梨香才明白，恐怕她在可爱的层面上相当看不起梨香。


虽然因为梨香不在意这些所以没有什么特别的想法，但是，在她们的这个年纪里会拥有像梨香这样的想法的人非常少。


正因如此，许多女同学都讨厌她... ..。




「嘛，虽然外表看起来确实很棒。我想，应该没有一个男人会对她的谄媚行为，或者说是讨好行为感到不顺心吧。」


「 ... ... 嗯。」




梨香又顺从地点了点头。


作为呆在梅身边的恋人，她知道有很多男人正迷恋着她。


比偶像更可爱的容貌，以及超过写真模特的身材。


因此，如果她做了一些会让男人心烦意乱的动作的话，没有这方面经验的男生就会很容易的沦陷。


因为即使是像老师这样的成熟男性也有迷恋着她的人，所以她的魅力可以说是惊人的了。


但梨香并不理解这种被男人奉承的快乐，所以她并没有去卖弄风骚。




「然后，我却感受不到那家伙的魅力。她确实长得很漂亮。但就像我之前说的那样，我觉得鬼冢你更有魅力一些。」


「 ... ... 嗯，嗯。」




受到如此直接的赞美，就算是梨香也会脸红。


虽然她并不在意自己的外表，但她还是会因为受到赞美而感到高兴的。


而且，因为她自认为梅比自己更可爱，所以当被人称赞自己比梅更有魅力时... ... 更是如此。




「我和她一起玩的时候也是这么和她说的哦。然后，她可能也觉得在哪里吸引不住我了吧。她就把我带到她的房间里，就这样... ... 她想要用色情的事情让我沦陷。」


「梅... ...」




虽然是面无表情，但梨香散发出的氛围却变得呆滞了起来。


因为吸引不住而感到焦虑，所以才吵闹地脱掉了衣服的梅的身影在她的眼前浮现了出来。




「虽然有点对不住鬼冢，但是因为我已经忍无可忍了，所以才做了这种事情。」




荣治尴尬的把脸转了过去。


男人是用下半身行动的，虽然我不这么认为，但如果受到有魅力的女人的强烈诱惑的话，会做出这种事情也是可以理解的。




「 ... ... 那么，刚才在教室里... ... 那种事情... ... 要做的人是... ...？」


「 ... ... 是照井起的头。」


「 ... ... 这样啊?」




梨香轻轻的叹出了一口气。


这样一来的话，她就不能向荣治发泄愤怒或者去找警察控诉了。


但是，梨香心中的丧失感却并没有那么大，对梅来说也是没办法的事啊........这种感情被逐渐的放大了。




「虽然我说也没什么用，但如果可以的话，我希望你能原谅那家伙。毕竟还是向她出手了的我不好。」


「 ... ... 呵呵，我知道了。」




始终庇护着梅的荣治让梨香轻轻的笑了出来。


这是她第一次在荣治的面前露出笑容，而且，她已经很久都没有在除了父亲以外的男人面前露出过笑容了。


荣治的目光也下意识地被她美丽的笑容所吸引。


虽然这让他有点忍不住想要靠近她... ... 但现在并不是时候。


荣治很清楚这点，而且他也并不想要陷害梨香。




「哈~... ... 既然想说的话都说完了，那就差不多该回家了。我送你回去吧，因为天已经有点黑了。」


「 ... ... 嗯。」




对于荣治的提议，梨香微微的点了点头。


这是一个以前的梨香绝对不会接受的提议。连和男人一起走在昏暗的街道上对她来说都是无法想象的事情，就更别提让男人知道她住在哪里了。


但是，梨香却微微的点了点头，接受了他的提议。


他们两人一起向前迈开了脚步。


但是，他们二人之间还保持着一种微妙的距离感。


虽然不像是梨香对其他男人那样明确的表现出了警戒的那种距离感，但至少也不是像亲友那样的距离感... ..。




「 ... ... 还在害怕吗? 不对，因为我做了那种事情，所以不害怕才会奇怪吧。」


「啊... ...」




对于苦笑着的荣治，梨香感到有些抱歉。


虽然她根本没有必要这样去想，但这也体现出了她的善良。


这样下去的话，对于以善意对待自己的荣治来说是不是有点太失礼了？


温柔的她这么想到... ..。




「 ... ... 你能听我说一件事吗? 」




她决定要说出自己的秘密。




「行。」




荣治也点了点头来表示接受。
