梨香到了自己的班级之后，先是确认了贴在黑板上的学籍号然后走向了按照学籍号分配的位置。


真是的... ... 真希望梅能再小心点。


她不管怎么说都有松散或者说无忧无虑的一面。


真希望这不会导致致命的失误... ..。


嘛，只要身为恋人的自己在她的身边提醒她就可以了吧。


梨香和梅是一对百合情侣... ... 所以互相帮助也是理所当然的。


梨香一边这么想着，一边坐在了自己的座位上。


在教室里，有从去年开始就是好朋友的人，也有今天才第一次见面的人，他们都在进行着深刻的交流。


但是没有一个人来靠近梨香。


她不仅因为一直躲着男生而被认为讨厌男生，还被女生们当作是高岭之花，就因为这样所以没人想来和她说话。




「(... ... 孤独?)」




「(不不不。)」梨香摇了摇头。


我不是还有梅吗。所以应该并不孤独。


... ... 但梅有很多男性友人。反过来说的话，由于她不仅受到男生的欢迎，还喜欢装可爱，所以也有被女生们避开的一面。




「(... ... 如果可以的话，我希望不是和男生，而是和女生们能好好相处啊)。




梨香一边这样想着，一边瞥了一眼旁边。




「…………！？」




那里，有一堵肌肉墙。


是女人怎么努力也得不到的发达肌肉。


面对这从未见过的压倒性的肌肉量，平时面无表情的梨香也变得哑口无言了。


抬高视线的话，就好像是在仰望高山... ..。




「（啊，是那个人... ...）」




我还认得那张脸。


因为梨香平时并不和男生接触，所以老实说就算是同班同学的男生的脸也记不清楚。


但是，这个男人是刚刚遇到的，梨香还在想着希望有一天能向他道歉。所以他的脸梨香是不会忘记的。




「（好、好可怕... ...）」




她虽然不喜欢瘦弱的男生，但还是更不喜欢这样壮硕的男生。


但在善良的梨香的脑内没有去不道歉这一选项。




「那、那个... ...」


「嗯？」




所以，梨香只能去跟他打招呼。


虽然发出的声音非常微弱，但是他还是立刻的转过身来。


面对着这样严肃的脸，梨香纤弱的身体忍不住不停的颤抖着。


深吸一口气，让心情平静下来。


　没问题、没问题……。




「啊，那个... ... 刚才真是对不起了... ...」




梨香微微点头，以表歉意。

真是个严肃的男人。也许他可能会要求我做一些荒唐的事情。


梨香第一次感受到如此的恐惧... ..。




「啊，别放在心上。 ... ... 话说回来，刚才你不是也道歉了吗?」


「诶，啊... ...」




男人并没有像梨香所想象的那样回答，而是用女人不可能会发出的浑厚声音回答了句体谅梨香的话。




「我在意的是你的身体。如你所见，我的身体倍棒，所以你没有受伤吧?」


「 ... ... 是的，没受伤。」




梨香点了点头。


什么啊。虽然是个男人，但他挺正经的啊。


我想了很多，有点失礼了。




「是吗? 那就好。」


「 ... ... 那个，你有没有什么东西想让我当作道歉礼物送给你？。」




梨香之所以这么问，一方面是因为她还对这个男人怀有罪恶感，另一方面是为了避免以后被他用这件事来勒索。


的确，这男人很关心我，但我对真正的他还一无所知。


那么，我最好还是把顾虑都排除掉比较好。




「不，我不需要... ...」


「 ... ... 我想要送。」




男人因为为难而明显的皱起了眉头。


但梨香却还是紧追不舍。... ... 当然，因为她很讨厌男人，所以在物理上并没有对他紧追不舍。


男人思考了一会儿... ... 点点头。




「这样啊。那么，别再对我说敬语了。」


「诶……？」


「我和你是同级的，而且还是同班同学。从现在开始一整年都会在一个班级里，如果还说敬语的话会不会很奇怪?」




梨香瞪大了双眼。


因为她讨厌男人，所以她并不会主动接近男人，而男生们也都对此心知肚明，所以也不会主动来接近她。


因此，平时就算使用敬语也很少会和男生进行交流，所以梨香觉得这没什么大不了的... ..。


但她并没有想到他会要求她更改对他的措辞，而不是要金钱或者物品什么的。这让她不由得「噗呲」的笑了起来。


如果能有幸看到平时面无表情的她绽放笑容的话，不论男女都一定会为之倾倒的。




「 ... ... 嗯，我知道了。」


「啊，这样就行了，今后请多多关照。」




看到点了点头的梨香之后，男人也露出了亲切的笑容。


虽然他的表情很严肃，但梨香却觉得他的笑容很孩子气。


也许是因为心不在焉，这让她平时绝对不会说的话也脱口而出。




「……名字」


「嗯?」


「我们都还不知道对方的名字对吧? 如果是同学的话，我认为知道名字会比较好。」




听了梨香的话，男人「啊」地点点头。




「我是荻原 荣治，多多关照。」


「 ... ... 鬼冢 梨香。也请多关照。」




说着，他们互享微微一笑。


果然我还是不喜欢男人，但是如果是和这个男人---荣治的话我还是可以比较正常的来接触的。


... ... 当然，我并不会像和梅那样和他成为亲密的朋友。




「噢噢噢噢噢噢噢噢噢噢噢噢! ! ! 」




空气正平静的流动着，但就在这时梅突入了进来。


为了停止那迅勐的速度而发出了「滋滋滋」的刹车声。


她介入到了梨香和荣治之间。




「不要和我的梨香酱这么亲密啊! 」


「 ... ... 这家伙是谁? 」




她摇晃着丰满的胸部进行抗议，紧接着马上抱住了梨香。


荣治看着她，皱起了眉头。


梅因为深受男生的欢迎，所以已经构建起了偶像般的地位。居然有男生会对梅露出烦躁的表情，这让梨香很是吃惊。但是正压在她脸上的巨乳已经快要让她窒息了，所以她决定先把梅扯下来再说。
