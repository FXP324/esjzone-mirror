「你这家伙在说什么? 」


「 ... 你在说什啊，梅? 」




面对荣治和梨香呆滞的表情，梅依然露出了一脸自信的笑容。


特别是梨香现在露出的这个表情，可以说是非常稀有了，但身为她恋人的梅却因为正闭着眼睛，挺着胸膛，所以并没有注意到。




「所以说我才要把我的魅力教你这个大猩猩。」


「我才不是大猩猩」




你对荣治有点太过于刻薄了吧，梨香这么想到。


因为梅和自己不同，她不仅对男生很温柔还会像朋友一样的去接触，所以像对荣治一样去严厉的指责别人这种事情是非常罕见的。


如果荣治把他的愤怒转到我身上的话... ... 我应该会感到非常害怕的，但他看起来像个沉稳的成年人一样，所以应该不会去使用暴力的。


想到这里，我感觉梅更加的像一个孩子了，作为恋人我真的感到十分抱歉。




「话说回来，魅力应该怎么去教?」


「哼哼~我会用一种特殊的方式去把一个不懂事的大猩猩教会的!」


「 ... ... 特殊?」




荣治和梨香疑惑的歪着头。


面对这样的他们，梅先是哼哼一笑，然后「乒」的用手指指着荣治说道。




「没错! 我要给大猩猩一个可以和我在放学后一起玩的机会! 」


「不需要。」


「！？」




是即答。


对此，梨香虽然也感到有点惊讶，但梅显然感到更加的惊讶。


她在男生面前只会展现出可爱的样子，但是现在的她把眼睛和嘴巴都张得大大的，看起来非常的不雅。


然而，因为梅被巨大的冲击袭击了所以………



「你，你不对能和我一起去玩感到惊讶吗... ... !！? 只要是这个学园的男生，他们都会感到非常高兴的! !」


「所以说，你对我而言一点魅力也没有啊。」


「真让人火大大大大大大大! ! 不可原谅! 我绝对要让你和我一起去!」


「唉... ...」



梅正满脸通红的像发狂一样挥舞着双臂。


荣治的脸颊抽搐了起来，他平时虽然没有到梨香那种程度但也不怎么会表现出感情。


这样的他所寻求帮助的对象理所当然的就是身为梅的恋人的梨香了。




「喂，鬼冢，你是这家伙的女朋友? 还是男朋友? 对吧。 快帮我想想办法。」


「 ... ... 如果梅变成这样的话，她就不会再听别人的话了。」


「 ... ... 作为一个人怎么会这样啊? 」




荣治呆呆的叹了口气。


梨香也只能露出苦笑。即便如此，她还是喜欢着梅，因为爱情是盲目的。




「欧拉! ! 你在和我的梨香酱说什么话啊! ? 赶紧过来! ! 」




不知道什么时候站在了教室门口的梅正对着和梨香说话的荣治大吼大叫。


荣治对着这样的她投去了麻烦的目光。




「 ... ... 我知道这句话可能是多余的，但你最好再考虑一下和那家伙的交往吧。 」


「 ... ... 因为梅也是有魅力的。」


「这样啊?。嘛，鬼冢感觉不错的话那就没什么关系了吧。」




这么说着，他把文具放进自己的包里，并把包扛了起来。


虽然说了那么强硬的话，但他好像还是打算去陪梅的。


因为这份温柔梨香眯起了眼睛，并想要目送着他离去。


但因为看到梨香想要留在这里，所以荣治用惊讶的目光看向她。




「啊? 鬼冢不来吗? 作为女朋友? 还是男朋友? 你不介意我和她独处吗?」


「 ... ... 那个，和男人一起的话有点。」


「啊啊，这样啊」




荣治也能从她对自己的眼神和举止中看出她总是避开男人。


不，也有可能她是在害怕着自己，但是既然她没有直接的说出来那就这样算了吧。




「 ... ... 而且，我也不认为荻原君会做出那种事。」




梨香淡淡一笑，接着说道。


那种事情... ... 也就是说，男女之间的性行为。


梅作为自己的恋人，当然不会把男人作为恋爱对象，但是如果是要强上的话，就算在男人中也很强壮的荣治应该可以轻而易举地压制住梅。


本来是不能让梅和荣治单独出去玩的... ... 但是荣治的温柔触动了梨香，这让她可以信任的让他去和梅一起玩。


她也希望自己的恋人能和男人中唯一可能成为朋友的他成为好朋友。


但是... ..。




「 ... ... 这样的话，会怎样呢? 」


「诶……？」




因为荣治喃喃自语的话梨香并没有听清，所以她反问道。




「欧拉拉拉拉拉拉拉! ! 我叫你赶紧过来! 」




但是梨香还没来得再听一边这句话，梅的怒吼声就又传了过来。


梨香也知道她是那种如果事情不按照自己的意愿发展，就会变得暴躁的人。


如果我再继续留住荣治的话，他和梅就不可能成为好朋友了。




「那么，回见。」


「 ... ... 嗯。」




于是，梨香目送着荣治和梅的背影离开。


但在梨香内心的深处，还怀有着一丝的不安。

「 ... ... 梅到底怎么样了啊?」




下一周的周一，来到天童学园的梨香喃喃自语道。


自从上周五她和梅分开之后，就再也没有她的消息了。


平时的话只要一到休息日，梅就会打电话邀请梨香去约会，但是这个周末却没有发生。


不过，如果梨香主动去联系她的话，也会好好的被回复，所以梨香并不是特别担心... ..。




「 ... ... 啊，梅。」




这么想着的梨香用眼睛捕捉到了梅。




「啊—，天哪! 你在开什么玩笑啊! 」


「难道不是你先挑衅的?」


「虽说如此。啊--，真让人火大。」


「你可真是乱来... ...」




梅正和荣治在一起。


虽然这件事这让梨香的心里有些忐忑不安，但他们似乎并没有表现出特别亲密的样子，反而营造出了就像前几天刚刚分手一样的恶劣气氛，不知为何这让她松一口气，紧接着她走近了两人。




「 ... ... 梅，荻原。」




梨香刚一出声，发出「咕噜噜噜噜」的低喘，就像受伤的野兽一样瞪着荣治的梅突然绽放出了非常可爱的笑容。




「啊~! 是梨香酱!」


「哦。」




梅扑向了梨香，而梨香则设法接住了她。


由于她丰满的乳房和臀部，要支撑住比梨香体重更大的她是一件困难的事。




「请听我说啊，梨香酱! 这只猩猩真的很让我火大! 」


「你这家伙真敢说啊」


「 ... ..你们就不能成为好朋友吗? 」




我抱着她，并用力地揉着她的头，而她则开始开始滔滔不绝地说起了荣治的坏话。


荣治也对她无奈地叹了口气。


和前几天真的没有什么不同。明明他们应该一起出去玩过了，但似乎一点都不亲密。


梨香虽然对此感到安心，但还是再问了一次... ..。




「当然不可能的~。这样的大猩猩和可爱的我是不可能会般配的。」




he~tui~，如果这里不是教学楼的话，梅看起来就像要吐口水一样。


明明她在公共场合都会去扮演一个闪闪发光的可爱的照井梅，但是现在她的感情却坦率地表达出来了。




「 ... ... 荻原君呢，你明白梅的魅力了吗?」


「不，一点也没有。我都不知道为什么鬼冢会和她去交往呢。」




为了慎重起见，我去问了问荣治，但他的回答还是那么的尖酸刻薄。


看来，他们还是没能成为好朋友。




「 ... ... 不对，也是啊。虽然魅力不怎么强... ... 但我也知道了她还是有些优点的。」


「…………咕！！」


「 ... ... 是吗?」




荣治走近了之后，改口说道。


梨香本可以去和他保持距离的，但因为被梅抱住了，所以动弹不得。


虽然全部被他明白了的话会有些困扰，不过梨香也希望荣治能多少理解到一点梅的优点，所以梨香内心还是感到很开心的。


... ... 但不知为何，抱住了自己的梅突然变得安静了下来。


看到她开始微微的颤抖，这让梨香感到很惊讶。




「 ... ... 梅?」


「啊—梨香酱! 说起来，河村老师在叫你来着!」




我正要问她发生了什么的时候，她却猛地抬起头来，对我说道。


她的脸有些泛红，是因为和自己拥抱吗？




「 ... ... 河村老师? 我知道了，谢谢。」




「叫我有什么事呢?」我这么想到，但如果有人叫我的话，我就必须得离开了。


我向梅道谢后，向荣治轻轻的点了下头，然后就离开了他们。




「 ... ... 请不要在梨香面前碰我! 」


「 ... ...不在鬼冢面前就没事了? 」


「 ... ... 这是诡辩! 」




因为渐行渐远，我逐渐听不到他们两人的对话了。
