「 ... ... 我，讨厌男人。 ... ... 比起说是讨厌，倒不如说是难以应付?」




梨香一边缓慢的行走着，一边说起了话。


因为除了他们以外，街道上的行人非常少，而且梨香说话的声音也很小，所以是不会被别人听到的。


她也不想在这里呆到太晚，于是他们就边走边聊了。


讨厌男人。虽然这么说了出来，但梨香却在回味了这句话之后，摇了摇头表示不合适。




「 ... ... 我很害怕。」




没错，害怕。这可能就是最合适的词语了。


讨厌，不好应付，这些都是副产品。


最先出现的是，她自己对男人的恐惧。


为了不让那些可怕的存在靠近自己，所以这让她产生了厌恶以及难以应付的意识。




「 ... ... 以前，我一个人走在放学回家的路上的时候，突然遭受了袭击。」




这大概就是梨香会对男人感到恐惧的主要原因了吧。




「 ... ... 就像刚才去过的公园一样的地方。当然，因为搬家了，所以并不是同一场所。」




梨香回想起了刚才和荣治进行对话的公园。


那是一个比这个公园还要小一点的小公园。




「 ... ... 我被带到了那里，并被压倒在灌木丛中。因为那里本来就人迹罕至所以就算我大喊大叫，可能也不会有人来帮助我。」




梨香被从背后突然的控制住了身体，然后被带到了那个公园里。


这件事发生得太快了。到底有没有人在，她也已经记不清了。


而且，这么快的速度，不仅别人注意不到，自己也没来得及发出求救的声音。


... ... 不对，梨香摇了摇头。




「 ... ... 不对，已经发不出声音了。男人的力量太大了... ... 根本抵挡不住。如果被捂住嘴的话，就一点声音也发不出来了。」




她回忆起了当时发生的事情。


虽然现在的梨香也不算胖，但她还是小孩的时候的身体更加纤细。


如果这样的孩子被一个成年男性控制住了的话，会束手无策也是理所当然的。




「 ... ... 庞大的身躯把我罩住，我根本爬不起来。衣服被扯掉... ... 哭也好，尖叫也好，都没有用... ...」


「这样啊。果然，你会讨厌男人是有原因的。」




面对梨香的坦白，荣治并没有说些无聊的安慰般的话。


因为这些话她的父母和友人应该早就说过了吧。


所以，他只要接受梨香鼓起勇气的坦白就好了。




「 ... ... 但是，在被做到最后之前，我被救了。因为救了我的那个人是个男性，所以我也知道我讨厌所有男人这件事很奇怪。」




自己... ... 当时的鬼冢梨香，什么也做不到。


尽管如此，她还是在没有被侵犯的情况下度过了难关，这是因为有人帮助了她。


听到这里，荣治多少有点瞠目结舌。




「这样啊。那真是太好了。明明人迹罕见却还能发现你。」


「 ... ... 嗯。虽然我有点记不清了，但救我的人应该不是一个成年人。一个男孩？我感觉是... ... 但因为被一直捂着嘴，所以当时有点头晕目眩... ..」


「是吗。」




由于当时被压制住了，再加上这是梨香不愿回忆起的记忆，所以她对当时的事情记的已经不是很清楚了。


但她还是隐约记得，帮助自己的并不是一个成年男人，而是一个和自己年纪相仿或者稍大一点的男孩。


一直压制着自己的男人随着一声巨响无力地倒下。


虽然梨香正闭着眼睛尖叫，但是，有一个男孩拉着她纤细的胳膊把她从男人的身下救了出来。


当时的我连一声谢谢都没能对他说出来，现在的他在干什么呢？




「鬼冢，讨厌男人... ... 不，应该说是难以应付吧。你难道不想治好吗?」


「 ... ... 应该，是想吧。」




荣治的话让她的意识浮现了出来。


应付不来男人。因为在今后的生活中是不可能不跟男人扯上关系的，所以一直这样的话就有点麻烦了。




「那么，和我一起去玩怎么样?就我们两个人。」


「 ... ... 和荻原君?」




虽然梨香也想治好，但荣治的提议还是让她感到了惊讶。




「是啊。就像是我和照井一起玩的时候的那种感觉。不能一下子消除掉的话，慢慢淡化不就好了?」


「 ... ... 这就是和荻原君一起去玩的理由吗?」


「是啊。当然，如果你讨厌的话那就这么算了吧。光是把我当成同班的男生说说话就好了。」




听到了他的话之后，梨香沉思了一会儿。


虽然梅和荣治有过不忠的行为，但现在最能扯上关系的男性依然是他。


不会被强行侵犯，而且还因为说出秘密而拉近了心灵上的距离，所以梨香... ..。




「 ... ... 不，这件事我可以拜托你吗？」


「好的。」




她决定要接受荣治的提议。


就这样，他们约定好了几天之后的初次约会。
