「 ... ... 哈~... ...」




刚从泳池中浮出的梨香大口的呼吸着氧气。


变得更加水润的嘴唇吸引住了男人们的目光。


因为平时都是戴着泳帽的缘故，所以能看到金色的头发漂浮在泳池中到底是多么不可思议的光景啊。




「喂—!」


「……？」




回头一看，荣治正奋力的向这边游来。




「不是，先等一下。你大概没有用全力去游吧。不过，如果你这样潜进去嗖嗖的游就有点麻烦了。一般人可追不上啊。」


「 ... ... 对不起?」




确实，仔细一看的话，荣治的样子也多少有些疲惫。


虽然梨香一点事情都没有，但她毕竟是在游泳部中取得了优秀成绩的实力派人物。


如果是这样的她的话，即使没有认真的去游，也会和荣治拉开很远的距离。




「姑且我也是说过了要缓解你的厌男心理。所以能尽量和我呆在一起吗?」


「 ... ... 嗯。」




梨香有些心情低落。


因为荣治不想让她露出这样的表情，所以他开始绞尽脑汁的想打破僵局。


需要采取与梅明显不同的应对方法。


嘛，也许是因为她有被虐癖的原因，所以她只要被她看不起的男人支配就会感到兴奋，但是如果她看到了荣治因为梨香而慌慌张张的样子的话，可能会发自内心的感到生气。


但荣治也应该会熟练的搪塞过去吧。




「哦，对了。你想去那里玩吗? 」


「.........诶?」




梨香向着荣治所指的方向看去。


在那里，有着作为这个室内水上公园最大卖点而存在着的超大型滑水道。






















　♡






「……真高啊」




正在排队等待着滑水道的梨香一边从楼梯上向下面看去，一边发表着感想。


是可以理解会受欢迎的高度。


当然，不仅设计的很完善，安全也有保障。但是恐高症的人应该不敢向外边看吧。




「 ... 为什么一直看着上面? 」


「因为，男人就是要向上爬的啊。」


「…………？」




虽然一个很明显有恐高症的人就在她的旁边，但她却只是歪了歪头。


与此同时，队伍也在不断前进着，终于轮到梨香和荣治了。




「 ... ... 你先好吗? 」


「当然。」




说完，梨香就想先上前... ..。




「啊，是情侣吗—?     这样的话，请使用这个双人座椅的—。」


「……诶？」




不知道在看到他们二人之后误会了什么，工作人员微笑着引导他们。


而在二人因惊讶而瞪大眼睛的时候，他就已经换成了双人座椅的类型。


这是一种会让身材庞大的荣治坐在后面，而身材纤细的梨香则坐在荣治的两腿之间的形状。




「……咕！！」


「没，没事吧? 」




梨香的身体颤抖了起来。


因为差点被男人侵犯的经历，她变得厌恶男人。


当然，因为只要和男人接触，她就会想起那段记忆，所以对她来说不可能没事。


她想要推开荣治逃离现场，但在此之前，荣治先担心的拉住了她的手。




「诶............?」




就在这一瞬间，梨香的脑子里有一道光闪过。


「刷」的一下子浮现出来的，是过去的情景。




『没事吧？』




说出了这句话，并向我伸出手的男孩。


小时候的梨香，泪流满面的抓住了那个男孩的手。


即使都是男人，但他给我带来的安全感与温暖是那个压制住我的男人根本无法比拟的. ..。




「喂，真的没事吗? 」


「……嗯」




梨香被「啪」的一声拉回了现实。


在她面前的是一个露出了严肃表情并忧心忡忡的看着她的男人。




「 ... ... 应该没问题。」




做不到和男人接触。如果精神创伤被唤醒的话就会变得浑身无力，汗流浃背。


但是... ..。




「 ... ... 是你的话，应该没问题。」




梨香一下子就靠到了荣治厚实的身体上。


和男人进行身体接触，而且还毫无防备的露出后背，这是我无法相信的一件事。


但是，在那本该被遗忘的过去闪回了之后，不知为何，我感觉不到对荣治的避讳感。


不如说，是安心吗? 这真是太荒唐了。




「这样吗。 既然鬼冢觉得可以的话，那就这样吧。」




荣治也没有拒绝走来的梨香。


也许是因为游过泳的原因，梨香经过锻炼的紧实屁股贴到了他的身上，这让他也感到很愉悦。


就这样，两个人一起滑下了滑水道。




「咕噢噢噢噢噢噢噢噢噢噢噢噢噢噢噢噢噢噢噢噢噢噢噢噢噢噢噢噢! ? 」


「…………」




有恐高症的荣治，理所当然的会害怕猛地滑下去。


他会主动提议去滑滑水道，也许是他想用自己的方式给梨香加油也说不定。


梨香小幅度的回头看了看荣治的样子。


这是为什么呢。仅仅只是看着他的脸，我就会萌生出安全感，而且，心中就像是点了一盏灯一样温暖。


结果，最后在不知道理由的情况下，他们一起掉进了泳池里。


一下子冷静下了头脑的梨香，完全没有注意到自己刚才脸颊微微发红，双目湿润的样子。






















　♡






「 ... ... 我去买点喝的。你要什么?」


「 ... ... 橙汁。」


「真坚定啊。」




显得有些憔悴的荣治在听到梨香的回答之后笑了出来。


而梨香则一边微笑地目送荣治拿着钱包走出去的背影，一边为了休息而爬上了岸。




「 ... ... 真好玩，呢。」




她伸着懒腰小声嘟囔道。


形状优美的乳房隔着泳衣被衬托了出来，这吸引到了男人们的目光。


真有趣。除了游泳以及和梅的约会之外，我从来没有想过会感受到这种感觉。


不对，虽然我认为在游泳池里我会稍微开心一点，但和荣治聊天也会让我感到开心，这就让我意外的感到不明白了。




「 ... 那个，到底是怎么回事? 」




在与荣治接触的瞬间，梨香脑内出现的是过去记忆的闪回。


而且，本该被遗忘的记忆也被唤醒了。


为什么会这样？难道... ... 想到这里，梨香摇了摇头。


现在不是考虑这个的时候吧。




「 ... ... 如果我说我想再去滑一次的话，他还会陪着我吗? 」




想起荣治那恐惧的样子，梨香的嘴角微微上扬了。


我从未想过我会能做到为男人的事情而微笑这种事。


在这样平稳的心情包裹着梨香的时候... ..。




「你好~「


「呐呐。现在一个人? 有空吗?」




就像要把她的这种心情打消掉一样，两个男人走了过来。


是两个看起来就很轻浮的男人。


对梨香来说清爽系她都不喜欢，这样的容貌就更加不行了。




「 ... ... 不是一个人。和男人一起... ...」




这样的话，大多数的男人都会退缩。


听梅这样说过的梨香，用这种方法果断的拒绝了。




「啊~... ... 是那个大块头吧? 不过，现在你只有一个人，所以没问题吧?」




... ... 虽然那么做了，但遗憾的是他们可不是那种轻易放弃的男人。


其中一个理由就是因为梨香的外貌。


端正的五官以及被包裹在泳衣中的苗条身材，这些都是很少见的东西。




「蛤... ... ? 那个... ...」


「喂。就别管那家伙了，来和我们一起玩吧。」


「我们经常来这里，可以教你更有趣的玩法哦~」




对着因没有赶走他们而惊讶的瞪大了眼睛的梨香，男人们越说越起劲。


但是，在听到贬低荣治的话语之后，梨香的眼神立刻就变得冰冷起来了。




「 ... ... 不用了，谢谢。」


「诶~。 和那家伙一起玩不会感到无聊吗? 他看起来就像是只大猩猩一样。」


「没错没错。大猩猩可不会懂人类的游戏哦。」




他们「哈哈」的嘲笑起了不在场的荣治。


梨香一边想着「以后不能像梅那样说话了」，一边生起了气。


昨天之前，即使心情不好她也不会生气，但因为之前的经历，她已经变得对荣治相当有好感了。


正因如此，虽然这既不是她的风格，她也能理解这并不是上策，但她还是开了口。




「请不要说他的坏话。我感到很开心，心里也暖暖的。... ... 我认为他和你们不一样。」




梨香那冰冷的眼神和话语让男人们不由自主的后退了几步。


... ... 但是，接下来他们就因为被一个较弱的女人吓到而感到愤怒。




「切。诶~，是这样吗」


「所以说啊，你要和我们一起多玩玩才能进行比较啊。」




男人们的声音变得有些低沉。


果然还是难以应付。这样想着的梨香刚要逃跑... ..。




「啊... ... !」



突然间，胳膊被抓住了。


男人的力量。就算是运动社团的，因为不会格斗技，所以还是敌不过。


而且，他们还向我投来了饱含欲望的目光。


这让我想起了那个把我拖进公园里施暴的男人... ..。


梨香的身体变得无力。


男人们误以为是她愿意了，但实际上，她只是因为过去的创伤被唤醒，所以吓成了一团。


好不容易因为和荣治的交流而开始变弱的厌男情绪就像变硬了一样，再次沉入无底的沼泽当中。


一旦沉到底部，就再也无法浮起并变得淡薄了吧。


梨香湛蓝色的瞳孔中的光芒消散了，身体也变得摇摇晃晃。


就这样，她又加强了对男人的避讳以及厌恶，她变得再也无法和男人互相了解了---------。




「不是不是。这种事情没关系的。」




但是，有一个男人把这样的梨香拉回了现实。


他发出低沉的声音，紧紧的抓住了抓着梨香手臂的男人的手臂。




「啊？」




男人之所以发出这样的声音，是因为那条手臂比自己的粗壮的多的缘故。


接着，他转过头，看到了一个大块头卸下严肃的表情并露出笑容。




「你好啊。这里是大猩猩。」
