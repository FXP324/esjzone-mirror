第一部分 被邀请到巷子里深夜 。
我离开客栈上街去了 。
没睡好是因为平时是夜猫子吧 。
相反健治做完明天的准备就一瞬间睡着了 。
一定累得一塌煳涂吧 。
我也很累，但不知道为什么眼睛很清澈 。
城塞城市格奥尔良和他们是大城市 。
晚上到处都亮着酒馆的灯，路上都是醉汉并肩唱歌 。
有个男孩给那些醉汉拉客，一个了不起的大叔和一个年轻女人挽着胳膊走着 。
这里是站前的繁华街道吧 。
对我来说很熟悉 。
因为原来老家就在那种地方 。
所以，大致知道去什么样的地方有什么样的人 。
这不是能力或技能，而是多年经验带来的坎 。
无论是外国还是异世界，城市的气氛都不变 。
好家伙，坏家伙，这种人的味道也一样 。
「狒狒……！喂哥哥，怎么了」「你迷路了吗？」所以我敢于去那些家伙所在的地方 。
华丽的商店背面，死胡同 。
第一次来这个城市，装作分不清左右的旅行者——不，真的分不清左右 。
总之，在做那种头脑似乎不好的动作的同时，不小心误入歧途 。
「哥哥，你穿得真少见啊」在小巷里的是四个男女 。
前面三个人，后面一个人 。
每个人都有武器，戴着皮革盔甲 。
盔甲下面也是皮革制品吧？虽然看到这家伙总觉得脏，但比现在我的衣服好 。
「别跑」像切断我的退路一样站在背后的男人 。
拔出刀，直截了当地说事情 。
还是习惯了 。
比起威胁的话，我知道只要出示刀具就能传达 。
「果然在这边的路上是正确答案 。
真讨厌」太目的性了，令人捧腹大笑 。
但是，没什么我能预料的 。
聚集在不受欢迎的地方的家伙们，如果被谁看到的话，就想做糟糕的事 。
即使我不是故意路过，也一定是在袭击别人吧 。
「嘿，哥哥，你听到了吗？你会说官方语言吗？』从背后走过来一个刀子男人 。
我看着那家伙，轻信了 。
博加德冒险者LV9  HP77/80 MP9/11级9吗？比我高四个啊 。
值得特别指出的地方有那么多吗？看起来像打了一百发卖不出去的艺人一样的假发 。
你看起来不像过着幸福的人生 。
接着站在正面的那些人 。
基因组冒险者LV9  HP81/81 MP0/0唐宁冒险者LV7  HP70/71 MP30/32帕梅拉冒险者LV8  HP71/77 MP24/24全员，水平都比我高吗？原来如此，也就是说，如果是四个人一起袭击普通人的小流氓，就是这种程度的水平 。
我总算理解了这个世界上普通人的强度。
也就是说我还是普通人水平啊 。
相反健治很强 。
这样的小流氓一只手就能把它拧下来 。
但是，我也不是不能战斗 。
「喂，哥哥，听到了吗——啊！」抓住靠近脸的刀子男人的下巴 。
「你啊……刚才就口臭了」
「放手……！」想把手中的刀刺进我的小流氓，但那只手停下来了 。
刀扑通一声掉在地上 。
我一松手，流氓当场屁股就摔在了地上 。
「啊……」我一边看着坐在那里什么也做不了的男人，一边捡起他的刀 。
嗯，不是磨得很好吗？这样就得到了一件武器 。
与普通的小刀不同，战斗用的小刀在刀店也很贵吧 。
刀钱涨了啊 。
「嘿！你……你在干什么！？」三人组吼的更大声 。
「魔法……不是吧！？」
「大概是这样吧」我也不知道这是不是魔法 。
我想大概是魔法的一种 。
我对礼物的详细原理不感兴趣 。
能用那个就足够了 。
我又看了一遍刀子男 。
博加德冒险者LV3  HP1/12 MP0/1虽然没死，但已经不能战斗了吧 。
因为我吸取了6级的经验值 。
看我的状态，等级从5上升到8 。
一下子上升了三个等级，很好 。
正因为是低等级的恩惠吧 。
「好吧，下一次是什么时候？来吧？」使劲动手指挑衅 。
「……混蛋！」秃顶的大叔拔出长剑向我砍来 。
我躲不开那把剑，肩膀被砍了 。
「说啊！」就算是游戏的世界，我也是活生生的人 。
被砍的话会流血，继续流血的话会死 。
但是——「怎么会！？」和刚才的男人一样，我抓住剑客的脸 。
于是，我感觉到索茨的力量通过手流入了我 。
疼痛越来越少，伤口愈合 。
不久伤口完全愈合后，我就放手了 。
倒在地上的大叔痉挛得动不了了 。
唐尼松冒险者LV1  HP2/10 MP0/1这就是我的礼物——「吸魔」的力量 。
果然，街道附近的弱怪物和拥有如此强大力量的人能吸的HP和经验值的量是不同的 。
明明吃得饱饱的，什么都进 。
桐谷娜奥特LV11  HP90/90  MP33/33，等级11吗？这样就比这些流氓们强了 。
好吧，接下来——「嘘，好吧！什么，这家伙，笨蛋！」扔掉手中的小刀，剩下的男人就把我撞倒逃跑了 。
我想追，但是算了了。
比起那家伙的经验值，掉下来的刀更有价值吧 。
这样就动不了的小流氓有两个人 。
两个都口吐白沫晕过去了 。
不仅仅是武器，衣服也可以吗？我和健治正好两件衣服 。
为了在异世界生活，这件衬衫、裤子和鞋子很难移动 。
那么，还有——「——」帕梅拉冒险者LV8  HP71/77 MP24/24，最后留下的女人 。
