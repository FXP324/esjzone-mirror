魔王周围聚集了黑色的气息 。
强大的气场，足以把雪白的房间染成黑色 。
快要被吸入的气势凝结并被他的身体吸入 。
「退后！」健治伸出手来庇护我们 。
同时拔出了圣剑 。
就像抹去黑色气味一样，产生白色气味 。
彼此的气场在魔王城内部混合，相互争夺 。
明明还处于对峙状态，好像已经在外面战斗了 。
「虽说是被创造的存在，但我也有生的执着 。
即使这种存在是上帝游戏的棋子，也有生存的权利- -」「对不起，我们也有战斗的理由！」摆好圣剑，健治开始跑了 。
「「速度提升」！「连环六合剑」！」从上段、中段、下段同时送出，共计六次斩击 。
健治留下流星般的气息，以最大技能砍向魔王 。
那一击试图斩断魔王的身体——「啊！』被像屏障一样的东西反弹了 。
「该死！」健治的身体就像被退回的速度一样飞走了 。
躺在白色地板上，不插头站起来，再次砍头 。
「哦，哦！「三倍化」！「硬克伦特」！」积蓄了力量的圣剑的斩击 。
但是它也会被魔王的屏障挡住 。
健身剑像敲橡皮球一样被弹，向后拉 。
「为什么……圣剑应该能斩魔王……」「没错」魔王也不否认 。
「所以编织了防止的技术 。
花了几千年时间建造了防止人类所有攻击的防火墙 。
这不是为了延长我的生命」「什么……那个……防止人类的攻击」的队伍吗！我本想说多么卑鄙- -，不是吗？「被这堵墙救了三次 。
像你们一样踢散了三个以我生命为目标的来自异界的勇者 。
两者都是为了生存而必然的战斗」，如果有弱点，当然要弥补这一弱点 。
魔王方面不能采取对策，就算是「神」也没有决定吧 。
如果说还有圣剑，就能顺利获胜的「神」，这样的该死的游戏一开始就没想到吧 。
「健治先生！」「师傅！」艾莉亚和尤诺跑到倒下的健治 。
「嘿嘿……没事，因为没有受伤」起来的健治没有最初那么大的力量，都是心理的错 。
圣剑不起作用的话，我在想怎么赢 。
但是他没有那个技术吧 。
「艾丽尔！尤诺！拜托他！」所以我叫 。
接下来提倡技能名称 。
「影子海德！「印象」！「镜像漫游」！』暂时看不见我的身体 。
更有消音效果 。
即使攻击不起作用，这边的抛光也对付不了吧 。
「『速度提升』！」使用加速的技能，我悄悄地靠近魔王的背后 。
偷偷摸摸也是全速 。
在转到小椅子后面的地方——「明白了，你的位置也是」魔王把手指对准我 。
从那个手指的前端发出苍白的光束，穿透了我的头骨 。
「你花了！」但那是残像 。
尤诺前几天记住的，发出重重残像扰乱敌人攻击的技巧 。
记住那个是「吸血鬼」的礼物，是正确答案 。
我对魔王薄壁后，会进一步提倡技能 。
「召唤！」手掌尖的空间出现红色魔法阵 。
从发光的圆圈中出现的是——蓝色皮肤的魔族 。
「: :」！「死……！」只露出上半身的家伙，费德尔林茨的术式束缚了魔王的身体 。
我用这家伙的技能夺走了，这家伙自己的权利 。
如果叫的话，我只想到随处可见的手机「鸿沟」程度，这样也有用 。
「哦……原谅我，父亲！」费德尔林茨这么说，退缩到魔王阵中 。
但是毕竟100级，魔王的动作停止了，真是一瞬间 。
但是那一瞬间就好 。
因为我已经在魔王的眼睛和鼻尖上了 。
「: :」！」从我的指尖发出的红色光线刺进了魔王的堤坝 。
「啊——！？』起作用了！凝结所有魔力打入，一击必杀的技能 。
攻击力是最高级的，但也有不接近就无法使用的弱点 。
因此移动技能也很接近了 。
「笨蛋……啊……！「现在，健治！「哦！》健治跑 。
把气从圣剑中迸出，一直对着魔王 。
一直直奔到最后是健治的战斗风格 。
「哎呀……不让！」魔王把手指对准健治 。
——「硬打」！」和我一样因潜伏技能而消失的尤诺之剑，敲打魔王的手臂 。
从用剑弹奏的魔王的指尖放出光束，向健治旁边偏移 。
「兽人——！？」面对吃惊的魔王，健治这次才放了一击 。
「哦哦哦哦哦哦哦哦！』把挥舞的圣剑扔了出去 。
像小星星一样闪闪发光的剑 。
被技能强化的健身剑斩断风，发出清爽的声音飞翔 。
它像主人一样笔直，刺穿了魔王的胸部 。
「哦……哦哦哦………！圣剑从魔王的身体穿过背部，光的气味从伤口溢出 。
就像打破了金币袋一样，光从破旧的长袍里不断地漏出来 。
「啊……啊………………」跪在地板上，低下头的魔王 。
笼罩着他的可恶气场逐渐消散 。
「打倒了…………？」我战战兢兢地靠近了蹲下的魔王 。
反击什么的，不会吧……？「……是吗………………」魔王盯着自己的伤口，点头 。
「扔了圣剑吗……既然离开了人类的手，那只是一把剑…………能讨伐我的神圣力量本身……」在那一瞬间，大家都很注意到了 。
如果设置了人类用的屏障，让人类以外的人攻击就行了 。
「然后兽人的一击……………虽然是一瞬间，但我经常踩我的芒草……」尤诺也经常模仿我，接近魔王 。
从水平差来说，是一击即死的力量关系 。
一定需要相当大的勇气 。
「……但是………你是……什么……？」魔王看着我 。
「知道人类的攻击不通……………首先召唤魔族，你自己也用魔族的技能攻击了我……你不是人类吗………？「……是人啊」，试着回答了一下，怎么样？这个技能是我夺走的 。
「那红光…………那是……………我一族的技能……」于是魔王从我身上转移了视线 。
一直不参加战斗，看着魔王被打倒的样子，艾丽尔 。
「你……爸爸………」「艾丽……我的女儿啊……」魔王说要制造魔族和怪物，这样下去艾丽会成为这家伙的女儿吗？那，谁都不想杀父亲啊 。
犹豫不决是因为那样的理由吗？但是，夺走魔王生命的契机是我使用过的咏叹调技能 。
「咏叹调啊……传授给人类技能了吗………？为什么……背叛了我……」于是魔王的眼睛被睁开了 。
「……是吗………是这样吗……」血淋淋的眼睛看着咏叹调 。
然后接下来交替地看着我和健治 。
想伸手，发现连那个力量都没有 。
「………… …………」翻脸的咏叹调 。
无论是魔王还是我 。
在她的视线前面——健治 。
有眼睛的健治大概把它解释为什么答案了吧 。
「是的 。
艾莉亚选择了和我一起创造未来的道路」，靠近魔王，健治这样回答 。
「不管艾丽尔是你女儿吧 。
我要和艾莉亚建立一个和平的世界 。
这是一条与魔族和人类无关的创造和平世界的道路」「…………你……」魔王想说点什么，然后终于倒下了 。
「爸爸！」咏叹调也是徒劳的，魔王的身体发光 。
被耀眼的光芒遮住眼睛，等一会儿再恢复视野，那里只剩下破旧的椅子和长袍 。
安静的空间再次迎来了安静的时间 。
「到此为止……结束了，吗？』我不太清楚 。
没有魔王，就是这样吗？健治也环顾四周，没有什么特别的变化 。
然后犹豫了几十秒左右——在魔王坐的椅子上，看到了像萤火虫一样的东西在发光 。
那家伙是用只有我和健治才能听到的声音这样告诉我的 。
「恭喜你 。
漂亮地击破魔王的木下健治先生，作为报酬送给桐谷娜奥特先生「愿望券」——」
