冒险者生活的第二天开始，我和健治去了冒险者公会，成为了晴天霹雳「冒险者」 。
嗯，谁都可以注册，所以也不是那么伟大的头衔 。
状态画面也好好地记载了「职业冒险者」 。
这是游戏 。
游戏是有目标更容易玩 。
最近的游戏大家都是这样吧 。
告诉我接下来该做什么 。
虽然有打倒魔王的大目标，但在不知道为此做什么好的情况下，从事某种工作真的很感谢 。
冒险者的工作也比较轻松 。
每天早上去公会总部，和弗里达打招呼 。
接受她选择的初学者用的任务，在城外处理它回来 。
报酬当场支付，再休息到第二天 。
完全是临时工，但与此大不相同的地方有很多 。
首先接受的任务几乎都是怪物的讨伐 。
我不知道其他人怎么样，但我们打倒怪物会提高水平 。
如果持续一周的讨伐任务，我也能得到相应的经验值 。
成为冒险者当初是14的水平，现在也超过了20 。
即使没有「成长」的礼物，转移者似乎也有比一般人更快提高水平的体质 。
而且健治已经达到了50级 。
格奥尔恩周边已经不存在能战胜健治的怪物了 。
得到了可以用一只手简单应付的强度 。
「喂，娜奥特！今天就到这里回去吧！」自己的数倍大小背上某兽型怪物的尸体，那家伙挥手 。
无论多么坚强，和蔼的笑容都不会改变 。
「我饿了 。
提高水平会增加消耗卡路里吗？」「你原来不是经常吃吗？」「水平一提高，疲劳也会和伤口一起消失 。
但是肚子会减少吧」「啊，我明白 。
感觉很奇怪啊」几乎感觉不到和现实世界的差异 。
所以这种小变化很有趣，经常共享信息 。
健治和我都脱下衬衫，穿着适合这个世界的皮革衣服 。
因为被施了魔法，所以不容易破，不弄脏，皮鞋也很好走 。
我还不能用，但魔法好像是非常方便的替代品 。
「今天的晚饭怎么办？「海风亭」可以吗？「偶尔去别的店吧 。
在吉尔德，有人告诉我肉很好吃的餐厅 。
」一边回到街上，我们一边说今晚的计划 。
最初连左右都分不清的格奥尔恩的街道，但能掌握行动范围内的地理程度 。
现在抬起头来，耸立着无愧于城塞城市之名的巨大墙壁 。
大概有三十层楼那么高吧？据说圆形的墙壁复盖着街道，防止了怪物的入侵 。
我在网上见过棒球场里排列着样板房的照片 。
正是那种感觉墙保护着街道 。
和守卫墙壁入口的卫兵打招呼进入全镇，首先以冒险者公会为目标 。
雕刻有剑和鞋的浮雕是世界通用的标志 。
「啊，欢迎回来！」一进入政府大楼，弗里达就在入口附近的接待处从椅子上站起来 。
一边摇晃着长长的金发和丰满的胸部，一边走出柜台向这边跑来 。
「今天也辛苦了！开头怎么样？」「很顺利哦！被委托的暴风雪，讨伐来了！』摆出嘎吱嘎吱的姿势 。
「谢谢你！我家的公会很难打败人！像健治和娜奥特这样坚强的人成为了冒险者，真是帮了大忙了！」弗里达侦察我们的理由好像就是这样 。
坚强的我们如果在人手不足的冒险者公会工作的话，她的评价也会提高 。
我们也能得到报酬，还能还她借的房租 。
最重要的是，健治很活泼 。
完全明白，在弗里达面前时的图拉 。
「是的，这是今天的报酬 。
」弗里达给我们每人一个小皮袋 。
打开里面，几张银币 。
确实是这个国家的货币17000戈茨 。
这样就付房租和饭钱，剩下三分之一左右 。
大概换算成一梵高一日元可以吗？「……真便宜啊」我说出了正直的感想 。
「嘿，娜奥特！对弗里达先生很无礼吧！「你说不愿意，暴风雪太大了吧 。
打倒那个就这么不划算吧？』刚才弗里达不是也说了吗？据说在公会也很难打败人 。
话说回来，能打倒这个城市的家伙真是水平很强啊 。
尽管如此，一天生活才算是工资——「难道冒险者是不受欢迎的职业……？「不，不，不是，那是有理由的」弗里达慌忙辩解 。
「冒险者也有排名 。
你们才刚刚成为冒险者，所以是青铜等级，也就是最低等级 。
如果等级提高的话，基本报酬也会增加」「那怎么才能提高等级呢？「需要审查……行市长的」「要让我审查怎么办？我想快点上榜」「嘿，纳奥特，适可而止吧」健治抓住我的肩膀 。
「弗里达先生不是不好吧 。
因为是公会的结构」「……嗯，虽然是这样」我姑且不说健治是50级的 。
怎么想都不是安于青铜的好强度 。
「对不起，纳奥特先生，为了能尽快上榜，我也试着拜托你审查一下 。
我想如果是现在的业绩的话一定会通过的」低头的弗里达 。
「……不好的弗里达先生，我也说得有点过分了」「嗯，没办法 。
因为我觉得只是说说而已」难道在异世界也会被政府机关工作折腾吗？只有市民和底层吃亏，在哪里都一样啊…… 。
「只是，如果排名上升的话可能会有更危险的请求 。
就算那样也没关系吗？「要和比那只大蜥蜴还强的怪物战斗吗……」我面面相觑，笑着 。
「宁可咚咚地来，哦！」对健治来说，已经只能是强大的怪物等饵食了 。
猎杀更强的怪物，得到更高的经验值的话就容易打败魔王 。
为此，必须快点提高排名 。
*掌握了比对手早打倒魔王的游戏这一规则 。
但是现状是，我们的玩法可以说是极其认真的 。
受公会的委托赚钱，提高水平稳步增强 。
虽然在能力上应该是球队的，但是打得很普通 。
当然想做的话，也采取了别的方法吧 。
但是健治好像喜欢现在的做法 。
孜孜不倦地认真工作，一步一步前进的正统派打法 。
这样想来，「成长」的礼物可能是最适合他的能力 。
因为努力确实会有成果 。
老实说，我也不是不喜欢这种生活 。
脱离该死的现实，在异世界轻松度过，很舒服 。
适度的运动，没有后遗症的人际关系，适度的刺激——积攒的时候花了赚的钱去了妓院 。
这个世界上好像也有流氓一样的组织，出门对妓女的管理也很好 。
所以我充分使用了「吸血鬼」的礼物 。
目标是前冒险者的妓女 。
指定了因受伤和金钱原因不得不辞去冒险者职务的女性 。
虽然水平很低，但技能很有意思 。
也许是进一步锻炼的缘故，技巧熟练的女性很多 。
如果没有「吸魔」的技能，也会提名前冒险者的妓女吧 。
所以水平是25级，但技能丰富了 。
这个世界的技能分为自动发动的被动和谨慎发动的主动两种，掌握这些好像需要熟练度 。
也就是说，任何东西只要使用就能学会 。
健治也学会了几个长剑技能和回避技能，我也学会了匕首技能和回避技能，还有潜伏技能 。
是努力确实有成果的世界 。
「吸魔」的礼物是跳过那个努力的团队能力 。

那天和健治说，想休息一下冒险者的工作，治愈疲劳和整理装备 。
所以我也想去妓院旅行顺便买点东西，所以在街上闲逛…… 。
