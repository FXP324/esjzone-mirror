我决定先带你去我们住的城镇——要塞城市格奥尔恩 。
艾莉亚和尤诺好像都来自不同的城市，左右往返于第一次来的城市，好像在看着前面的我们，微笑着 。
这次轮到我们带路了 。
让疲劳的她们在旅馆休息，在离那里最近的酒馆吃晚饭 。
费用是刚才狩猎的一张怪物皮就能给出来的 。
尤诺一边舔着沾着肉汁的手指，一边说「本来，我打算委托这边的冒险者公会哦 。」
「我想要你们夺回我的村庄。」
「村庄……？发生什么事了？」总之，健治挺身而出，想听听故事 。
我在它旁边看尤诺 。
我想用鉴定眼，但看不了 。
不仅仅是尤诺，具有一定强度的人似乎不能使用这个技能 。
素的抵抗力拒绝技能 。
嗯，那也是吗？那种惠惠手里被揭穿的话，那才是齐特 。
为了不被注意到，要裸体，或者要得到观察眼[S]那么高的眼睛，还是处于无法抵抗的状态——这样的话就无所谓了 。
现在是尤诺 。
「我的村庄被魔族占领了 。魔王的部下占领着，大家都逃离了。」
「魔族……？不是怪物吗？」
「嗯，魔族。」
「娜奥特，娜奥特。」旁边拉着我袖子的健治 。
我用眼神示意他想问什么 。
「啊，所谓『魔族』，就是像人类一样有智力的家伙们 。是有意志，明确要毁灭人类的种族哦 。相反，像没有智力的野兽，这样的家伙被称为怪物。」
「原来如此，也就是说是坏人。」
「听得懂就帮了大忙了」实际上，我也没有光凭知识见过 。
魔族似乎是相当高水平的存在，不在人类城市的周边 。
住在魔王城附近，像贵族吗？
「那么，那个魔族为什么占领尤诺的村庄？为什么特意去你的村子？」
「那是……因为我的村庄是世代守护圣剑的村庄。」
「啊，圣剑啊……」嘛，因为是异世界，所以也有那样的武器吧 。
现在也对这样穿的衣服的魔力感到吃惊 。
真的是一件衣服就能战斗就寝的好东西 。
连一般冒险者穿的衣服都是这个，魔族瞄准的圣剑有多大魔力呢——「据说圣剑连魔王都能斩 ，害怕那种力量的魔王，让部下占领了村庄」
「是占领 ，破坏啦，灭亡啦，不是那样的。」健治向咏叹调的问题击中了好地方 。
「是的……甚至魔族，不，连魔王都没有拿出圣剑 ，所以我只好盯着看 。」
是那么强的剑吗？用那个的话，健治也能轻松打败魔王吗？嗯，会吧 。
那把圣剑的锋利度，我想试试——「知道了，去帮忙吧」，健治比我的思考时间还快 。
在和这家伙商量的时候，我明白了没有拒绝的选择 。
其实，我也赞成去救尤诺村 。
格奥尔恩的冒险者公会已经不够了 。
我认为与其接受白金级的委托提高水平，不如和怪物的上位级魔族战斗，效率更高 。
而且村子里有圣剑 。
如果圣剑与健治的力量相吻合的话，魔王不是就能轻易打败了吗？取决于魔王的水平——离开格奥尔良向西走，绕过第一次和两个人相遇的森林再向西走 。
据说尤诺村所在的地方大约有400公里 。
大概是东京到大坂左右的距离吗？汽车的话当天就能到达，但没有那么方便的替代品 。
也许能用移动技能去，但是没有咏叹调和允诺就不知道了 。
因此，他一边坐马车旅行，一边和健治一起提高等级 。
「哈哈！」健治的剑一闪一闪的击中了马形怪物 。
「嗨！」在他旁边，尤诺挥舞着两把刀，砍下另一只马形怪物 。
看起来比健治快，但攻击很轻，怎么也打不过怪物 。
虽然是想用后腿踢飞的怪物，但在尤诺避免那种攻击的同时，健治会从死角斩杀 。
「哦，太棒了！果然比不上师傅！」
「阿哈哈，不要叫我师傅了。」比起倒下的怪物，尤诺更佩服健治的技能 。
同样是剑士，我知道你向往健治 。
而且不管结果如何，都是听了话马上接受的好人 。
也有恩义吧 。
「为了和魔族战斗，我也会变强的！所以要好好锻炼哦！」
「嗯，嗯，我会尽我所能的。」健治被强势的气势压着 。
哪个立场更高？「……好吧，差不多完成了。」我和艾丽一边看着那样的两个人，一边准备午饭 。
一边用野营地做的简易灶做汤，一边用别的篝火烤肉 。
「你的手指意外地很灵巧哦，纳奥特先生 。」
咏叹着切蔬菜的我 。
「自己做饭是最近学会的 ，这样的生活很方便啊。」烹饪技能和野营技能是用「吸魔」的礼物夺走的，某种意义上比战斗技能更有用 。
就算回到原来的世界，我也不想失去这个技能 。
而且会做饭的男人很受欢迎 。
「尤诺也很活泼啊 ，和我见面的时候，明明是死了一样的眼睛」
「这边也很活泼哦 ，因为到现在为止只有两个混蛋的旅行 ，果然和女孩子旅行气氛完全会改变嘛。」
「嘛」咯咯笑着的咏叹调 。
实际上，女人混在一起的话情绪就不一样了呢 。
但是并不是只有男人的旅行不好 。
两个都很开心，但都是两码事 。
「顺便问一下艾丽尔，我想问一下。」
「什么？」
「你和尤诺是同一个村子的人吗？」
「啊……不，不是的。」
「那你为什么和尤诺一起旅行？」
「那是——我也是作为冒险者旅行，中途偶然遇见的 。听了她的情况，想要想办法帮她……但是，光靠我的力量是不行的。」握住衣服的胸部周围，说话的咏叹调 。
大致有道理 。
我想更详细地知道，但是鉴定眼不起作用 。
因为对剑士尤诺不起作用，所以担任神官这个魔法职位的咏叹调几乎可以无效 。
其实，我并不完全相信尤诺的话 。
我在想，不是像弗里达那样，用某种手段注意健治的能力，欺骗然后利用吗？我完全不知道战略什么的，但我知道健治的强度已经是特级的了 。
我想如果那家伙和我齐心协力的话，就能够毁灭一个国家 。
虽然说是游戏，但平衡一团糟 。
所以，有可恶对手的组织、想支配的国家、想毁灭的岛屿——一定会出现为了破坏这种东西而利用我们力量的家伙 。
弗里达的时候，只是碰巧的朝着好的方向发展 。
「师傅！有什么挥动武器的诀窍吗！？」
「不，所谓诀窍……对不起，几乎没有被教过 ，因为我可能没有你打得那么多。」
「这难道是天赋的才能！？太棒了！那至少让我看看吧！」粘在健治身上，就像是兄妹一样 。
像亲近的猫一样嬉戏的尤诺是否会欺骗我们——不，女人全部无一例外都在撒谎 。
从魔族手中夺回村里的圣剑？其实尤诺是魔族，可能想把圣剑作为诱饵把转移者赶出去 。
魔王方面有转移者的情报也不奇怪 。
这样的超群能力者，一定在世界的某个地方引起了轰动 。
无论哪里，魔王都在收集情报吧 。
我是个混蛋，习惯利用也习惯了 。
但是健治不一样 。
我不能欺骗那么纯粹的笨蛋 。
准确地说，不允许存在欺骗那家伙，从中受益而窃笑的人 。
是在这个该死的游戏中，信任我的男人 。
我只是希望你不要暴露在恶意中，清除它 。
所以——我会确认你们的 。
