kenzi看到的东西只要打败魔王就能当场实现愿望，但并非如此 。
拿到的「愿望券」什么的，是随时随地都能实现愿望的道具，只有拿到的人才能使用 。
就像状态一样，只要念念不忘随时出现在眼前 。
听说也不会被别人夺走 。
纳奥特说，总之，在告别这个世界之前，是对「上帝」的关怀，让他把剩下的事情做完 。
我也没做完 。
如果只是打败魔王，这趟旅行就结束不了 。
下定决心回到城塞城市格奥尔良，突然被卫兵包围了 。
我以为你做了什么坏事，被抬进了城市领主等待的官邸，在那里被几个胡子脸大的叔叔哭着拥抱了 。
「哦，勇者健治啊！永远赞颂你吧！真的非常感谢！代表国家向我道谢！」作为打败恶魔王的勇者被狠狠地揍了一顿之后，在整个城市都举行了祝胜派对 。
不是为了我们，大家都想高兴 。
国王也从王都来了，和我握手 。
历史上第一次特意来这个城市见普通人的我 。
虽然不太了解这个国家的风俗，但总之好像是很伟大的事业 。
在做了那种形式的事情之后，我们终于也参加了派对 。
「康帕伊！」除了一家经常吃晚饭的叫海风亭的餐厅外，我们一行人还在为过节而特设的桌子上吃晚饭 。
夜空中燃放烟花，广场上乐团从早到晚持续演奏 。
全部免费招待的肉、鱼、还有酒都特别好吃 。
「干得好，健治！终于打败了我们魔王！「啊！这一切都结束了！」我一边干杯，一边和娜奥特笑了 。
回想起来是一次漫长的旅行 。
被转移到这个世界大约半年了吧？「倒下后，成为了冒险者，战斗了一会儿……」「彼此没死就好了，真的」「我没有娜奥特就死了」「没什么事」 。
如果是你的力量，无论做什么都要活下去 。
」明明贬低我是个笨蛋，但我们一夸奖，娜奥特就会这样谦虚 。
就像人们说我是傻瓜正直一样，我总是正当地评价娜奥特 。
正因为有他，我才能战斗得笔直 。
相信某人，我帮了大忙 。
怀疑一直是他的工作 。
费了很大的劲，我也觉得很抱歉 。
「得救的不仅仅是娜奥特 。
尤诺，你也帮了我好几次」「得救的是我！多亏了师傅大家才有了笑容！』嘴里塞满了串的烤鱼的允浩 。
虽然水平不到我的一半，但魔王对手也勇敢地战斗了 。
我觉得没有礼物的那个身体，很好地跟着我 。
她接下来会重建村庄吧 。
召集被魔王军分散的村民，再活下去 。
那时把圣剑还给我吧 。
确实是旅游业吃的村子，没有圣剑应该很为难 。
然后——「艾丽尔，也真的谢谢你 。
多亏了你，我打败了魔王」「……我」「很好 。
我已经知道你是魔族了」魔族也有好的也有坏的 。
就像在那个魔王城生活过的居民一样，既是热爱和平的魔族，也有憧憬冒险者的人 。
「喂，你能听我说一下吗？」放下背心，我就向伙伴们传达自己的想法 。
「我觉得魔王倒下后，魔族的动作也会改变 。
至今为止和平生活的魔族的生活应该会发生变化 。
我想关注那个动向」「嗯……？」纳奥特发出惊讶的声音 。
「那是说你要承担魔王军的善后工作吗？」「暂且不说折磨人的魔族，我想尽量保护温柔的魔族……，不行吗？「不，不可以……嗯」纳奥特沉思着 。
那是啊 。
他想和我一起回到原来的世界 。
但是我也想珍惜有咏叹调的这个世界 。
「肯齐先生……谢谢你」盯着我，艾丽尔微笑着 。
我以为那是答案 。
吃饱了，喝到有点醉的感觉 。
其他大家也对吃饭很满意，所以决定解散一次 。
也有想在旅馆休息，在节日里散步的人吧 。
决定使用愿望券是在魔族的问题解决后，让娜奥特先回到原来的世界 。
虽然很不安，但是如果是我的水平的话没关系 。
我一边心里这么想着，一边走在格奥尔恩的街上 。
现在穿着带兜帽的斗篷，街上的人也注意不到我 。
比起那个，我更执着于国家免费招待的酒和吃饭 。
我一边听着从远处传来的乐团演奏，一边面对的是冒险者公会 。
「哎呀，健治！欢迎光临！「晚上好，行市长」今天冒险者行工会中人也很少 。
大部分的冒险者都忙于祭典的警备打工，职员们也忙于管理 。
现在在公会的只有女性公会长 。
虽然我想说「真的恭喜你」，但作为公会来说是个复杂的地方」「是吗？「魔王死后生态系统发生变化的话，调查和那个擦屁股，工作就会增加 。
虽然会变得复杂，但最终还是和往常一样 。
」面对魔族的任务对格奥尔恩的冒险者来说负担很重 。

如果弗里达先生也能享受节日就好了 。
「承蒙您多方关照，我想谢谢您……还有其他想问的事情」「想问的事情？如果是公会的事我也承包」「啊，不，不是那样的 。
私人……」但是，行会长也是女性 。
也许你知道 。

今天是节日，很多工艺匠都开着露天 。
听说里面可能有古老但优质的作品廉价出售，我跑到了广场 。
在像夜店一样大量排列的工艺匠开店时，我买了我认为最漂亮的项链 。
不问价格，把最贵的金币强加给工匠，让他马上包装 。
拿着那个礼物，我回旅馆 。
艾丽尔，你还醒着呢 。
创造一个与魔族和平的世界，当然是为了咏叹调 。
总有一天她是魔族，必须创造一个让我以外的人知道的世界 。
那样能一直和艾莉亚在一起，该有多好 。
也不是对原来的世界没有留恋 。
但是有一个关心我的人的世界比较好 。
是这样向她坦白的 。
我的脚加快了 。
多次撞到醉醺醺地走着的人，低头往旅馆跑 。
注意到的话连技能都用了 。
我想尽快接近咏叹调 。

经常在二楼的单间 。
一边上楼梯，一边想语言——「———…！』楼上有声音 。
咏叹的叫声，还是吼叫的声音？你在打架吗？稍微抑制一下偏离的心情，冷静地上楼梯 。
尖叫声至今不止 。
「艾丽尔？』我打开单间的门 。
于是——咏叹调和娜奥特裸体拥抱着 。
