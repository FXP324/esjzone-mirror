- -这是游戏 。
——打倒魔王就过关了 。
这里是异世界 。
我被转移到不知道是上帝还是恶魔的混蛋身上了 。
——只要打倒魔王，任何愿望都能实现 。
——如果想回到原来的世界，就让它回来吧 。
——如果想成为这个世界的统治者，就实现吧 。
在简直像游戏一样的幻想世界里，真正的游戏被做了 。
提高水平，记住技能，装备齐全，打倒魔王 。
只是用自己的身体做那个，简单的工作 。
与用手指操作垫子不同，拿起沉重的武器，与臭味魔物战斗，汗流浃背地收集物品，死了就无法复活的游戏 。
保守地说是该死的游戏 。
如果不付出生命，谁会这样做？——但是魔王的数量有限 。
——魔王的数量比转移到这个世界上的人的数量还少 。
除了我以外，还有被转移的多人 。
我必须和他们争夺魔王 。
不仅仅是RPG 。
这是拿椅子的游戏之类的 。
如何摆脱其他转移者是一场比赛 。
用正攻击法比任何人都早打倒魔王，也可以以某种形式妨碍其他转移者 。
什么方法都行，总之先赢者 。
正因为是简单的规则，性格才会受到考验 。
像我这样的混蛋不适合正攻法 。
*曾经——我打倒过一次魔王 。
那时我和朋友在一起 。
总之和强大的伙伴一起合作打倒魔王 。
在任何人眼里都是明显的正攻击法 。
没有像伙伴一样的强大是做不到的荒诞技巧 。
我现在还很感谢那个伙伴 。
在这个该死的世界里，如果没有他我就活不下去 。
但是，我背叛了那家伙 。
那家伙把爱的女人们，一个接一个地睡着了 。
因为是好家伙，所以在被坏女人骗之前——打着那样的大义名分，我犯了那些接近那家伙的女人 。
那样只是借口 。
我只是想打败那家伙，装作正义的朋友 。
当她一边哭一边看着我在做爱中怯场的时候，从来没有这样想过 。
只是太好了 。
那个报应来了 。
重要的伙伴全部暴露，所有的关系破裂了 。
不得不破坏一切，让一切恢复原样 。
我向打败魔王得到的「愿望券」许愿，决定全部重做 。
在重新开始的世界里，没有人被侵占，也没有和那家伙相遇 。
就我一个人，从头开始 。
那样就行了 。
那样的好家伙，对我来说太耀眼了 。
我一个人就行 。
有女人就讨厌，如果出现瞄准生命的敌人就无情地杀人 。
与转移者密切相关，用我自己的做法打倒魔王 。
再一次，在这个世界上重新开始——我继续向东旅行 。
一定程度上提高了水平，获得了技能，鬼鬼祟祟地变强了 。
然后——我现在被监禁了 。
