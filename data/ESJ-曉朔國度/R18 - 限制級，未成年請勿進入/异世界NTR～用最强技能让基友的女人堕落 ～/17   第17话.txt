果然，魔王城附近到处都是非常强的怪物 。
一百多级的怪物通常在那边走 。
有像该死的大恐龙一样的家伙，也有和人类一样大小的魔族 。
因为在毫无隔阂的荒野上很容易找到，所以也无法避免战斗 。
虽然步伐会慢慢变慢，但还是稳步推进着去魔王城的道路 。
但是与最初相比，我们也学会了战斗的方法 。
健治为了提升水平，我从怪物那里夺走的技能也增加了，战略范围扩大了 。
被认为是绊脚石的艾莉亚和尤诺也编织出了适合自己身高的战斗方法 。
咏叹调为了让我和健治容易战斗，重点是强化魔法，允诺开始采取扰乱敌人眼睛的战法 。
特别是尤诺，他活用了比健治更快的身体，学会了多次产生残像的技能，甚至发展到了健治叮咬敌人动摇的芒草的组合 。
「太好了！师傅，干得好！」「干得好，尤诺！有这个组合大部分怪物都可以哦！」击掌的两个人 。
虽然是有身高差的组合，但很合得来，看着很有趣 。
战力雄厚 。
剩下的只是以魔王城为目标 。
当然，也要好好休息——那天晚上，调教也在继续 。
「啊，啊，啊！啊——！」洞里乱七八糟的尤诺 。
骑在我身上，用下面的嘴小心地舔着肉棒一样地摇着腰 。
也许是自己找到了舒服的地方，一边前后左右挥动腰，一边不时地颤抖身体 。
一遍又一遍地像振动一样对待我 。
「好舒服！霍乱真舒服！娜奥特的，满了，啊！」被黑暗照射的允浩纤细的身体发光的不仅仅是汗水 。
也有涂在全身的油的效果 。
就像用炼金术技能制作的乳液 。
虽然也含有一点媚药成分，但主要是作为润滑油的作用 。
就像嬉戏的动物一样搓着身体，像海绵一样使用我的皮肤 。
「哈哈……纳奥特奥…………纳奥特奥……！」一边拥抱着空空的触感，一边用更有粘性的汁液的器官品尝我的东西 。
「啊，啊，啊……！』身体又跳起来了 。
这是第几次绝顶了，已经连数都懒得数了 。
「嗯……嗯………嗯……中……」雪诺朝我倒下 。
代替不动腰，舔我的舌头 。
「累………中……纳奥特……喜欢……」「喂，健治的事好吗？」「师傅……作为师傅………我尊敬你…………中……」一边接吻一边用溺水的声音回答 。
「但是……现在是娜奥特………和娜奥特陷害……最喜欢……啊」「嘿嘿……因为，健治」我把脸转向一边 。
在旅行用的床单上有一个躺着的健治 。
在不到一米的地方，什么也没注意就睡着了 。
真不愧是让我使用了技能 。
不管你在旁边怎么喊，应该直到早上都不会起床 。
「啊，啊，啊——！」不知不觉，比平时更气喘吁吁的尤诺 。
因为最近老是抱着咏叹调 。
一天一发左右，尤诺好像也不满意 。
「明天是魔王城——吗？」洞窟外面，广阔着夜晚的荒野 。
在那个荒野里，孤零零地建着的城堡 。
灰姑娘城堡不是很大 。
远近感都快疯了 。
我听说我们作为据点的城塞城市格奥尔恩，正好和东京23区一样大，不是一半左右吗？就像格奥尔良是被墙包围的城市一样，那座城堡本身也可能就是城市 。
「是不是要久违地回乡了，对艾丽来说」紧贴上下的我们的旁边，健治的对面坐着艾丽 。
「啊…………！』今晚什么也没插手 。
只是展示了和尤诺的交往 。
「对不起，我也是拼命的 。
你的主人吧，我会全力以赴的 。
结果这个国家也会变得和平吧 。
我也被健治感染了吧」「……喜欢就行了……」咏叹调坐着一动不动 。
把手伸进裙子里，在里面兜圈子 。
在尤诺的尖叫声中，我听到滴滴答答的水声 。
「如果魔王会输的话…………那就是……战斗的结果……健治先生一定会堂堂正正地……战斗……嗯……！」呼吸急促 。
自慰是不能满足的吧 。
我们在做更舒服的事 。
「啊……啊……更好，更好！舒服点！」失去理智，看到像野兽一样在我身上运动的尤诺，咏叹调也想这样吧？即使假装没兴趣，也已经给我看了一个小时了 。
「啊，啊，啊，啊！」尤诺又生气了 。
调制好的乳液也被汗完全流下来，感觉很粘 。
「啊…………啊…………」这么雅诺也满足了吧 。
从我身上滚下来，就这样在旁边喘着粗气 。
比和怪物战斗更疲惫吧 。
但是我还没有射精 。
体力也很好 。
「好吧，艾丽尔 。
」我对自己安慰的她说 。
「怎么办？这个，空了」「啊……！」指着胯股之间，艾丽尔露出了眼睛 。
仰面朝天，就很清楚自己的状态 。
就像那里的魔王城塔一样高耸着 。
自古以来就是这么大，所以没有自觉，但好像比别人大很多 。
比较健治的胯间和我的咏叹调 。
我也不知道勃起的健治有多大 。
尽管如此，比起健治，我的技术和经验都更好 。
是唯一赢的部分啊 。
把视线集中在那个优势上的咏叹调 。
「………我明白了……哦」我什么都没有命令 。
但是对艾莉亚来说，她想要某种「强制的外表」吧 。
当场脱下衣服，无言地跨在我身上 。
「……嗯………啊！」把早就准备好的胯股之间推到我的东西上，一下子塞进去 。
「啊……啊啊啊！」与用坚硬的肉壁捋平的允浩之洞相比，咏叹调的是变幻自如的名器 。
现在完全放荡了，一直等的是暴露 。
身体不会再撒谎了 。
不管你态度如何拒绝，身体都要求和我做爱 。
「啊！啊！啊……嗯！」乳房在我身上咕噜咕噜地摇晃 。
剧烈摇晃，能看到残像的白色肉体，和动作更快的腰部 。
和尤诺一样在寻找让自己舒服的点 。
说到挖掘到矿脉时高兴的脸 。
「虽然很讨厌，但还是主动摇了摇头」「8，不……这个……被你说了没办法……」「我什么都没下命令啊？」「沉默的……压力啊………啊啊啊！「是的，就这样吧」完全这个女人 。
明明旁边有健治，一开始就不客气地注意到乱了吗？已经不在眼里了吧 。
我也抓住泡菜的大腿，从下面往上顶 。
同时在前端像摩擦g点一样移动 。
「哈哈！那个……不行！』我知道 。
就像是对正直的咏叹调的奖励 。
「啊啊啊啊啊啊啊啊！会变得奇怪的！会变得奇怪吗！不行，伊克，伊克，讨厌，啊啊！哦，天啊！」——明天是和魔王决战的日子 。
尽管如此，艾丽和尤诺最终还是干了十多次 。
我担心战斗中会不会腰疼 。
算了，那样的我也让你进去五次了 。
自我恢复的技能在这种时候也有用 。
