　金曜日。
当天也是在休息时间里夹杂着纪更转移而来的口交和与米莉娅的淫乱行为完成工作。米莉娅以有事为由按时上班，新则加班1小时后出门。
如果是夏天的话，时间刚好是日落的时间，但现在天已经完全黑下来了，在办公室里看日落已经成了习惯。
虽然每年让人感受到青春梦里人和冬天的到来的时候都会感到没有理由的孤独，但是现在只要回到家，就会有重要的恋人。我再次感觉到，自己身处一种令人难以置信的幸福之中。
「好冷啊... ...」
下班后走出大楼，有如穿透夹克般的寒冷。平常他都会挽着米莉娅的胳膊回家，所以这种时候更想念她。
今天三个人都因为事情回家晚了，现在回家也只有我一个人。我决定好久没出去吃饭了。在和米莉娅交往之前，我经常一个人在外面吃饭，但是一想到久违了就兴奋起来。光是吃牛肉盖饭就已经很开心了。
「……ん？」
我一边想着，一边拐过拐角，看到一个眼熟的女子从对面走过来。穿着衬衫一手拿着智能手机走路的女性突然抬起头，发现了新的存在，眨了眨眼睛。
「哎呀... ... 不是新的吗?」
蜘蛛女人卡特琳娜把手机放进口袋里，面带艳丽的微笑。在路灯很少的地方看到蜘蛛的下半身令人毛骨悚然，它的美丽让新不知不觉地发出了喉咙的响声。

　数分後。
「从那以后，你再也没来过我的店，所以我很难过，你能再来一次吗?」
「哎呀，哈哈... ...」
(怎么办呢，这个... ...)
新和卡特琳娜并肩走在空无一人的地方。卡特琳娜工作的酒吧好像是休息日，平时几乎不在外面吃饭，但是今天不知为何出去了。
「这个身体哪里都会让人吃惊。」
爽朗的卡特琳娜的侧脸一如既往。但一想到「一如既往」是建立在无数的痛苦和悲伤之上的，我的心就像被涂了铅的绳子勒紧了一样。
当我告诉她我又要出去吃饭的时候，她自然而然地问我: 「要不要一起吃?」我记得米莉娅的忠告，所以心里很不是滋味。
「怎么了? 你从刚才开始就很紧张。」
被卡特琳娜的问话吓了一跳，静静地转过脸去。
「不，那个... ... 重新一看，卡特琳娜真的很漂亮，包括蜘蛛的身体。」
「嗯... ...」
咦，好冷啊... ... 我抬起头一看，卡特琳娜不知不觉地面对着后天的方向。
(害羞? ... ... 好可爱啊)
在有很多美女的虚构种族中，卡特琳娜的美貌非常出众。虽然使用「绝世」这个形容词并不常见，但即使是对卡特琳娜使用，似乎也没有人会提出异议。
这样的她所表现出来的可爱表情，在平时的美貌上就会产生令人难以置信的差距。阿新不由得看得目瞪口呆。
「 ... ... 你有没有想过什么愚蠢的事情?」
「哇! ?」
突然，身体忘记了重力，本应处于比自己更高位置的卡特琳娜的脸出现在眼前。
(这是... ... 蜘蛛之丝还是... ... ! ?)
由于触感过于纤细，迟迟没有注意到，只见自己的上半身缠着一根闪闪发光的白线。
卡特琳娜在路灯照耀下的脸颊微微泛红，轻轻地把手放在新的脸颊上。虽然脸上浮现出艳丽的笑容，但不知是不是还有些害羞，表情有些僵硬，而且还是那么可爱。
「 ... ... 看来你很擅长这样的赞美... ... 难道你以为我会为此感到羞耻吗?」
「不、不，因为脸红了... ...」
「啊?」
「哇! ?」
卡特琳娜弯曲食指的瞬间，与令人眼花缭乱的美貌的距离瞬间缩短到十分之一以下。
「等、等等... ... 哦哦... ... ! ?」
不仅如此，她还用纤细的双臂搂着我的背。他的额头紧贴着她的额头，仿佛在对她施加压力。
卡特琳娜的额头烫得像感冒了一样，自然而然地搔痒着鼻腔的甜甜的口香糖就像唾液是用蜂蜜制成的一样。午饭后刷牙，离开公司之前也总觉得喝着护肤品真是太好了。
「我在那家酒吧工作了将近4年，也见过各种各样的顾客。这样就不会害羞了吧?」
在连鼻尖都能碰到的距离上，卡特琳娜露出挑战性的笑容，与他说的话形成了鲜明的对比。
「不，不，这和这个没有关系... ... ?」
如果他在酒吧工作4年，肯定会接触到不少人。但是这和现在两人的距离感是两码事。距离太近了，或者说几乎没有。很柔软。闻起来不错。在夜晚的路上，我到底在干什么。
「我、我说了吗?」
「呜呜呜... ... ! ?」
卡特琳娜温柔地摩擦着额头，身体越来越贴近。如果只用胳膊搂着，就会掉下来，但下半身缠着几条蜘蛛腿，上面还缠着蜘蛛之丝。在不可思议的稳定状态下，新的身体前面被至上的柔软包围着。
(糟、糟糕，这样下去的话... ... !)
一种女性特有的甜味和柔软。
一张比眼前更近的，兼具可爱与美丽的妖艳脸庞。
作为一个虚伪的男人，他的身体不可能没有反应。
「嗯? 怎么了，下面有什么东西太硬了... ... ! ?」
蜘蛛女拼命想让对方承认自己并不害羞，满脸诧异地从新移开脸，低下了头。
「 ... ... ! ? ... ... ! !」
　瞬間。
纤细的白皙，带着淡淡的红色的极美的容貌，瞬间就煮熟了，简直让人误以为是看到了火山口。
(不行，被发现了)
看到丰满膨胀的裤子，新露出了遥远的眼神。最近，不仅是纪更，连坦娅和爱丽丝也有了交集，所以肉棒的长度、粗细和硬度都更高了。
明明是工作用的西装，一勃起就会膨胀得快要把拉链弄坏了。顺便说一句，睡衣把裤子推得太紧，里面的东西都露出来了。有时米利亚觉得很有趣，伸长舌头舔着伸向裤子的勃起肉。
「这、这是什么? 真、真是没办法... ...」
「咕咕... ... ?」
光用线支撑着新的身体，卡特琳娜颤抖着声音用细指抚摸着鼓胀的隐秘部位。与刚才可爱的羞耻不同，白皙的美貌上清晰地浮现出充满情欲的雌性。
「这么大的话，应该能对付好几个人吧... ...」
「等、等一下... ... !」
卡特琳娜的脸颊依然红润，但是他的六只眼睛逐渐变得冷静，带着一种像收割猎物的动物一样的锐利。显然和刚才不一样，好像打开了什么开关。
「呵呵... ...」
他愉快地望着因急躁的快感而颤抖的新面孔，舔舔嘴唇。10根细手指贴在密密麻麻勃起的肉棒上，露出陶醉的表情，美得令人毛骨悚然。
这样下去实在不妙... ... 正这么想着，突然浮起来的身体被放下了。缠着线的她轻柔地落地，突如其来的安全感让她眼花缭乱。
「那么，我们走吧。我最喜欢的店有。」
「什么? 啊，啊，是啊... ...」
卡特琳娜似乎没有了过去的发展趋势，令人感到困惑。十几秒前，他那双凶猛的眼睛还没有回应，脸上的表情非常平静。
(怎么回事，刚才那个... ...)
我以为是在做梦，但那眼神给我留下的印象太深了，不可能是我的错觉。
我再次想起了米莉娅的忠告。
但是，如果对现在的卡特琳娜说「果然不行，对不起」，一定会露出悲伤的表情。这么一想，我怎么也无法拒绝。
「…………？」
突然，从斜上方感觉到了锐利的视线。他下意识地抬起头。
「嗯? 怎么了?」
与依然面带笑容的卡特琳娜四目相对。
「 ... ... 没什么。」
新无法抹去胸中自然流露出来的不协调感，缓缓摇了摇头。

和卡特琳娜一起走过来的是距离繁华街步行5分钟左右的有点偏僻的地方。房子看起来有四十年历史了，一个男人在一栋破旧的大楼前拉客，一对打扮得非常华丽的夫妇。
这是一个充满陌生景象和难以闻到气味的空间。如果平时自己在这里闲逛，首先会在到达不了的空间里东张西望，即使不是来旅行的也不会。
「第一次散步来的时候，感到非常困惑。没想到虚构种族也很多，现在比较满意。」
对于卡特琳娜的话，他回答说: 「哦，是吗?」并环顾四周。的确，仔细一看，有穿着夹克的下班回家的猫耳，不知为何从屋顶上向住在这个城市的叔叔搭话的两只胳膊叉在翅膀上的女性，有在货摊上用洪亮的声音卖馒头(日本)的牛和长着耳朵的丰满身材的女性等。
生活在自己不知道的世界里的虚构种族们的形象使新的世界观大幅扩散。
沉浸在感慨之中，脑海中不知为何有一种舒适的触感。
「 ... ... 你为什么摸我的头?」
抬头一看，卡特琳娜带着淘气的笑容，把手放在新的脑袋上，轻轻地抚摸着。
「嗯? 总觉得来得正好。」
卡特琳娜腰部附近有个新头。的确，要说正合适也正合适... ... 。
「 ... ... 我有点不好意思。」
可能是因为一边走路一边被女性摸头的男人这种看不到的光景，擦身而过的半人马和单眼的女性配对给了他一个愉快的眼神。我觉得他们把我当小孩子对待，这让我很困惑。
」你看起来很舒服，所以我继续，不行吗?」
一边说着，一边不断地抚摸着新的脑袋。
「 ... ... 不，感觉很好是事实，还是很丢脸。」
感觉一定很好。通过在酒吧里看到的表演和谈话，我们都明白，但卡特琳娜善于关心，并不寻常。接待客人的时间长短似乎也起到了一定的影响。他一定是看出了新的不满，才问的。

「是吗? 那我就不干了。乐趣留到以后再说。」
「不，不是说以后再说吧... ...」
「啊，看见了，就在那里。」
因为以后还会有人抚摸我的头吗，脸上微微泛红的新闻虽然想要不满地反驳，但是却被卡特琳娜的话打断了。
看到卡特琳娜美丽的手指指向的地方，发现夹在古老办公楼之间的矮小建筑闪闪发光。在没有路灯的地方静静呼吸的建筑物，仿佛是照亮夜晚大海的灯塔。
「我经常去的那家店，我很喜欢。」
「哦，是的。」
一边点头一边走向店里。
「放轻松吧，放轻松。」
「 ... ... 啊，是啊。」
抬头看着自言自语的卡特琳娜的脸。虽然眼前浮现出灿烂笑容的瞬间，眼中似乎闪现出凶猛的光芒，但无法再次确认是否已融化在了黑暗的街道气氛中。
