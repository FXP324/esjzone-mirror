心理咨询
「是的，先生。」
我恭恭敬敬地盘腿坐在地板上，他是这么回答我的。
「虽然有些不礼貌，但是我想拜托主人为这个孩子进行心理咨询。」
「对我来说... ... 应该说是催眠吧。」
我皱起眉头，轻轻地回答。
没有什么心理咨询技巧。然而，心灵只能依靠催眠的技巧。
「你认为我是个方便的上司吗?」？总的来说，辅导是你的工作。自尊心也应该有的。「不管什么理由，你都明白拜托我的意思吗?」
「 ... ... 是。」
「那就表现出诚意。」
这就是我所要求的。大概是有自己的理由吧。我的所作所为一定很有意义。
但是如果你要求我这么做，那就太荒谬了。比起一百个理由，请用一个行动来表示。
那么，怎么样。
心情慢慢地脱去衣服。他把便服叠在地板上，把内衣也脱掉，向我下跪。
「 ... ... 就这些吗? 不，没关系。从那以后，今天一整天，心里全裸着，心里也禁止行走。除了四二足行走以外，不允许行走。」
「我接受。」
「那就告诉我，你有什么理由吗?」
心没有抬起头，开口了。
「那个希望我去做心理咨询，正常上学的孩子，是我的远房亲戚。准确地说，是叔叔妻子的妹妹的孩子。」
「你是说亲人?」
我知道这很难。
我在跪下时的背上坐下。然后无所事事地抓住他的屁股。
「啊... ...」
「但是让你做家庭咨询是什么意思?」
「那个... ... 孩子，就是现在所在学院的学生... ...」
「差不多吧。」
我把手放在心口，发现自己被淋湿了。有点恶心，我要拍拍屁股。
如果你高兴的话就没意义了。
「呀!」
「好吧，我会的。」
但是他们建议我行动。你会看到一些有意义的东西。
＊
第二天放学后，我已经安排好了，只等我进心理咨询室。
敲门。
「是的，请进。」
「对不起。」
虽然是熟悉的室内，但还是好奇地环顾了一下。
然后发现在平时那两张沙发的一边，有个陌生人。
那个女孩，用一句话概括，就像个小娃娃。
「初、中○部的人? 」
「是的，她的名字是东方小姐，是学部的。」
只有猛犸象学校。
东方人用警戒的眼神看着我，一脸闷闷不乐的样子。
他的表情本来就很不高兴，与其说是咄咄逼人，倒不如说是对别人贴墙的表情。头发是用如何保养才能变得干干净净的直发做成的。他的肤色和气质和普通人不同，就像以前所说的一样，他是个混血儿。他们的妻子似乎是外国人。
最引人注目的是她的穿着。她全身纤细，穿着一件仿佛是定做的、自由滑动的哥特洛丽塔。
她的相貌和与之相配的气质太完美了，一碰就会坏掉，简直就像一个冰雕玩偶。
「先坐下吧。」
「啊，是的。」
在心灵的催促下，我成了和那个叫苏宝的孩子面对面的居所。
「 ... ... 你。」
烟囱上的小嘴张开了。没有人侵犯过的嘴唇紧紧地盯着他。
为了掩饰这一点，我喝了下心拿来的果汁
「你的男朋友?」
「呸!」
从惊讶中迸发出来。
「你没事吧... ...」
「呵呵... ...」
心会尽快赶来，仔细地擦拭我的嘴巴。
他凝视着这一连串的对话，完全不为所动。
「你为什么突然这么做?」
「最近，心里变漂亮了。之后，在这里相遇的时候，我也会像突然想起了谁一样仰望天空。「嗯。你出现的时候，我从没见过你的表情。」
「是、是的。」
我把视线转向心灵，却被她的目光转向了别处。晚点再看吧。
只是，这也是情报所说的。
东方人对人的感情非常敏锐。
他们说，他们不会错过任何表情上的任何一点变化，他们很快就会发现对方在动摇和撒谎。
这似乎是他与生俱来的天赋，甚至使他与父母保持距离。
当然，这也给学校生活带来了障碍，到了现在还不上学的状态。
除了学业之外，他的才能在整个模拟游戏中毫无遗憾地发挥出来，实际上被称为世界著名的棋盘游戏玩家。
也许是我的视线引起了他的注意，他的表情变得更加警惕。
「姐姐，你最好和这个人分手。」
「什么? 你说什么?」
「这家伙绝对不妙。」
昨天，他想把在我旁边的心狠狠地拉到自己身边。
不是什么敏锐的东西。这家伙通过刚才的小会面察觉到了我的危险性。看穿了下意识，本能地回避着。
「等，等一下，亲爱的!」
」快点过来!」
昨天拼命地想和我分开。
虽然心里因为突如其来的事情而慌乱，但她还是不肯离开我。
该怎么说呢，该怎么说呢。
我觉得自己明白了为什么昨天没去上学。
不管能够敏锐地读取别人的感情，如果没有行动的话，只会让人毛骨悚然。有能力的鹰不是隐藏爪子，而是拥有隐藏的技术，所以才有能力。
「糟了... ...」
想不到你这么快就看穿我的真心。我听说你很厉害，但这不是人类的范畴。
没办法。
「是的。」
我不再奉承别人了。用平时的声音，像是在攻击似的呼唤着宇智波的名字。
不，不像是要攻击。事实上，他已经把笔插到了天空中。
「什么! 什么... ...」
他条件反射般地看着我的威胁，渐渐地意识模糊起来。
他只是瞥了一眼笔，有点被催眠了。
「哈!」
一切都按计划进行。
在某种程度上，他们早已预料到了敌对的局面。
这次和 mai 的时候水平不同。从真正意义上说，是以强制催眠为前提，做好了准备。
心理咨询室里到处都是我做的记号。窗帘图案的一部分，类似墙上的污渍。书架上的书里，无论看到哪里，都能看到那个符号。
是次级效应。人的大脑无法抵御那些无意识识别出来的东西。
而且最后的饮料里还含有酒精。我的心是出于善意。因为那个行动，没有被注意到。
他不知道善意会露出獠牙，直到真正的邪恶——我来到之前，他都毫无防备。
剩下的就只有我一个人了。
「 ... ... 啊... ... 啊啊... ...」
昨天或许注意到了什么不妙的事情。尽管如此，在失去判断力的状态下，而且如果回过头去看，就会暴露在善意的獠牙之下，抵抗是徒劳的。
渐渐地，他的眼睛阴沉下来，陷入了深深的催眠之中。
「…………」
「 ... ... 这、这里。把记号取下来。」
「 ... ... 是。」
我拿起笔，让心里准备的催眠对象回收。
事实上，他已经失去了视力，已经进入了催眠状态。
这个房间变成了催眠的空间。如果我不熟悉所有的位置，即使不熟悉，如果不是这样，长时间待着也不会安然无恙吧。
我一边确认着茫然的表情和身体的摇晃状态，一边决定已经没事了。
催眠过度也会对精神产生负面影响。到目前为止培养的实验和经验起到了作用。
「摘下来了... ...」
「好吧，我听着呢。」
我悄声耳语，只有心情解除催眠状态。
心灵在瞳孔中重新找回了光芒，默默地注视着我的所作所为。
「你能听见我说话吗? 如果你能听见我说话，请回答。」
「 ... ... ... 是... ... 是。」
」现在你身处一个，非常美好的世界。暴露自己的一切，也不会有人生气。你想要什么就有什么。那里很幸福，很平静」
「 ... ... 是。」
「那么... ... 以后在平常的时候，当我的『规则很简单』的指示出现时，你随时都可以来到这个世界。我有打开那个幸福世界的钥匙。所以，只有当我这么说的时候，钥匙才能打开，进入幸福的世界。明白了吗？如果明白了，在这个世界上的时候，请一边说自己明白的事一边回答
」... 通往这个世界的钥匙，你的声音... ... ... 是的」
之后，我再次仔细地暗示他确认好几次。一下子掉到很深的地方，确认没有精神上的打击。
「在这个世界上，我所说的话，是为了在这个幸福的世界里的规则。虽然你无法回忆起被告知的事情，但是规则会牢牢记在心底，能够按照那样执行。」
「 ... ... 你所说的... ... 规则... ... 服从... ... 是的」
「那么，让我们把我们说过的话保存在一个更深的地方。慢慢来就好。如果你觉得你的大麻保存得很好，告诉我你的名字。」
「 ... ... 保存... ... 是的。」
「 ... ... 呼」
我想我是今年以来最大的一次叹息。
虽然经历了相当麻烦的步骤，但还是成功了。
「辛苦了。」
心拿着毛巾，为我擦汗。
目前处于集中状态，无法下达指示。
「 ... ... 比预想中更危险的对手啊。走错一步就失败了。」
「主人的职责就是不要走错一步，即使走错了，也要纠正错误。」
心平静的表情可以治愈我的疲劳。把身体紧贴在一起，帮我按摩肩膀和按摩。
「在游戏圈子里，还有一个外号叫『拉普拉斯』。虽然觉得有些夸张，但看起来不像是在虚张声势。」
拉普拉斯。当然，这并不是什么别扭的拉普拉斯妖。
「东方... ... 烟。」
「好吧。」
总之催眠已经完成了。剩下的就是如何修正她。
